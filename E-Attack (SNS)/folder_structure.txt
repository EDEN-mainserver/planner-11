# SNS 비즈니스 프로그램 폴더별 역할 요약

| 폴더/파일 명        | 역할 설명                         | 클로드 코딩 활용 팁                                   |
| :---------------- | :------------------------------- | :-------------------------------------------------- |
| docs/             | 기획서, API 명세, 비즈니스 로직     | 클로드에게 "기획 배경은 이 폴더를 봐"라고 지시         |
| src/              | 실제 실행 가능한 파이썬 소스 코드   | 모든 개발의 중심지. 실행 로직은 여기에만 배치          |
| src/core/         | 공통 엔진 (스케줄러, 베이스 클라이언트) | 어떤 SNS든 공통으로 쓰이는 핵심 기능을 관리           |
| src/services/     | SNS 플랫폼별 독립 폴더             | 새로운 SNS 추가 시 이 폴더 안에 새 폴더만 생성          |
| src/services/threads/ | 스레드 전용 API 및 로직          | 스레드 관련 수정 시 클로드가 이 경로만 집중하게 함      |
| src/utils/        | 공통 도구 (이미지 변환, 텍스트 정제) | 단순 반복 계산이나 포맷 변경 함수를 모아둠             |
| tests/            | 코드 검증용 테스트 스크립트         | 클로드에게 "테스트 코드 짜서 검증해줘"라고 할 때 사용   |
| .env              | API Key, 계정 비밀번호             | 보안 파일. 절대 외부 유출 금지 (Git 제외 필수)         |
| CLAUDE.md         | 프로젝트 가이드 및 규칙             | 클로드가 작업 전 가장 먼저 읽는 '매뉴얼'              |
| requirements.txt  | 설치 필요한 라이브러리 목록         | 프로젝트 환경 구축 시 필수 정보                       |
| main.py           | 프로그램 전체 실행 진입점           | 프로그램의 '시작 버튼' 역할                           |
