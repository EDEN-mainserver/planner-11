# SNS 트렌드 분석 및 AI 재구성 자동 배포 — 유저 플로우

## 전체 흐름
```
[Step 1] 크롤링 실행 (새로고침)
    ↓
[Step 2] 인기글 대시보드 렌더링 (카드/테이블)
    ↓
[Step 3] 레퍼런스 게시물 선택 (행 클릭)
    ↓
[Step 4] AI 키워드 추천 + 사용자 키워드 입력 (모달)
    ↓
[Step 5] 타겟 플랫폼 선택 (스레드/블로그/X/아이보스/링크드인)
    ↓
[Step 6] 프로젝트 계정 선택 (드롭다운)
    ↓
[Step 7] AI 콘텐츠 재구성 (채널별 규격 맞춤 생성)
    ↓
[Step 8] 최종 편집 + 즉시/예약 배포
```

---

## 상세 플로우

### 1. 인기글 기반 트렌드 분석

```
1. 인기글 수집
   1.1. 스케줄링 설정 — 주기적 크롤링 작업 실행
   1.2. 외부글 크롤링 — 아이보스/스레드/X 인기 게시물 수집
   1.3. 데이터 정제/저장 — 수집 데이터를 정제 후 DB 저장
   1.4. 수집 실패 처리 — 사이트 접속 실패/HTML 변경 시 에러 로그 + 알림

2. 트렌드 대시보드
   2.1. 인기글 목록 API — 필터링/정렬 제공
   2.2. 인기글 UI 렌더링 — 카드 형태 목록
   2.3. 게시물 상세 조회 — 원문 링크 + 요약 모달
   2.4. API 요청 실패 — 에러 메시지 + 재시도
```

### 2. 프로젝트 및 채널 관리

```
1. 프로젝트 생성/관리
   1.1. 프로젝트 목록 API — 전체 프로젝트 조회
   1.2. 프로젝트 생성 UI — 이름/설명 입력
   1.3. 프로젝트 정보 저장 — DB 저장
   1.4. DB 저장 실패 처리 — 롤백 + 실패 메시지

2. 채널/톤앤매너 설정
   2.1. 채널 연동 API — OAuth 인증
   2.2. 톤앤매너 설정 UI — 전문적/유머러스 등 선택
   2.3. 설정값 DB 저장
   2.4. 채널 연동 실패 — 원인 표시 + 가이드
```

### 3. AI 레퍼런스 기반 콘텐츠 생성

```
1. 생성 조건 설정
   1.1. 레퍼런스 선택 UI — 인기글 대시보드에서 선택
   1.2. 키워드 추천/입력 — AI 추천 3개 + 직접 입력
   1.3. 타겟 채널 선택 — 프로젝트 내 설정된 채널 중 선택
   1.4. 입력값 검증 — 필수값 확인

2. AI 초안 생성/요청
   2.1. 프롬프트 엔지니어링 — 레퍼런스+키워드+톤앤매너+채널 규격 조합
   2.2. LLM API 호출 — Gemini API
   2.3. 결과 파싱/저장 — 채널별 초안 DB 저장
   2.4. API 타임아웃 처리 — 재시도 or 실패 알림

3. 생성 결과 확인
   3.1. 채널별 초안 UI — 탭 구분 표시
   3.2. 초안 편집 기능 — TextArea 수정/저장
   3.3. 결과 없음 처리 — 재시도 버튼
```

### 4. 콘텐츠 게시 및 예약 관리

```
1. 콘텐츠 배포 실행
   1.1. 채널별 게시 API — 각 채널 API 규격 맞춤 게시
   1.2. 예약 작업 등록 — 스케줄러 큐에 등록
   1.3. 배포 UI — 즉시/예약 선택 + 날짜/시간 컴포넌트
   1.4. 게시 실패 처리 — 실패 원인 기록 + 알림

2. 예약 현황 관리
   2.1. 예약 목록 API — 상태(대기/성공/실패) 조회
   2.2. 예약 관리 대시보드 — 캘린더/목록 뷰
   2.3. 예약 취소 기능 — 스케줄러 큐에서 삭제
   2.4. 상태 업데이트 실패 — 재시도 + 로그
```
