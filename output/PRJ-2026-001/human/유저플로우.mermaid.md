## FLOW-001: 단순 문의(가격) 자동 응대 플로우

| 항목 | 내용 |
|---|---|
| **연결 시나리오** | SC-001 |
| **주체** | 시스템 |

크몽에서 새 메시지를 감지하여 '가격 문의'로 분류된 경우, 자동으로 가격 정보를 포함한 응답을 발송하고 내역을 기록하는 자동화 흐름입니다.

### 노드 목록

|---|---|---|---|---|---|
| N-001 | start | 5분 주기 스케줄러 | system | - | - |
| N-002 | action | 크몽 메시지 수집 | system | SPEC-001 | ACT-001 |
| N-003 | decision | 새 문의 존재? | system | - | - |
| N-004 | action | AI 문의 유형 분류 | ai | SPEC-002 | ACT-002 |
| N-005 | decision | 가격 문의인가? | system | - | - |
| N-006 | action | 자동 응답 생성 | ai | SPEC-003 | ACT-003 |
| N-007 | action | 크몽 자동 응답 발송 | system | SPEC-003 | ACT-004 |
| N-008 | action | 구글 시트 로그 기록 | system | SPEC-005 | ACT-006 |
| N-009 | end | 프로세스 완료 | system | - | - |
| N-010 | action | 담당자 알림 (예외) | system | SPEC-004 | ACT-005 |
| N-091 | error | 크몽 세션 만료 처리 | system | - | - |
| N-092 | error | LLM API 실패 처리 | system | - | - |

### 에러 처리 경로

| 발생 노드 | 에러 유형 | 처리 노드 | 설명 |
|---|---|---|---|
| N-002 | auth_failure | N-091 | 크몽 세션 만료 시 재로그인을 시도하고, 실패하면 담당자에게 알림을 보냅니다. |
| N-004 | api_failure | N-092 | LLM 분류 API 호출 실패 시, 담당자 알림으로 프로세스를 전환합니다. |
