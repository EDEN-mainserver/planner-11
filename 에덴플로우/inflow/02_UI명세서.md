### Typography

| 토큰명 | 크기/굵기 | 용도 |
|--------|-----------|------|

### 1. 공통 레이아웃 (Layout)

```

│  SIDEBAR (260px, 고정)   │   MAIN CONTENT (1fr)     │
│  ─────────────────────   │                           │
│  [INFLOW 로고]           │   <각 페이지 콘텐츠>      │
│                           │                           │
│  계정진단 ∧              │                           │
│    📄 보고서 확인하기    │                           │
│                           │                           │
│  AI 기획 ∧              │                           │
│    ⊙ 계정 방향성 기획   │                           │
│    AA 이름 추천          │                           │
│    👤 프로필 세팅        │                           │
│    🎬 릴스 기획          │                           │
│    🕐 AI기획 보관함      │                           │
│                           │                           │
│  콘텐츠 ∧               │                           │
│    ▶ 릴스 모음           │                           │
│                           │                           │
│  지원 ∧                 │                           │
│    🔔 공지사항           │                           │
│    ❓ 원격지원            │                           │
│  ─────────────────────   │                           │
│  [정 아바타] 정지한 님  │                           │
│       이메일             │                           │
│  [→ 로그아웃]           │                           │
└─────────────────────────────────────────────────────┘
```

| 요소 | 스펙 |
|------|------|
| 사이드바 배경 | `#F4F4F8` |
| 로고 | SVG, "IN" 검정 + "FLOW" 그라데이션 (보라→핑크) |
| 섹션 라벨 | 12px, #999, Letter-spacing 0.05em |
| 메뉴 아이템 | 14px, 아이콘(16px) + 텍스트, hover 시 배경 `#EEEEF8` |
| 활성 메뉴 | bg `#6C63FF`, 텍스트 흰색, border-radius 10px |
| 아바타 | 40px 원형, bg `#6C63FF`, 텍스트 흰색 이니셜 |
| 로그아웃 버튼 | border 1px `#E5E5EF`, rounded-xl, 전체 너비 |
| 그룹 접기/펼치기 | 화살표(∧/∨) 클릭으로 토글 |

### 4. AI 스텝 폼 공통

```
┌──────────────────────────────────────────┐
│  STEP N OF N                             │
│                                          │
│  [아이콘] 질문 텍스트 (24px Bold)        │
│                                          │
│  ┌──────────┐  ┌──────────┐             │
│  │  옵션 1  │  │  옵션 2  │             │
│  └──────────┘  └──────────┘             │
│  ┌──────────┐  ┌──────────┐             │
│  │  옵션 3  │  │  옵션 4  │             │
│  └──────────┘  └──────────┘             │
│                                          │
│  [리스트에 없다면 직접 입력해주세요.]    │
│                                          │
│  < 이전          [다음 단계 →]           │
└──────────────────────────────────────────┘
```

| 요소 | 스펙 |
|------|------|
| STEP 표시 | 12px, `#999`, Letter-spacing |
| 질문 텍스트 | 24px Bold, 아이콘 + 텍스트 |

| 옵션 선택 상태 | border `#6C63FF` 2px, bg `#F5F4FF` |
| 직접입력 필드 | `#F9F9FF` bg, border 점선, placeholder 회색 |
| 이전 버튼 | 텍스트만, `<` 화살표 |
| 다음/생성 버튼 | 검정 배경, 흰 텍스트, 화살표 |
