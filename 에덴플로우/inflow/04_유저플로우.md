# 🗺️ INFLOW 유저 플로우

> 단일 사용자 기준 (로그인 1회 후 지속 사용)

---

## 전체 플로우 맵

```
[앱 진입]
    │
    ├─ 미로그인 ──────────────────→ [로그인 페이지]
    │                                      │
    │                              이메일+비밀번호 입력
    │                                      │
    └─ 로그인됨 ──────────────────→ [대시보드 /]
                                           │
              ┌────────────────────────────┼──────────────────────────┐
              │                           │                           │
         [계정진단]                   [AI 기획]                  [콘텐츠/지원]
              │                           │                           │
       보고서 확인하기           ┌────────┼────────┐              ┌───┴───┐
                           방향성기획  이름추천  프로필세팅    릴스모음  공지사항
                                │    릴스기획
                                │    보관함
                                │
                         [AI 생성 흐름]
```

## UF-01. 첫 진입 및 로그인

```
[브라우저에서 URL 접속]
        │
        ▼
[로그인 상태 확인]
        │
   ┌────┴────┐
  예         아니오
   │         │
   ▼         ▼
[대시보드]  [로그인 페이지]
              │
        이메일 입력
              │
        비밀번호 입력
              │
        [로그인 버튼 클릭]
              │
        ┌─────┴─────┐
      성공           실패
        │             │
        ▼             ▼
   [대시보드]    에러 메시지 표시
                  → 재시도
```

## UF-02. 대시보드 탐색

```
[대시보드]
        │
        ├─ [최근 AI기획 카드] 클릭
        │         └→ [AI기획 보관함] 이동
        │
        ├─ "전체 기획내역 확인" 클릭
        │         └→ [AI기획 보관함] 이동
        │
        ├─ [공지사항 항목] 클릭
        │         └→ [공지사항 상세] 이동
        │
        ├─ "전체 공지사항 확인" 클릭
        │         └→ [공지사항 목록] 이동
        │
        └─ [하단 바로가기 카드] 클릭
                  ├→ 계정진단 → [보고서 확인하기]
                  ├→ 릴스기획 → [릴스 기획 페이지]
                  ├→ 릴스모음 → [릴스 모음 페이지]
                  └→ 사용방법 → (가이드 페이지)
```

## UF-03. 계정 방향성 기획

```
[사이드바 "계정 방향성 기획" 클릭]
        │
        ▼
[랜딩 페이지]
 - 타이틀/설명/미리보기(blur) 표시
 - 사용 횟수 확인 (FREE: 0/3)
        │
        ├─ 한도 소진 시 → 안내 메시지, 버튼 비활성화
        │
        └─ [기획 시작하기] 버튼 클릭
                  │
                  ▼
         ┌─────────────────┐
         │  STEP 1 OF 4    │
         │  닉네임 입력    │
         │  [텍스트 입력]  │
         └────────┬────────┘
                  │ [다음 단계] 클릭 (입력값 있을 때만)
                  ▼
         ┌─────────────────┐
         │  STEP 2 OF 4    │
         │  관심 주제 선택 │
         │  (6개 옵션 or   │
         │   직접 입력)    │
         └────────┬────────┘
                  │ [다음 단계] 클릭
                  ▼
         ┌─────────────────┐
         │  STEP 3 OF 4    │
         │  운영 목적 선택 │
         │  (4개 옵션 or   │
         │   직접 입력)    │
         └────────┬────────┘
                  │ [다음 단계] 클릭
                  ▼
         ┌─────────────────┐
         │  STEP 4 OF 4    │
         │  인스타 경험    │
         │  (3지선다)      │
         └────────┬────────┘
                  │ [생성하기] 클릭
                  ▼
         [OpenAI API 호출 중...]
         (로딩 스피너 표시)
                  │
          ┌───────┴───────┐
         성공             실패
          │               │
          ▼               ▼
   [결과 렌더링]     에러 메시지
   - 방향성 리포트   → 재시도 버튼
   - 자동 보관함 저장
   - 사용 횟수 +1
```

## UF-04. 이름 추천

```
[사이드바 "이름 추천" 클릭]
        │
        ▼
[랜딩 페이지]
 - 타이틀/설명/샘플 리스트(blur)
 - 사용 횟수 확인 (FREE: 0/3)
        │
        └─ [시작하기] 클릭
                  │
                  ▼
         [키워드/특성 입력 폼]
          (자유 텍스트 입력)
                  │
                  └─ [생성하기] 클릭
                            │
                    [OpenAI 호출 중...]
                            │
                     ┌──────┴──────┐
                    성공           실패
                     │             │
                     ▼             ▼
             [닉네임 리스트]    에러 메시지
             #1~#N 후보 카드
                     │
             카드 클릭 → 선택 하이라이트
                     │
             자동 보관함 저장 완료
```

## UF-07. AI기획 보관함

```
[사이드바 "AI기획 보관함" 클릭]
        │
        ▼
[보관함 목록 페이지]
 - 전체 카드 그리드 표시
        │
        ├─ [카테고리 탭] 클릭
        │    └→ 해당 타입 카드만 필터링
        │
        ├─ [검색창] 입력
        │    └→ 제목 실시간 필터링
        │
        ├─ [카드] 클릭
        │    └→ [상세 페이지]
        │              │
        │         ┌────┴────┐
        │      [PDF 저장]  [X 닫기]
        │         PDF 다운  목록으로
        │
        ├─ [카드 hover] → 휴지통 아이콘 표시
        │    └→ [삭제] 클릭
        │              │
        │         "삭제하시겠습니까?" 확인
        │              │
        │         ┌────┴────┐
        │        확인       취소
        │         │
        │        카드 삭제
        │
        └─ [페이지네이션] < 1/N > 이동
```

## UF-08. 릴스 모음

```
[사이드바 "릴스 모음" 클릭]
        │
        ▼
[릴스 목록 페이지]
 - 3열 그리드 릴스 카드
        │
        ├─ [카테고리 탭] 클릭
        │    └→ 해당 카테고리 필터링
        │
        ├─ [< >] 화살표 클릭
        │    └→ 카테고리 탭 좌우 스크롤
        │
        ├─ [검색창] 입력
        │    └→ 캡션/해시태그 실시간 필터
        │
        ├─ [X] 클릭 → 검색 초기화
        │
        ├─ [정렬설정] 클릭
        │    └→ 최신순/인기순 드롭다운
        │
        ├─ [기간설정] 클릭
        │    └→ 날짜 범위 선택
        │
        └─ [릴스 카드] 클릭
              └→ 인스타그램 링크 이동 또는 모달
```

## 에러 처리 플로우

```
[AI 생성 실패]
        │
        ├─ 네트워크 오류 → "요청에 실패했습니다. 다시 시도해주세요." + 재시도 버튼
        ├─ 사용량 초과  → "사용 횟수를 모두 사용했습니다." 안내
        └─ 서버 오류   → "일시적인 오류가 발생했습니다." + 재시도 버튼
```
