# 📁 INFLOW 프로젝트 폴더 구조

## 기술 스택

| 분류 | 선택 |
|------|------|
| 프레임워크 | Next.js 14 (App Router) |
| 언어 | TypeScript |
| 스타일 | Tailwind CSS |
| 상태관리 | Zustand |
| AI | OpenAI API (GPT-4o) |
| DB | JSON 파일 or SQLite (로컬 전용) |
| 인증 | 로컬 세션 (단일 사용자) |

---
