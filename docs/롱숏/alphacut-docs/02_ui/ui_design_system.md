## 2. 컬러 팔레트

### 2.1 배경 컬러
| 역할 | 색상 설명 | 용도 |
|------|-----------|------|
| Page Background | 짙은 블랙 (#0A0A10 계열) | 전체 페이지 배경 |
| Section Background | 약간 밝은 다크 (#111118 계열) | 카드, 섹션 배경 |
| Card Background | 반투명 다크 (#1A1A25 계열) | FAQ 아코디언, 후기 카드 |
| Input Background | 다크 그레이 (#1C1C28 계열) | 텍스트 입력 필드 |

### 2.2 브랜드 컬러 (그라디언트)
| 역할 | 색상 | 용도 |
|------|------|------|
| Primary Gradient | 파란-보라-핑크-오렌지 좌→우 | 주요 CTA 버튼, 탭 활성 상태 |
| Accent Purple | #7B5CF0 계열 | 헤드라인 강조 텍스트, 보조 강조 |
| Accent Blue | #5B8AF0 계열 | 링크, 아이콘 포인트 |

### 2.3 플랜별 컬러 코드
| 플랜 | 색상 | 용도 |
|------|------|------|
| 무료 | 오렌지 (#FF6B35 계열) | 무료 플랜 카드 테두리, 아이콘 배경 |
| Basic | 초록 (#22C55E 계열) | Basic 플랜 카드 테두리, 아이콘 배경 |
| Pro (추천) | 보라 (#7B5CF0 계열) | Pro 플랜 카드 테두리, "추천" 배지 |
| Max | 오렌지-앰버 (#F59E0B 계열) | Max 플랜 카드 테두리, 아이콘 배경 |

### 2.4 텍스트 컬러
| 역할 | 색상 | 용도 |
|------|------|------|
| Primary Text | 흰색 (#FFFFFF) | 헤드라인, 주요 본문 |
| Secondary Text | 연한 회색 (#9CA3AF 계열) | 서브텍스트, 설명 문구 |
| Muted Text | 더 연한 회색 (#6B7280 계열) | 캡션, 날짜, 부가 정보 |
| Highlight Text | 보라/핑크 그라디언트 | 헤드라인 강조 단어 (예: "10편의 숏폼") |

### 2.5 기타 컬러
| 역할 | 색상 | 용도 |
|------|------|------|
| Border | 반투명 흰색 (#FFFFFF 10~15%) | 카드 테두리, 구분선 |
| Success | 초록 (#22C55E) | 체크마크 아이콘 |
| Warning | 오렌지 (#F97316) | 알림 배너 |
| Chatbot Button | 파란색 (#3B82F6 계열) | 우하단 CS 챗봇 버튼 |

## 3. 타이포그래피

### 3.1 폰트 패밀리
| 용도 | 폰트 | 비고 |
|------|------|------|
| 한글 텍스트 전반 | 시스템 한글 폰트 (고딕 계열) | Noto Sans KR 추정 |
| 영문/숫자 | 시스템 영문 폰트 | Inter 또는 유사 산세리프 추정 |
| 로고 | 커스텀 폰트 또는 이미지 로고 | 아이콘+텍스트 조합 |

### 3.2 타입 스케일
| 역할 | 크기 | 굵기 | 용도 |
|------|------|------|------|
| Display Heading | ~56~64px | ExtraBold (800) | Hero 메인 헤드라인 |
| H1 | ~40~48px | Bold (700) | 섹션 대제목 |
| H2 | ~28~36px | Bold (700) | 섹션 소제목 |
| H3 | ~20~24px | SemiBold (600) | 카드 제목, 기능명 |
| Body Large | ~16~18px | Regular (400) | 주요 본문, 설명 |
| Body | ~14~16px | Regular (400) | 일반 본문 |
| Caption | ~12~14px | Regular (400) | 캡션, 부가 정보 |

### 3.3 텍스트 강조 패턴
- **2색 헤드라인**: 흰색 텍스트 + 보라/블루 그라디언트 강조 텍스트 조합
  - 예: "1편의 롱폼 영상으로 **10편의 숏폼을 만들어보세요.**"
- **섹션 레이블 배지**: 작은 배지 안에 텍스트 → 섹션 상단에 배치

## 4. 버튼 컴포넌트

### 4.1 Primary CTA 버튼
| 속성 | 값 |
|------|----|
| 배경 | 파란-보라-핑크-오렌지 그라디언트 |
| 텍스트 색상 | 흰색 |
| 테두리 반경 | 12px (둥근 사각형) |
| 패딩 | 16px 상하, 32px 좌우 |
| 폰트 | SemiBold, ~16px |
| 너비 | 전체 너비(Full-width) 또는 자동 |
| 예시 | "쇼츠로 변환하기 →", "무료로 시작하기" |

### 4.2 Secondary (Outline) 버튼
| 속성 | 값 |
|------|----|
| 배경 | 투명 |
| 테두리 | 흰색 or 그레이 1px |
| 텍스트 색상 | 흰색 |
| 테두리 반경 | 12px |
| 예시 | "로그인" 버튼 |

### 4.3 플랜 CTA 버튼
| 플랜 | 배경 색상 |
|------|-----------|
| 무료 | 오렌지 단색 |
| Basic | 초록 단색 |
| Pro | 보라 단색 |
| Max | 오렌지-앰버 단색 |

### 4.4 아이콘 + 텍스트 버튼
- 텍스트 앞에 아이콘(링크, 업로드 등) 삽입
- 예: "🔗 유튜브 링크", "↑ 파일 업로드"

## 6. 카드 컴포넌트

### 6.1 기능 소개 카드
| 속성 | 값 |
|------|----|
| 배경 | 다크 (#1A1A28 계열) |
| 테두리 | 반투명 흰색 1px |
| 테두리 반경 | 16~20px |
| 패딩 | 24~32px |
| 구성 | 상단 이미지/아이콘 + 제목(H3) + 설명 |

### 6.2 요금제 플랜 카드
| 속성 | 값 |
|------|----|
| 배경 | 다크 (#111118 계열) |
| 테두리 | 플랜별 색상 1~2px |
| 테두리 반경 | 16px |
| 추천 플랜 | 상단에 "추천" 배지 (보라색) + 테두리 강조 |
| 구성 | 아이콘 → 플랜명 → 가격 → 이용량 → CTA → 기능 체크리스트 |

### 6.3 후기 카드 (Testimonial)
| 속성 | 값 |
|------|----|
| 배경 | 다크 (#1A1A28 계열) |
| 테두리 반경 | 16px |
| 구성 | 큰따옴표 인용 텍스트 + 하단 프로필 (이미지 + 이름 + 분야) |
| 인용 부호 | 큰 따옴표 아이콘 (우측 정렬, 불투명) |

## 7. 탭 컴포넌트 (Tab)

| 상태 | 스타일 |
|------|--------|
| 활성(Active) | 그라디언트 배경 + 흰색 텍스트 + 아이콘 |
| 비활성(Inactive) | 투명 배경 + 회색 텍스트 + 아이콘 |
| 컨테이너 | 다크 배경 둥근 사각형, 탭 2개 나란히 배치 |

---

## 8. 배지 컴포넌트 (Badge)

| 종류 | 스타일 | 용도 |
|------|--------|------|
| 섹션 레이블 배지 | 둥근 필 배경(다크) + 테두리 + 흰색 텍스트 | 섹션 상단 레이블 |
| 추천 배지 | 보라 배경 + 흰색 텍스트 "추천" | Pro 플랜 카드 상단 |
| 할인 배지 | 초록 필 + 흰색 텍스트 "16% 할인" | 연간 결제 토글 |
| 알림 배너 배지 | 오렌지 그라디언트 + 아이콘 + 텍스트 | 우상단 플로팅 배너 |

---

## 9. 아코디언 컴포넌트 (Accordion / FAQ)

| 속성 | 값 |
|------|----|
| 배경 | 다크 카드 배경 |
| 테두리 반경 | 12px |
| 질문 행 | 텍스트 (Bold) + 우측 화살표 아이콘(▽) |
| 펼침 상태 | 화살표 회전 + 답변 텍스트 슬라이드 다운 |
| 구분 | 항목 간 여백(margin) 으로 분리 |

## 10. 캐러셀 컴포넌트 (Carousel)

### 10.1 쇼츠 갤러리 캐러셀 (자동 슬라이드)
| 속성 | 값 |
|------|----|
| 방향 | 가로 무한 스크롤 (좌→우 자동) |
| 행 | 2행 구성 (위/아래) |
| 카드 | 세로형 쇼츠 썸네일 + 프로필 이미지 + 채널명 |
| 애니메이션 | 연속 흐름(continuous scroll) |

### 10.2 후기 캐러셀 (수동 슬라이드)
| 속성 | 값 |
|------|----|
| 방향 | 가로 슬라이드 |
| 한 화면 | 3개 카드 노출 (추정) |
| 컨트롤 | 이전(←) / 다음(→) 원형 버튼 (하단 중앙) |
| 버튼 스타일 | 다크 원형 + 흰색 화살표 / 활성 버튼은 밝은 색 |

## 11. 토글 컴포넌트 (Toggle)

| 속성 | 값 |
|------|----|
| 요금제 페이지 | 월간 / 연간 2-option 토글 |
| 비활성 스타일 | 투명 배경 + 회색 텍스트 |
| 활성 스타일 | 다크 필(pill) 배경 + 흰색 텍스트 |
| 할인 배지 | 연간 옆에 "16% 할인" 초록 배지 병기 |

## 12. 그라디언트 패턴 정리

| 패턴명 | 방향 | 색상 구성 | 적용 위치 |
|--------|------|-----------|-----------|
| Primary CTA | 좌→우 | 파랑 → 보라 → 핑크 → 오렌지 | 메인 버튼, 탭 활성 |
| Heading Accent | - | 보라 → 블루 | 헤드라인 강조 텍스트 |
| Section BG Glow | 방사형 | 보라/블루 희미한 글로우 | 일부 섹션 배경 |
| Alert Banner | 좌→우 | 오렌지 → 핑크 | 상단 알림 배너 |

## 13. 간격(Spacing) 시스템

| 레벨 | 크기 | 용도 |
|------|------|------|
| xs | 4px | 인라인 요소 간격 |
| sm | 8px | 컴포넌트 내부 소간격 |
| md | 16px | 일반 패딩, 요소 간 기본 간격 |
| lg | 24~32px | 카드 패딩, 섹션 내 요소 간격 |
| xl | 48~64px | 섹션 간 상하 여백 |
| 2xl | 80~120px | 대형 섹션 여백 |

## 14. 반응형 브레이크포인트 (추정)

| 브레이크포인트 | 너비 | 변화 |
|---------------|------|------|
| Mobile | < 768px | 1열 레이아웃, 햄버거 메뉴 |
| Tablet | 768~1024px | 2열 레이아웃 |

## 15. 아이콘 시스템

| 사용 위치 | 아이콘 스타일 |
|-----------|--------------|
| 네비게이션 | 라인 아이콘 |
| 기능 카드 | 컬러 일러스트 이미지 |
| 체크리스트 | ✓ 초록 체크 아이콘 |
| FAQ 화살표 | ∨ 라인 화살표 |
| 입력 필드 | 링크(🔗), 업로드(↑) 라인 아이콘 |
| CTA 버튼 | → 화살표 텍스트 |
| CS 챗봇 | 채팅 버블 아이콘 (파란 원형) |
