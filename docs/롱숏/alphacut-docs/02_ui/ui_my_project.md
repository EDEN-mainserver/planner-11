## 4. 프로젝트 목록 영역

### 4.1 레이아웃
- 카드 그리드 (3~4열, 모바일 1~2열)
- 최신 순 정렬 (기본)

### 4.2 프로젝트 카드 구성
| 요소 | 내용 |
|------|------|
| 썸네일 | 9:16 세로형 쇼츠 대표 이미지 |
| 제목 | 쇼츠 제목 (AI 추천 또는 수정된 제목) |
| 원본 영상 | 원본 유튜브 영상명 또는 파일명 |
| 생성일 | 날짜 표시 |
| 상태 배지 | 처리 중 / 완료 / 오류 |
| 액션 버튼 | 미리보기 / 편집 / 재생성 / 다운로드 / 유튜브 업로드 |

### 4.3 처리 중 상태 카드
| 요소 | 내용 |
|------|------|
| 썸네일 영역 | 로딩 스피너 또는 진행률 표시 |
| 상태 텍스트 | AI 처리 중... |
| 액션 버튼 | 비활성화 |

---

## 5. 프로젝트 카드 액션 버튼 상세

| 버튼 | 아이콘 | 동작 |
|------|--------|------|
| 미리보기 | 재생(▶) 아이콘 | 인라인 또는 모달 영상 재생 |
| 편집 | 연필(✏) 아이콘 | 편집 화면으로 이동 |
| 재생성 | 새로고침(↺) 아이콘 | 이용권 50% 소모 후 재생성 |
| 다운로드 | 다운로드(↓) 아이콘 | 파일 저장 (플랜 제한 적용) |
| 유튜브 업로드 | 유튜브 로고 또는 업로드 아이콘 | 유튜브 채널 업로드 플로우 |

---

## 6. 빈 상태 (Empty State)

| 요소 | 내용 |
|------|------|
| 일러스트 | 빈 프로젝트 일러스트 또는 아이콘 |
| 안내 텍스트 | 아직 만든 쇼츠가 없어요. 첫 쇼츠를 만들어보세요! |
| CTA 버튼 | 쇼츠 만들기 → 입력 UI로 이동 |

## 7. 이용권 잔량 표시 UI

| 요소 | 내용 |
|------|------|
| 잔량 수치 | 남은 이용권 분(min) 표시 |
| 진행 바 | 전체 이용권 대비 잔량 비율 프로그레스 바 |
| 현재 플랜 | Basic / Pro / Max 플랜명 표시 |
| 업그레이드 링크 | 요금제 페이지로 이동 링크 |
