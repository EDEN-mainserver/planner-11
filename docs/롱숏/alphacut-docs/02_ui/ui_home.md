## 섹션 1: Hero 섹션

### 레이아웃
- 전체 너비 다크 배경, 모든 요소 수평 중앙 정렬
- 소셜 배지 → H1(2행) → 서브텍스트 → 탭 UI → 입력 필드 → CTA 버튼 순 수직 배치

### 구성 요소
| 요소 | 텍스트 내용 | 스타일 |
|------|------------|--------|
| 소셜 배지 | # 탑급 MCN 파트너사 도입 등 (자동 순환) | 다크 필 배지, 흰색 테두리 |
| H1 1행 | 1편의 롱폼 영상으로 | 흰색 ExtraBold ~56px |
| H1 2행 | 10편의 숏폼을 만들어보세요. | 보라-블루 그라디언트 ExtraBold ~56px |
| 서브텍스트 | 알파컷은 링크만 입력하면 롱폼 영상을 여러 편의 쇼츠로 바꿔드립니다. | 연한 회색 Regular ~18px |
| 탭 (활성) | 유튜브 링크 + 링크 아이콘 | 그라디언트 배경, 흰색 텍스트 |
| 탭 (비활성) | 파일 업로드 + 업로드 아이콘 | 투명 배경, 회색 텍스트 |
| URL 입력 필드 | 플레이스홀더: 유튜브 영상 링크를 입력하세요 (예: ...) | 다크 배경, 좌측 링크아이콘, 라운드 12px |
| CTA 버튼 | 쇼츠로 변환하기 → | 그라디언트 필, 전체너비, SemiBold 16px |

### Hero 소셜 배지 순환 문구
- # 방송사 공식 채널 사용
- # 탑급 MCN 파트너사 도입
- # 서울대생 제작/운영
- # 마케팅 대행사들의 선택
- # 유튜버 10,000명의 검증

### 기능 카드 6종
| 카드 No | 제목 | 설명 텍스트 |
|---------|------|------------|
| 1 | AI 자동 자막 | 귀찮은 자막 편집은 이제 그만! AI가 음성을 인식해서 자막을 만들어줘요. |
| 2 | 무음 구간 제거 | 지루함은 금물! 말 없는 구간만 골라내어 쇼츠를 속도감 있게 만들어드려요. |
| 3 | AI 후킹 보이스 | 영상 앞에 전문 성우 같은 AI의 소개로 영상의 몰입감을 2배 더 높여보세요. |
| 4 | 첫 3초 후킹 기능 | 가장 중요한 첫 3초에서 시작하자마자 시선을 사로잡으세요. |
| 5 | 쇼츠 템플릿 | 미리 준비된 템플릿으로 트렌디한 배경부터 로고까지 모두 설정해보세요! |

### 기능 카드 UI 구조
- 상단: 기능 설명 일러스트 또는 실제 UI 스크린샷
- 하단: H3 기능명(흰색 Bold) + 설명 텍스트(연한 회색)
- 다크 배경 + 라운드 테두리 카드 (border-radius 16px)

## 섹션 8: 크리에이터 후기 섹션

| 요소 | 내용 |
|------|------|
| 레이블 | 크리에이터 후기 |
| H2 | 이미 1만 명이 넘는 크리에이터가 알파컷을 통해 성장하고 있어요 |
| 레이아웃 | 3열 카드 슬라이더 캐러셀 |
| 슬라이더 컨트롤 | ← 이전 / → 다음 원형 버튼 (하단 중앙 배치) |
| 버튼 스타일 | 다크 원형, 활성 버튼은 밝은 색 강조 |

### 후기 카드 UI 구조
- 다크 배경 카드 + 라운드 테두리
- 본문: 큰따옴표로 묶인 후기 텍스트 (Regular, 연한 회색)
- 우측 하단: 큰따옴표 아이콘 장식 (불투명 처리)
- 하단 프로필 행: 원형 프로필 이미지(32~40px) + 이름(흰색 Bold) + 분야(회색 Regular)

## 섹션 9: 성과 지표 섹션

| 요소 | 내용 |
|------|------|
| 레이블 | 실제 쇼츠 성과 |
| H2 | 알파컷만의 AI 기술로 / 조회수가 잘 나오는 쇼츠를 만들어요 |
| 핵심 수치 | 2100% — 보라-블루 그라디언트, 초대형 폰트 (~80~96px) |
| 주석 | * 기존 쇼츠 대비 알파컷으로 제작한 쇼츠의 조회수 |
| 그래프 | 꺾은선 그래프 (보라→핑크 그라디언트 선) — 알파컷 사용 전후 구분 점선 포함 |
| 레이아웃 | 다크 카드 내 수치+주석 좌측 + 그래프 이미지 우측 |
