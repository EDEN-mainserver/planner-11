## 3. 사용자 유형별 주요 경로

### 3.1 신규 방문자 (비로그인)
```
홈 접속 → 서비스 확인 → CTA 클릭 → 회원가입 → 30분 무료 체험 → 쇼츠 생성
```

### 3.2 기존 무료 사용자 (이용권 소진)
```
로그인 → 쇼츠 생성 시도 → 이용권 부족 → 요금제 페이지 → 플랜 구매 → 생성 재개
```

### 3.3 유료 구독 사용자 (정기 이용)
```
로그인 → 내 프로젝트 → 새 쇼츠 생성 → AI 처리 → 편집 → 유튜브 업로드
```

### 3.4 크리에이터 (추천인 참여)
```
로그인 → /referral-promo → 초대 코드 생성 → SNS 공유 → 커미션 정산
```

### 3.5 콘텐츠 기획자 (알파토픽 이용)
```
홈 → 무료 도구 → 알파토픽 → 키워드 검색 → 트렌드 분석 → 쇼츠 주제 선정
```

## 4. 핵심 전환 포인트 (Conversion Points)

| 전환 포인트 | 설명 | 관련 UI |
|-------------|------|----------|
| 비로그인 → 로그인 | 쇼츠 변환 시도 시 로그인 유도 | Hero CTA, 알림 배너 |
| 무료 → 유료 | 이용권 부족 시 요금제 구매 유도 | 이용권 부족 팝업, 요금제 페이지 |
| 결과 확인 → 편집 | 더 나은 결과물 원할 때 편집 진입 | 결과물 카드 편집 버튼 |
| 결과 확인 → 재생성 | 결과물 불만족 시 재생성 선택 | 재생성하기 버튼 |
| 편집 완료 → 업로드 | 편집 후 유튜브 배포 | 유튜브 업로드 버튼 |
