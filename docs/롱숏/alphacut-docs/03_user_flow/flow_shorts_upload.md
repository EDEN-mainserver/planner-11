# 쇼츠 생성 플로우 — 파일 업로드 (FL-03)

> 파일 위치: alphacut-docs/03_user_flow/flow_shorts_upload.md
> 작성일: 2026-04-16

---

## 플로우 개요

| 항목 | 내용 |
|------|------|
| 플로우 ID | FL-03 |
| 대상 사용자 | 로그인된 사용자 |
| 시작점 | 홈(/) Hero 섹션 '파일 업로드' 탭 |
| 종료점 | 쇼츠 결과물 확인 화면 |
| FL-02와의 차이 | 영상 소스가 YouTube URL이 아닌 로컬 파일 |

---

## 상세 플로우

```
┌─────────────────────────────────────────────┐
│  STEP 1: '파일 업로드' 탭 클릭              │
│  - Hero 섹션 오른쪽 탭                      │
│  - 업로드(↑) 아이콘 + '파일 업로드' 텍스트  │
│  - 클릭 시 탭 활성화 (그라디언트 배경)       │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 2: 파일 선택 영역 표시                │
│  - 드래그 앤 드롭 영역 표시                  │
│  - '파일을 여기에 끌어다 놓거나              │
│     클릭하여 선택하세요' 안내 텍스트          │
└────────────────────┬────────────────────────┘
                     │
         ┌───────────┴───────────┐
         │                       │
  [클릭으로 선택]         [드래그 앤 드롭]
         │                       │
  [파일 탐색기 열림]      [파일 드롭]
  [파일 선택 후 확인]           │
         │                       │
         └───────────┬───────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 3: 파일 유효성 검증                   │
│                                             │
│  지원 형식 여부 확인                         │
│  ├── 유효 → 업로드 시작                      │
│  └── 무효 → 오류 메시지                     │
│       '지원하지 않는 파일 형식입니다'         │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 4: 파일 업로드 진행                   │
│  - 업로드 진행률 표시 (프로그레스 바)        │
│  - 파일명 + 업로드 % 표시                    │
│  - 취소 버튼 제공                            │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 5: 업로드 완료                        │
│  - 파일 정보 표시 (파일명, 재생시간, 용량)   │
│  - 이용권 예상 차감량 안내                   │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 6 ~ 9: FL-02와 동일                   │
│  - 이용권 잔량 확인                          │
│  - 구간 설정 (선택)                          │
│  - AI 처리 (로딩)                            │
│  - 결과물 확인 화면                          │
└─────────────────────────────────────────────┘
```

## FL-02(유튜브 링크)와 비교

| 항목 | FL-02 유튜브 링크 | FL-03 파일 업로드 |
|------|-------------------|-------------------|
| 소스 | YouTube URL | 로컬 동영상 파일 |
| 입력 방식 | URL 텍스트 입력 | 파일 선택 or 드래그 앤 드롭 |
| 업로드 단계 | 없음 (URL로 직접 처리) | 파일 업로드 단계 존재 |
| 파일 유효성 | URL 유효성 검증 | 파일 형식 및 크기 검증 |
| AI 처리 이후 | 동일 | 동일 |

## 오류 케이스

| 오류 상황 | 표시 메시지 | 대응 |
|-----------|------------|------|
| 지원하지 않는 형식 | 지원하지 않는 파일 형식입니다 | 지원 형식 안내 |
| 파일 크기 초과 | 파일 크기가 너무 큽니다 | 크기 제한 안내 |
| 업로드 실패 | 업로드 중 오류가 발생했습니다 | 재시도 버튼 |
| 이용권 부족 | 이용권이 부족합니다 | 요금제 페이지 이동 |
