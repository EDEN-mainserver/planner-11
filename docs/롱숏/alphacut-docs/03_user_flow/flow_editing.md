## 플로우 개요

| 항목 | 내용 |
|------|------|
| 플로우 ID | FL-04 |
| 시작점 | 결과물 확인 화면 → '편집' 버튼 클릭 |

## 편집 항목별 세부 플로우

### 자막 편집 세부 플로우
```
자막 탭 클릭
  → 자막 목록 표시 (타임코드 + 텍스트)
  → 수정할 자막 클릭
  → 인라인 텍스트 편집
  → 타임코드 조정 (필요 시)
  → 인접 자막 병합 (필요 시)
  → 스타일 일괄 적용
  → 캔버스에서 실시간 확인
```

### 템플릿 변경 세부 플로우
```
템플릿 버튼 클릭
  → 템플릿 선택 패널 오픈
  → SandPaper / Minimal / Vibrant / GreenLine / Newsletter 중 선택
  → 저장된 커스텀 템플릿 목록도 표시
  → 클릭 시 캔버스 즉시 반영
  → 마음에 들면 확정, 아니면 다른 것 선택
```
