# 템플릿 기능 명세 (Feature: Template)

> 파일 위치: alphacut-docs/01_feature/feature_template.md  
> 기능 ID: F-D  
> 작성일: 2026-04-16

---

## 개요

미리 디자인된 쇼츠 템플릿을 선택하거나 사용자가 직접 편집 설정을 저장하여
매번 반복 설정 없이 일관된 비주얼의 쇼츠를 빠르게 생성하는 기능이다.

---

## F-D-01. 제공 템플릿 선택

### 개요
AlphaCut이 사전 제작한 디자인 템플릿을 선택하여 쇼츠에 일괄 적용한다.

### 제공 템플릿 목록

| 템플릿명 | 스타일 설명 | 특징 |
|----------|-------------|------|
| SandPaper | 트렌디한 배경 디자인 | 거친 질감의 배경, 현대적 느낌 |
| Minimal | 깔끔한 미니멀 스타일 | 여백 강조, 텍스트 중심 |
| Vibrant | 생동감 있는 컬러풀 디자인 | 강한 색감, 에너지 넘치는 느낌 |
| GreenLine | 네온 그린 라인 스타일 | 형광 그린 라인 포인트 |
| Newsletter | 뉴스레터 스타일 디자인 | 정보 전달형, 텍스트 레이아웃 강조 |

### 템플릿 적용 요소
| 요소 | 설명 |
|------|------|
| 배경 디자인 | 영상 외 영역의 배경 그래픽 |
| 자막 스타일 | 폰트, 색상, 위치, 크기 |
| 레이아웃 구성 | 영상 영역과 텍스트 영역 배치 |
| 로고 이미지 | 채널 로고 삽입 위치 및 스타일 |

### 동작 방식
1. AI 쇼츠 생성 시 또는 편집 화면에서 템플릿 선택
2. 선택 즉시 쇼츠 전체에 해당 템플릿 스타일 적용
3. 미리보기로 결과 확인 후 다른 템플릿으로 전환 가능

### 지원 플랜
- 전 플랜 지원 (무료 포함)

## F-D-02. 사용자 정의 템플릿 저장

### 개요
사용자가 직접 설정한 편집 옵션(자막 스타일, 레이아웃, 배경 등)을
커스텀 템플릿으로 저장하여 이후 프로젝트에서 재사용한다.

### 저장 가능 설정 항목
| 항목 | 설명 |
|------|------|
| 자막 스타일 | 폰트, 크기, 색상, 위치 |
| 영상 레이아웃 | 화면 비율, 영상 영역 크기 및 위치 |
| 배경 디자인 | 커스텀 배경 색상 또는 이미지 |
| 커스텀 폰트 | 업로드된 폰트 포함 (Pro 이상) |
| 로고 위치 | 채널 로고 삽입 위치 |

### 동작 방식
1. 편집 화면에서 원하는 스타일 설정 완료
2. "템플릿으로 저장" 버튼 클릭
3. 템플릿 이름 입력 후 저장
4. 다음 프로젝트 시작 시 저장된 템플릿 목록에서 선택 적용

### 목적
- 쇼츠 제작 시간 단축 (효율 200% 향상 — 공식 표현)
- 채널 브랜딩 일관성 유지
- 매번 같은 설정 반복 제거

### 지원 플랜
- 전 플랜 지원
