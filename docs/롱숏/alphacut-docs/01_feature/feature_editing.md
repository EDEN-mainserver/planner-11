# 편집 기능 명세 (Feature: Editing)

> 파일 위치: alphacut-docs/01_feature/feature_editing.md  
> 기능 ID: F-C  
> 작성일: 2026-04-16

---

## 개요

AI가 생성한 쇼츠 결과물에 사용자가 직접 디테일을 추가하거나 수정할 수 있는 편집 도구 모음이다.
"AI가 편집을 잘할 수 있을까?" 라는 사용자 불안을 해소하기 위해 제공된다.

---

## F-C-01. 텍스트 편집

### 개요
쇼츠에 삽입된 제목 텍스트 및 자막 텍스트를 수정한다.

### 편집 가능 항목
| 항목 | 설명 |
|------|------|
| 텍스트 내용 | 직접 타이핑으로 수정 |
| 폰트 종류 | 기본 제공 폰트 선택 |
| 폰트 크기 | 슬라이더 또는 직접 입력 |
| 글자 색상 | 컬러 피커로 선택 |
| 텍스트 위치 | 드래그로 이동 (추정) |
| 텍스트 정렬 | 좌/중/우 정렬 |

### 대상 텍스트 요소
- 영상 상단 제목 텍스트
- 하단 자막 텍스트
- 추가 텍스트 레이어 (있을 경우)

### 지원 플랜
- 전 플랜 지원 (기본 편집 기능에 포함)

## F-C-02. 레이아웃 편집

### 개요
쇼츠의 화면 구성 방식(레이아웃)을 변경한다.
영상 영역과 자막/텍스트 영역의 배치를 조정한다.

### 편집 가능 항목
| 항목 | 설명 |
|------|------|
| 영상 영역 크기 | 화면 내 영상이 차지하는 비율 조정 |
| 텍스트 영역 위치 | 상단 / 하단 / 오버레이 |
| 배경 구성 | 템플릿 배경과의 결합 방식 |
| 영상 위치 | 화면 내 영상 위치 조정 (위/중앙/아래) |

### 관련 기능
- 템플릿 선택 시 레이아웃 일괄 적용
- 비율(Aspect Ratio) 조절 기능과 연동

### 지원 플랜
- 전 플랜 지원 (기본 편집 기능에 포함)

## F-C-03. 길이 및 비율 조절

### 개요
생성된 쇼츠의 재생 시간과 화면 비율을 조정한다.

### 편집 가능 항목
| 항목 | 설명 |
|------|------|
| 재생 시간(길이) | 시작/끝 지점 조절로 클립 길이 변경 |
| 화면 비율 | 9:16 (기본), 1:1, 16:9 등 선택 (추정) |

### 제약 조건
- 쇼츠 플랫폼(유튜브 Shorts, TikTok) 표준은 9:16
- 길이 조절 시 AI 추출 구간 범위 내에서 변경 가능

### 지원 플랜
- 전 플랜 지원

## F-C-04. 컷 편집 (타임라인 편집)

### 개요
타임라인 UI를 활용하여 영상의 특정 구간을 삽입하거나 삭제하고, 배속을 조정한다.

### 기능 상세
| 기능 | 설명 |
|------|------|
| 구간 삭제(컷) | 타임라인에서 불필요한 구간 선택 후 제거 |
| 구간 삽입 | 특정 위치에 클립 추가 |
| 배속 조절 | 1x, 1.5x, 2x 등 재생 속도 변경 |
| 타임라인 이동 | 좌우 스크롤로 클립 내 탐색 |

### 목적
- 무음 구간 제거 기능의 수동 보완
- 세밀한 편집 포인트 조정
- 영상 리듬감(속도감) 살리기

### 지원 플랜
- 전 플랜 지원 (기본 편집 기능에 포함)

## F-C-05. 커스텀 폰트 업로드

### 개요
사용자가 원하는 폰트 파일을 직접 업로드하여 쇼츠 자막에 적용한다.

### 동작 방식
1. 편집 화면 내 폰트 설정 메뉴 접근
2. "폰트 업로드" 버튼 클릭
3. 로컬 폰트 파일(.ttf, .otf 등) 선택
4. 업로드 완료 후 자막에 적용

### 목적
- 브랜드 아이덴티티 적용 (채널 고유 폰트)
- 기본 제공 폰트 외 자유로운 디자인

### 제약 조건
- **Pro 멤버십 이상** 전용 기능
- 지원 폰트 파일 형식: 공개 명시 없음 (.ttf/.otf 추정)

### 지원 플랜
| 무료 | Basic | Pro | Max |
|------|-------|-----|-----|
| ❌ | ❌ | ✅ | ✅ |

## F-C-06. 썸네일 설정

### 개요
쇼츠의 대표 이미지(썸네일)를 설정한다.

### 설정 방법
| 방법 | 설명 |
|------|------|
| 영상 프레임 선택 | 영상 내 특정 장면을 썸네일로 지정 |
| 이미지 업로드 | 별도 이미지 파일을 썸네일로 업로드 (추정) |

### 목적
- 플랫폼 업로드 시 클릭률 향상
- 채널 통일감 있는 비주얼 관리

### 지원 플랜
- 전 플랜 지원
