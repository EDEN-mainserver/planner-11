## F-J-01. CS 챗봇

### 개요
서비스 내 모든 페이지에서 접근 가능한 실시간 고객 지원 챗봇이다.

### UI 위치
- 모든 페이지 우하단 고정 플로팅 버튼
- 파란색 원형 채팅 아이콘

### 진입 방법
- 우하단 채팅 아이콘 클릭 → 챗봇 대화창 오픈

### 초기 메시지
> "알파컷에 오신 것을 환영합니다! 궁금한 점이 있으신가요?"

### 기능
| 기능 | 설명 |
|------|------|
| 자동 응답 | 자주 묻는 질문 자동 답변 |
| 상담 연결 | 복잡한 문의 시 담당자 연결 (추정) |
| 말풍선 알림 | 페이지 접속 시 자동 말풍선 노출 (닫기 버튼 있음) |

## F-J-02. FAQ (자주 묻는 질문)

### 개요
홈 페이지 하단에 위치한 아코디언 방식의 FAQ 섹션이다.

### 위치
- 홈 페이지(/) 하단 섹션
- 상단 네비게이션 "FAQ" 앵커 링크 (#faq-title)

### UI 방식
- 아코디언 (accordion): 질문 클릭 시 답변 펼침/접힘
- 한 번에 하나 또는 여러 개 열기 가능

### FAQ 항목 전체 목록

| 번호 | 질문 | 답변 요약 |
|------|------|-----------|
| 1 | AI는 어떤 기준으로 쇼츠를 제작하나요? | 핵심 하이라이트 구간 자동 추출 (관심 유도 장면, 중요 정보, 몰입도 높은 순간) |
| 2 | 이용요금은 어떻게 책정되나요? | 원본 영상 길이(분) 단위로 이용권 차감. 10분 영상 = 10분 차감 |
| 3 | 쇼츠는 몇 개 생성되나요? | 원본 영상 2분당 1개 생성. 10분 → 약 5개 (편차 있음) |
| 4 | 생성된 쇼츠를 수정하거나 재생성할 수 있나요? | 재생성 가능 (이용권 50% 소모), 편집 기능으로 수동 수정도 가능 |
| 5 | 여러 명이 이용권을 공유하는 팀 계정을 지원하나요? | 지원. 팀 계정 관련 문의: teamweexe@gmail.com |
| 6 | 해외에서 거주 중인데 이용할 수 있나요? | 가능. 해외 거주자는 계좌이체 방식 결제. 문의: teamweexe@gmail.com |
