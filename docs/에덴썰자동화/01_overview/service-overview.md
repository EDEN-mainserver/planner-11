### F-001. 인증 (Authentication)

| 항목 | 내용 |
|------|------|
| 방식 | Google OAuth 2.0 |
| 진입 경로 | 에디터 상단 `로그인/가입` 버튼 클릭 |
| 트리거 조건 | 로그인 필요 기능 사용 시 자동 팝업 |
| 모달 구성 | SunsakTool 로고 + 안내 문구 + `Google 계정으로 시작하기` 버튼 + X 닫기 |
| 비회원 상태 | 에디터 접근 가능, 로그인 필요 기능 사용 불가 |
| 회원 상태 표시 | 상단 우측 프로필 아바타 이미지로 로그인 상태 표시 |

### F-016. AI 대본 자동 생성 (AI Script Generation)

| 항목 | 내용 |
|------|------|
| 진입 | 우측 패널 `AI 대본 생성` 버튼 |
