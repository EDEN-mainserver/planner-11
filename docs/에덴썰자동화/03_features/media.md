# 미디어 관리 기능 명세

> **문서 경로**: `sunsaktool-docs/03_features/media.md`
> **작성일**: 2026-04-16

---

## 1. 이미지 업로드

| 항목 | 내용 |
|------|------|
| 버튼명 | `미디어 추가` |
| 위치 | 각 카드 하단 버튼 행 |
| 방식 | 파일 선택 다이얼로그 (file input) |
| 지원 형식 | JPG, PNG, GIF, WebP (투명 배경 PNG 포함) |
| 삽입 위치 | 클릭한 해당 카드의 이미지 슬롯 |

---

## 2. 이미지 붙여넣기 (Ctrl+V)

| 항목 | 내용 |
|------|------|
| 트리거 | 키보드 Ctrl+V |
| 전제 조건 | 구글 이미지 검색 등에서 `이미지 복사` 후 사용 |
| 동작 | 클립보드 이미지 > 현재 포커스된 카드에 즉시 삽입 |
| 안내 | 에디터 우측 패널 상단에 상시 안내 박스 노출 |

---

## 3. GIF 검색 & 삽입

| 항목 | 내용 |
|------|------|
| 버튼명 | `GIF 찾기` |
| 위치 | 각 카드 하단 버튼 행 |
| 지원 템플릿 | 썰 / 지식 템플릿 |

### GIF 검색 모달 구성

| 요소 | 설명 |
|------|------|
| 검색창 | 키워드 입력 (예: "박명수", "무한도전") |
| 검색 버튼 | 검색 실행 |
| 결과 그리드 | GIF 썸네일 목록 표시 |
| 삽입 동작 | GIF 썸네일 클릭 > 해당 카드에 즉시 삽입 |

---

## 4. 출처 추가

| 항목 | 내용 |
|------|------|
| 버튼명 | `출처 추가` |
| 위치 | 각 카드 하단 버튼 행 |
| 동작 | 텍스트 입력 > 카드 하단에 출처 표시 |
| 목적 | 뉴스/정보 콘텐츠 출처 명시 |

---

## 5. 배경 제거 이미지 처리

| 항목 | 내용 |
|------|------|
| 지원 형식 | 투명 PNG |
| 처리 방식 | 투명 영역 유지하여 렌더링 |
| 과거 버그 | 배경 제거 이미지 초록색/검은색 오류 (2025-02-04 패치 완료) |
