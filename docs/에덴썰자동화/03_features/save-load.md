# 저장 / 불러오기 / 프로젝트 관리 기능 명세

> **문서 경로**: `sunsaktool-docs/03_features/save-load.md`
> **작성일**: 2026-04-16

---

## 1. 임시 저장 (Auto Save)

| 항목 | 내용 |
|------|------|
| 저장 위치 | 브라우저 로컬 스토리지 |
| 지원 템플릿 | 썰, 지식, 커뮤니티 |
| 트리거 | 편집 중 자동 저장 |
| 복원 | 다음 접속 시 자동 복원 |
| 목적 | 브라우저 종료/새로고침 시 작업 내용 보호 |

---

## 2. 수동 저장

| 항목 | 내용 |
|------|------|
| 버튼명 | `💾 저장` |
| 위치 | 우측 스크립트 패널 상단 버튼 행 |
| 동작 | 현재 작업 상태 저장 |

---

## 3. 불러오기

| 항목 | 내용 |
|------|------|
| 버튼명 | `📂 불러오기` |
| 위치 | 우측 스크립트 패널 상단 버튼 행 |
| 버튼 색상 | 다크/옐로 |
| 동작 | 저장된 프로젝트 목록 조회 후 선택 복원 |

---

## 4. 새 프로젝트 시작

| 항목 | 내용 |
|------|------|
| 버튼명 | `새 프로젝트 시작` |
| 위치 | 우측 스크립트 패널 상단 버튼 행 |
| 동작 | 현재 작업 초기화 후 새 프로젝트 시작 |
| 주의 | 저장하지 않은 작업은 초기화됨 |

---

## 5. 수동 카드 추가

| 항목 | 내용 |
|------|------|
| 버튼명 | `+ 수동 추가` |
| 위치 | 우측 스크립트 패널 상단 버튼 행 (레드) |
| 동작 | 빈 카드를 마지막 위치에 추가 |
