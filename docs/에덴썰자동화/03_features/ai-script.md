## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| 기능명 | AI 대본 자동 생성 (고도화) |
| 버튼명 | `AI 대본 생성 (고도화)` |
| 위치 | 우측 스크립트 패널 상단 |
| 버튼 색상 | 블루 그라데이션 (전체 너비) |
| 목적 | 주제 키워드 입력 > 카드 단위 대본 자동 생성 |

---

## 2. 모달 구성

### 모달 제목

- `AI로 자동 대본 생성`

### 모달 내부 요소

| 요소 | 타입 | 설명 |
|------|------|------|
| 안내 문구 | text | "만들고 싶은 썰의 주제나 키워드를 입력하고 'AI로 생성하기' 버튼을 누르세요." |
| 주제 입력창 | textarea | placeholder: "예시) 헬스장에서 본 역대급 진상 썰" |
| AI로 썰 풀기 버튼 | button (블루, 좌측) | 입력 주제로 대본 생성 요청 |
| 기록 보기 버튼 | button (다크, 우측) | 이전 생성 기록 조회 |
| 생성 결과 영역 | textarea (읽기 전용) | placeholder: "AI가 생성한 제목과 스크립트가 여기에 표시됩니다..." |
| 취소 버튼 | button (레드) | 모달 닫기 |
| 스크립트만 추가하기 버튼 | button (레드) | 생성된 대본을 카드로 삽입 |

---

## 3. 동작 플로우

```text
[AI 대본 생성 버튼 클릭]

        ↓

[모달 팝업]

        ↓

[주제/키워드 입력] (textarea)

        ↓

[AI로 썰 풀기 클릭]

        ↓

[API 요청 > AI 대본 생성 중...]

        ↓

[생성 결과 textarea에 제목 + 스크립트 표시]

        ├─ 스크립트만 추가하기 > 카드 패널에 자동 삽입 (카드 분리)
        └─ 취소 > 모달 닫기 (기존 작업 유지)
```

---

## 4. 세부 동작 규칙

| 규칙 | 내용 |
|------|------|
| 카드 자동 분리 | 생성된 대본을 글자 수 기반으로 카드별 줄 수 자동 최적화 |
| 도입부 생성 | 시청자 즉시 몰입 유도 구조로 AI 생성 |
| 기존 카드 유지 | `스크립트만 추가하기` 클릭 시 기존 카드 아래에 추가 삽입 |
| 기록 보기 | 이전 생성 이력 열람 가능 |
| 사용 횟수 제한 | 운영 정책에 따라 사용 횟수 제한 가능 |
