# 글감 찾기 기능 명세

> **문서 경로**: `sunsaktool-docs/03_features/content-idea.md`
> **작성일**: 2026-04-16

---

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| 기능명 | 글감 찾기 (콘텐츠 아이디어) |
| 버튼명 | `💡 글감 찾기` |
| 위치 | 우측 스크립트 패널 AI 버튼 행 |
| 버튼 색상 | 블루 |
| 목적 | 콘텐츠 소재가 없을 때 실시간 인기 소재 추천 |

---

## 2. 모달 구성

### 모달 제목

- `콘텐츠 아이디어, 여기서 찾아보세요! 💡`

### 안내 문구

- "무엇을 만들어야 할지 막막하신가요? 지금 대한민국에서 가장 인기 있는 이야기들을 참고하여 당신만의 콘텐츠를 만들어보세요!"

---

## 3. 카테고리 및 외부 링크

| 카테고리 | 설명 | 제공 링크 |
|---------|------|----------|
| 🔥 실시간 종합 랭킹 뉴스 | 최신 트렌드/사건 소재 | 네이버 랭킹 뉴스, 네이트 랭킹 뉴스, JTBC 속보 랭킹 |
| 💼 경제/비즈니스 트렌드 | 재테크, 산업, 비즈니스 인사이트 | 한국경제 랭킹 뉴스 |
| 🎭 연예/엔터테인먼트 이슈 | 핫한 연예계 소식 | 다음 연예 랭킹 |
| 💬 사람들의 생생한 '썰' 모음 | 커뮤니티 경험담/이야기 | 네이트 썰, 썰 모음(ssul.kr) |

---

## 4. 동작 규칙

| 규칙 | 내용 |
|------|------|
| 링크 동작 | 클릭 시 새 탭으로 외부 사이트 열림 |
| 직접 삽입 없음 | 링크만 제공, 직접 대본 삽입 기능 없음 |
| 닫기 | `닫기` 버튼으로 모달 종료 |
