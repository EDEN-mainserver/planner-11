## CASE 1. 썰 템플릿으로 영상 제작

```
[썰 템플릿 선택]
│
▼
[에디터 진입 (/app?template=story)]
│
▼
[좌측 패널 채널 설정]
헤더 텍스트: "ㅇㅇ 채널" 입력
배경 색상: #E82C43 (기본) 또는 변경
폰트: G마켓 산스 (기본) 또는 변경
│
▼
[프로젝트 제목 입력]
예: "헬스장에서 본 역대급 진상 썰"
│
▼
[AI 대본 생성 or 직접 입력]
✨ AI 대본 생성:
→ "헬스장에서 본 역대급 진상 썰" 입력
→ "AI로 썰 풀기" 클릭
→ 생성 결과 확인
→ "스크립트만 추가하기"
→ 카드 자동 생성됨
│
▼
[카드별 미디어 추가]
각 카드마다:
→ "미디어 추가" 클릭 or Ctrl+V
→ AI 이미지: "헬스장에서 운동하는 남성" 묘사 입력
→ 생성 이미지 삽입
│
▼
[카드별 효과음 설정]
중요 장면: "두둥탁!" 또는 "깜짝 놀라는 전자음" 선택
전환 장면: "화면전환효과음" 계열 선택
│
▼
[전체 TTS 설정]
목소리: 표준 남성 1
속도: 1.35x
"모두 적용" 클릭
│
▼
[BGM 선택]
"spider" 또는 "laughter-in-the-breeze" 선택
볼륨: 20
│
▼
[▶ 미리보기 재생]
전체 영상 확인
어색한 카드 수정
│
▼
[🎬 영상 저장]
→ MP4 다운로드
→ YouTube Shorts 업로드
```

## CASE 2. 유튜브 댓글 템플릿으로 영상 제작

```
[유튜브 댓글 템플릿 선택]
│
▼
[에디터 진입]
│
▼
[YouTube 영상 URL 입력]
예: https://youtube.com/watch?v=xxxxxxx
(또는 공유 단축 URL 지원)
│
▼
[AI 인기 댓글 자동 추출]
타임스탬프 포함 댓글 자동 제외
닉네임 자동 마스킹 처리
배경 영상 자동 조합
│
▼
[추출된 댓글 카드 확인]
불필요한 댓글 카드 삭제
순서 드래그로 재배치
│
▼
[BGM 및 효과음 설정]
│
▼
[▶ 미리보기 확인]
│
▼
[🎬 영상 저장 → MP4 다운로드]
```

## CASE 3. AI 대본 생성 → AI 이미지 생성 풀 AI 플로우

```
[썰/지식 템플릿 선택]
│
▼
[에디터 진입]
│
▼
[✨ AI 대본 생성 클릭]
주제 입력: "도서관 책에서 돈 나온 썰"
│
▼
[AI 대본 자동 생성]
제목 + 카드별 스크립트 자동 완성
글자 수 기반 카드 최적화 자동 적용
│
▼
[카드 1부터 순서대로 AI 이미지 생성]
"미디어 추가" → AI 이미지 탭
각 카드 상황 묘사 입력:
  카드1: "조용한 도서관 내부, 나무 책장"
  카드2: "오래된 책 사이에서 돈봉투 발견"
  카드3: "놀란 표정의 대학생"
→ 생성 → 삽입
│
▼
[전체 TTS 일괄 설정]
목소리 + 속도 → "모두 적용"
│
▼
[카드별 효과음 설정]
│
▼
[BGM 선택]
│
▼
[▶ 미리보기 → 수정 → 저장]
```

## CASE 4. 기존 작업 이어서 하기 (임시 저장 복원)

```
선택 → 에디터에 로드
│
▼
[이어서 편집 → 저장]
```

## CASE 5. 글감 찾기 → 대본 작성 → 영상 완성

```
[에디터 진입]
│
▼
[💡 글감 찾기 클릭]
모달에서 카테고리별 소재 탐색:
  🔥 실시간 종합 랭킹 뉴스
  💼 경제/비즈니스 트렌드
  🎭 연예/엔터테인먼트 이슈
  💬 사람들의 생생한 썰 모음
│
▼
[외부 링크 클릭 → 새 탭에서 소재 탐색]
예: 네이트 랭킹 뉴스에서 화제 사건 확인
│
▼
[소재 파악 후 모달 닫기]
│
▼
[✨ AI 대본 생성에 주제 입력]
파악한 소재를 주제로 입력
│
▼
[이후 CASE 1 플로우와 동일]
미디어 추가 → 사운드 설정 → 저장
```

## 2. 에디터 내 세부 인터랙션 플로우

### 2-1. 카드 더블클릭 편집 플로우

```
[카드 텍스트 영역 더블클릭]
│
▼
[줄 단위 편집 모드 진입]
│
▼
[줄별 순서 변경 or 효과음 개별 설정]
│
▼
[편집 완료 → 클릭 아웃으로 모드 종료]
```

### 2-2. GIF 찾기 플로우

```
[카드 내 "GIF 찾기" 클릭] (🔒 유료 필요)
│
▼
[GIF 검색 모달 팝업]
│
▼
[키워드 입력] 예: "무한도전", "박명수"
│
▼
[검색 결과 GIF 목록 표시]
│
▼
[원하는 GIF 썸네일 클릭]
│
▼
[해당 카드에 GIF 즉시 삽입]
[모달 자동 닫힘]
```

### 2-3. 이미지 Ctrl+V 플로우

```
[구글 이미지 검색]
│
▼
[이미지 우클릭 → "이미지 복사"]
│
▼
[에디터 탭으로 전환]
│
▼
[Ctrl+V 입력]
│
▼
[현재 포커스된 카드에 이미지 즉시 삽입]
```

### 2-4. TTS 미리듣기 플로우

```
[카드 내 "▶ 길이(자동)" 버튼 클릭]
│
▼
[해당 카드 텍스트 TTS 음성 재생]
[재생 시간(초) 자동 계산 후 표시]
│
▼
[음성 확인 후 목소리/속도 조정]
│
▼
[다시 미리듣기 → OK 확인]
```

### 2-5. 템플릿 전환 플로우

```
[에디터 내 상단 탭 버튼 클릭]
예: "지식 템플릿" 탭 클릭
│
▼
[현재 작업 내용 유지 여부 확인]
│
▼
[지식 템플릿 레이아웃으로 즉시 전환]
[캔버스 디자인 변경]
```

## 3. 에러 및 예외 플로우

| 상황 | 발생 조건 | 대응 |
|------|----------|------|
| 로그인 필요 팝업 | 잠금 기능 클릭 | Google 로그인 모달 → 인증 후 재시도 |
| PC 권장 팝업 | 모바일 접속 | "불편함을 감수하고 계속하기" → 진입 |
| 영상 저장 실패 | 서버 오류 | 자동 횟수 복구 + 재시도 |
| AI 대본 생성 횟수 초과 | 월 제한 초과 | 플랜 업그레이드 안내 |
| AI 이미지 생성 횟수 초과 | 월 제한 초과 | 플랜 업그레이드 안내 |
| 임시 저장 없음 | 최초 접속 | 빈 에디터로 시작 |
| 유튜브 댓글 추출 실패 | 잘못된 URL / 비공개 영상 | 에러 메시지 표시 |
