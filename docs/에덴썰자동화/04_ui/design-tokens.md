## 1. CSS 변수 (Custom Properties)

> 실제 코드에서 추출한 값 (`--st-*` prefix 사용)

| 변수명 | 값 | 용도 |
|--------|-----|------|
| `--st-accent-color` | `#e82c43` | 주요 강조색 (레드), CTA 버튼, 활성 탭, 카드 테두리 |
| `--st-bg-color` | `#1e1e1e` | 전체 배경색 |
| `--st-panel-color` | `#2a2a2a` | 패널/탭 배경색 |
| `--st-border-color` | `#383838` | 기본 테두리 색상 |
| `--st-text-color` | `#e0e0e0` | 기본 텍스트 색상 |
| `--st-subtext-color` | `#aaa` | 보조 텍스트 색상 |
| `--st-interact-color` | `#4a90e2` | 인터랙션 강조색 (블루), AI 버튼, 링크 |

## 2. 색상 팔레트

### 2-1. 브랜드 색상

| 이름 | HEX | RGB | 용도 |
|------|-----|-----|------|
| 브랜드 레드 | `#e82c43` | rgb(232, 44, 67) | 메인 액센트, CTA 버튼, 활성 상태 |
| 브랜드 블루 | `#4a90e2` | rgb(74, 144, 226) | AI 기능 버튼, 인터랙션, API 섹션 |
| 브랜드 핑크 | 핑크/퍼플 그라데이션 | - | AI 이미지 생성 버튼 |
| 브랜드 옐로 | `#f0c040` 계열 | - | 안내 배지, 보조 강조 요소 |
