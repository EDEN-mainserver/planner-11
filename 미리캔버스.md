## 1. 서비스 개요

미리캔버스는 **웹 기반 그래픽 디자인 툴**로, 비전문가도 쉽게 고품질 디자인을 제작할 수 있는 SaaS 플랫폼이다.

| 항목 | 내용 |
|------|------|
| 서비스 유형 | 웹 기반 디자인 편집기 (Canva 유사) |
| 주요 언어 | 한국어 (다국어 지원: 180개국) |
| 월간 활성 사용자 | 200만 명 이상 |
| 제공 템플릿 수 | 53만 개 이상 |
| 다운로드 수 | 4억 건 이상 |
| 지원 플랫폼 | 웹(PC), 태블릿, 모바일 (앱 제공) |
| 요금 모델 | Freemium (무료 + Pro + 기업용 + 교육용) |


## 3. 유저 플로우 (User Flow)

### 3.1 신규 사용자 온보딩 플로우
```
[랜딩 페이지 방문]
        │
        ▼
[바로 시작하기 CTA 클릭] ──────────────── [템플릿 보러가기 클릭]
        │                                         │
        ▼                                         ▼
[회원가입/로그인 팝업]                    [템플릿 탐색 페이지]
  ├── 이메일 회원가입                            │
  ├── 소셜 로그인 (Google/Kakao 등)             │
  └── 기존 계정 로그인                           │
        │                                         │
        ▼                                         ▼
[디자인 에디터 진입]  ◄───────────────[템플릿 선택 → 에디터 진입]
```

### 3.2 디자인 제작 플로우
```
[에디터 진입]
        │
        ├── [빈 캔버스 시작] ──→ [크기 설정] ──→ [요소 추가]
        │
        └── [템플릿 선택 시작]
                │
                ▼
        [템플릿 패널에서 검색/탐색]
                │
                ▼
        [템플릿 클릭 → 캔버스 적용]
                │
                ▼
        ┌──────────────────────────────────┐
        │          편집 작업               │
        │  ├── 텍스트 추가/편집            │
        │  ├── 이미지/사진 삽입            │
        │  ├── 요소(도형/일러스트) 추가    │
        │  ├── 배경 변경                   │
        │  ├── 색상/폰트 조정              │
        │  ├── 애니메이션 적용             │
        │  └── 페이지 추가/복제/삭제       │
        └──────────────────────────────────┘
                │
                ▼
        [저장 (자동저장 + 수동저장)]
                │
                ▼
        ┌────────────────┐
        │  출력 방식 선택 │
        ├────────────────┤
        │  다운로드       │──→ [PNG/JPG/PDF/MP4/GIF 선택] → [파일 다운로드]
        │  공유하기       │──→ [링크 복사 / SNS 공유]
        │  슬라이드쇼 보기│──→ [프레젠테이션 모드]
        │  HTML 코드 삽입 │──→ [임베드 코드 복사]
        │  템플릿 만들기  │──→ [크리에이터 등록]
        │  인쇄물/상품 제작│──→ [프린트허브 연동]
        └────────────────┘
```

### 3.3 AI 도구 활용 플로우
```
[AI 도구 패널 진입]
        │
        ├── [AI 이미지 생성]
        │       └── 텍스트 프롬프트 입력 → 이미지 생성 → 캔버스 삽입
        │
        ├── [AI 프레젠테이션]
        │       └── 주제 입력 → AI 초안 생성 → 편집
        │
        ├── [배경 제거]
        │       └── 이미지 업로드 → 자동 배경 제거 → 적용
        │
        ├── [이미지 편집 (이미지 레이어 분리)]
        │       └── 이미지 업로드 → AI 요소 분리 → 텍스트/요소 수정
        │
        └── [AI 텍스트 도구]
                ├── 맞춤법 검사
                ├── 내용 정돈하기
                ├── 이어쓰기
                ├── 요약하기
                └── 다시 쓰기
```

### 3.4 협업 플로우
```
[프로젝트 패널 진입]
        │
        ├── [내 드라이브] ──→ 개인 디자인 관리
        ├── [팀 드라이브] ──→ 팀원과 공유된 디자인 공동 편집
        ├── [공유받은 디자인] ──→ 공유 링크로 받은 작업물 확인
        └── [즐겨찾기] ──→ 자주 사용하는 디자인 빠른 접근
```

## 4. 화면별 기능 명세

### 4.1 랜딩 페이지 (홈)

#### 레이아웃 구성

| 섹션 | 설명 |
|------|------|
| 서비스 상단 배너 | 미리캔버스 / 디자인허브 / 프린트허브 / 비즈하우스 탭 네비게이션 |
| GNB | 로고, 템플릿(드롭다운), 추천 기능(드롭다운), 기업용, 교육용, 요금제, miricle AI, 도움말, 디자인 만들기 버튼, 로그인 |
| 히어로 섹션 | 메인 카피 + 서브 카피 + CTA 2개(바로 시작하기, 템플릿 보러가기) |
| 소셜 프루프 | 월간 활성 사용자, 서비스 국가, 다운로드 수, 템플릿 수 지표 표시 |
| 템플릿 카테고리 | 프레젠테이션, 동영상, 소셜 미디어, 포스터, 상세페이지, 유튜브 썸네일 카드 |
| AI 기능 소개 | AI 프레젠테이션, AI 이미지 생성, AI 라이팅, 배경 제거, 영역 지우개, 화질 개선 |
| 디자인 툴 소개 | 1,000만 개 이상의 요소, 애니메이션 효과, 멀티 디바이스 지원 |
| 앱 다운로드 | iOS / Android 링크 |
| 푸터 | 회사 정보, 이용약관, 개인정보처리방침, 고객센터 |

#### 기능 명세

| 기능 ID | 기능명 | 설명 | 우선순위 |
|---------|--------|------|----------|
| LP-001 | 히어로 CTA | "바로 시작하기" 클릭 시 로그인/가입 후 에디터 진입 | 필수 |
| LP-002 | 템플릿 CTA | "템플릿 보러가기" 클릭 시 /ko/templates로 이동 | 필수 |
| LP-003 | GNB 고정 | 스크롤 시 상단 내비게이션 고정(sticky) | 필수 |
| LP-004 | 드롭다운 메뉴 | 템플릿/추천 기능 메뉴 hover 시 드롭다운 표시 | 필수 |
| LP-005 | 템플릿 캐러셀 | 카테고리별 템플릿 가로 슬라이드 (자동재생 옵션) | 중요 |
| LP-006 | 반응형 레이아웃 | PC / 태블릿 / 모바일 반응형 지원 | 필수 |
