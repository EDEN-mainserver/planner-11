## UF-03. 계정진단 — 인스타그램 연동  ← 전면 수정

```
[사이드바 "보고서 확인하기" 클릭]
        │
        ▼
[/engagement — 인스타그램 연동 페이지]
 - "인스타그램 연동을 통해 내 계정 진단을 이용할 수 있어요!" 안내

 - 장기간 미접속 시 재연동 필요 안내
        │
        ├─ [인스타그램 연동이 안되신다면 클릭!] 링크
        │         └→ 별도 안내 모달 or 페이지
        │
        └─ [📷 인스타그램 연동하기] 버튼 클릭
                  │
                  ▼
         [인스타그램 OAuth 인증]
          (인스타그램 로그인/권한 허용 팝업)
                  │
          ┌───────┴───────┐
         성공             실패/거부
          │               │
          ▼               ▼
   [계정 데이터 수집 중]  에러 안내
   (로딩 스피너)         → 재시도 or
          │               연동 불가 안내
          ▼
   [계정 진단 보고서]
    - 인스타그램 계정 분석 결과
    - 도달수/팔로워/인게이지먼트 등
    - (FREE 플랜: 제한적 접근, 유료 업그레이드 유도)
          │
          └─ [구독 및 결제] 클릭
                    └→ 결제 페이지 (현재 미구현)
```

## 에러 처리 플로우

```
[AI 생성 실패]
        │
        ├─ 네트워크 오류  → "요청에 실패했습니다. 다시 시도해주세요." + 재시도 버튼
        ├─ 사용량 초과   → "사용 횟수를 모두 사용했습니다." + 구독 안내
        └─ 서버 오류    → "일시적인 오류가 발생했습니다." + 재시도 버튼

[인스타그램 연동 실패]
        │
        ├─ OAuth 거부    → "인스타그램 연동이 필요합니다." 안내
        └─ 연동 오류    → "인스타그램 연동이 안되신다면 클릭!" 링크 안내
```
