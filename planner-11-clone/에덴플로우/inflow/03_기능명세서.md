### 랜딩 상태

| ID | 기능명 | 설명 | 우선순위 |

| F-05-2 | blur 미리보기 | "이런 형태로 제작해드려요!" 텍스트 + blur 처리된 방향성 리포트 예시 | P2 |

### 스텝 폼

| ID | 기능명 | 설명 | 우선순위 |

| F-05-4 | STEP 2 - 관심 주제 | 6개 선택지 + 직접 입력 | P0 |
| F-05-5 | STEP 3 - 운영 목적 | 퍼스널브랜딩/수익창출/홍보/취미기록 + 직접 입력 | P0 |

| F-05-8 | 이전 단계 버튼 | 이전 스텝 이동, 기입력값 유지 | P0 |
| F-05-9 | STEP N OF 4 표시 | 현재 단계 텍스트 표시 | P1 |
| F-05-10 | 작성 중인 정보 패널 | 우측 패널 항목별 실시간 표시 (완료 ✓) | P1 |

### AI 생성 및 결과

| ID | 기능명 | 설명 | 우선순위 |
|----|--------|------|----------|
| F-05-11 | 생성하기 버튼 | 클릭 시 `/api/ai/direction` POST 호출, 로딩 표시 | P0 |
| F-05-12 | 결과 렌더링 | "나의 인스타그램 방향성 리포트" 섹션 표시 | P0 |
| F-05-13 | 자동 보관함 저장 | 생성 완료 시 자동 저장 | P0 |
| F-05-14 | 사용 횟수 차감 | 생성 성공 시 방향성기획 카운트 +1 (FREE: 최대 3회) | P0 |
| F-05-15 | 사용 한도 초과 처리 | 3회 소진 시 생성 불가, 안내 메시지 표시 | P1 |

### API 명세

```
POST /api/ai/direction
Body: {
  nickname: string,
  topic: string,
  purpose: string,
  experience: "beginner" | "intermediate" | "advanced"
}
Response: {
  result: string  // AI 생성 방향성 리포트 마크다운
}
```
