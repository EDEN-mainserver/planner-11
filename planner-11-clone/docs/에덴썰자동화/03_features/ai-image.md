# AI 맞춤 이미지 생성 기능 명세

> **문서 경로**: `sunsaktool-docs/03_features/ai-image.md`
> **작성일**: 2026-04-16

---

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| 기능명 | AI 맞춤 이미지 생성 |
| 버튼명 | `🎨 AI 맞춤 이미지 생성` |
| 위치 1 | 우측 스크립트 패널 상단 (전체 이미지 생성) |
| 위치 2 | 카드 내 미디어 추가 > AI 이미지 탭 (카드별 삽입) |
| 버튼 색상 | 핑크/퍼플 그라데이션 |
| 목적 | 상황 묘사 텍스트 > 썰쇼츠 특화 일러스트 이미지 생성 |

---

## 2. 동작 플로우

```text
[AI 맞춤 이미지 생성 버튼 클릭]

        ↓

[이미지 생성 모달 팝업]

        ↓

[상황 묘사 텍스트 입력]
예: "대학교 교실에서 땀 흘리며 공부하는 여학생"

        ↓

[생성 요청]

        ↓

[AI가 썰쇼츠 감성 일러스트 생성]

        ↓

[생성된 이미지 미리보기]

        ├─ 카드에 삽입 > 선택한 카드에 이미지 자동 삽입
        └─ 재생성 > 새로운 이미지로 재생성
```

---

## 3. 세부 규칙

| 규칙 | 내용 |
|------|------|
| 화풍 | 썰쇼츠 감성 특화 일러스트 (커스텀 AI 모델) |
| 저작권 | 생성 이미지 저작권 프리 |
| 삽입 방식 | 선택한 카드의 이미지 슬롯에 자동 삽입 |
| 사용 횟수 제한 | 운영 정책에 따라 사용 횟수 제한 가능 |
| 배경 처리 | 투명 배경(PNG) 지원 |
