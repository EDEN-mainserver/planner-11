# 템플릿 시스템 기능 명세

> **문서 경로**: `sunsaktool-docs/03_features/template.md`
> **작성일**: 2026-04-16

---

## 1. 템플릿 개요

| 항목 | 내용 |
|------|------|
| 핵심 기술 | 템플릿 슬롯(Slot) 매핑 시스템 (특허 출원) |
| 원리 | 장르별 사전 정의된 레이아웃 슬롯에 텍스트/오디오 자동 동기화 |
| 전환 방식 | 에디터 상단 GNB 탭 버튼으로 즉시 전환 |

## 2. 템플릿별 상세 명세

### 2-1. 썰 템플릿

| 항목 | 내용 |
|------|------|
| 특징 | 가장 기본적이고 인기 있는 포맷 |
| 구성 | 상단 채널 헤더 + 제목/조회수 + 본문 텍스트 + 이미지 |
| 강점 | 시청 지속 시간이 길어 구독자 확보에 유리 |
| 배지 | 메인, 강력 추천 |
| 지원 기능 | GIF 삽입, AI 이미지, 임시 저장 |

### 2-2. 지식 템플릿

| 항목 | 내용 |
|------|------|
| 특징 | 뉴스 기사처럼 전문적인 정보성 콘텐츠 |
| 구성 | 뉴스 스타일 레이아웃 |
| 강점 | 저장 및 공유 많이 발생, 신뢰감 있는 디자인 |
| 배지 | 메인, 강력 추천 |
| 지원 기능 | GIF 삽입, AI 이미지, 임시 저장 |
| 세이프존 | 유튜브 쇼츠/릴스 UI에 글자 가려지지 않도록 위치 최적화 |

### 2-3. 커뮤니티 템플릿

| 항목 | 내용 |
|------|------|
| 특징 | 한국인에게 익숙한 실시간 베스트 게시판 UI 스타일 |
| 구성 | 커뮤니티 게시판 디자인 + 썰 템플릿 제작 방식 |
| 강점 | 시청자 시선 즉시 집중, 빠르고 쉬운 제작 |
| 추가일 | 2026.02.05 |
| 지원 기능 | 임시 저장 |

### 2-4. 유튜브 댓글 템플릿

| 항목 | 내용 |
|------|------|
| 특징 | YouTube URL 입력 > AI 인기 댓글 자동 추출 |
| 입력 | YouTube 영상 URL (일반 + 공유 단축 URL 지원) |
| AI 처리 | 인기 댓글 자동 수집 + 배경 영상 조합 |
| 필터링 | 타임스탬프 포함 댓글 자동 제외 |
| 마스킹 | 닉네임 자동 마스킹 처리 |
| 강점 | 제작 속도 가장 빠름, 트래픽 확보용 |
| 주의 | 저작권 허용 채널 사용 권장 |

### 2-5. 랭킹/TOP 시리즈

| 항목 | 내용 |
|------|------|
| 특징 | TOP 5, TOP 10 랭킹 영상 |
| 주제 예시 | 스포츠 하이라이트, 유머 모음, 맛집 순위 등 |
| 강점 | 시청 지속 시간이 길고 몰입감 높음 |
| 추가일 | 2026.02.15 |
| 전용 BGM | 랭킹템플릿노래 내장 |

### 2-6. 메시지 템플릿

| 항목 | 내용 |
|------|------|
| 특징 | 아이폰 메시지 대화 형식 영상 |
| 상태 | COMING SOON (준비 중) |
| 진입 | 불가 |
