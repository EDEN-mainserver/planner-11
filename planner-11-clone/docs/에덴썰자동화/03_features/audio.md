## 1. BGM (배경음악)

### 1-1. 내장 BGM 목록

| 트랙명 | 분위기 |
|--------|--------|
| 선택 안함 | BGM 없음 (기본) |
| 랭킹템플릿노래 | 경쾌/긴장 |
| big-cafe | 카페풍 |
| blue-jacket | 경쾌 |
| Cheerful Breeze | 밝고 산뜻 |
| cute-puppy | 귀엽고 경쾌 |
| Humorous Escalation | 유머러스 |
| laughter-in-the-breeze | 활기차고 유쾌 |
| smiley-composure | 차분하고 밝음 |
| spider | 긴장감 |
| Wagging Tails | 귀엽고 따뜻 |
| white-color | 깔끔하고 미니멀 |

### 1-2. BGM 설정 항목

| 항목 | 타입 | 기본값 | 위치 |
|------|------|--------|------|
| BGM 선택 dropdown | select | 선택 안함 | 좌측 패널 + 우측 패널 인라인 |
| BGM 미리듣기 | icon button (🔴) | - | dropdown 우측 |
| 볼륨 조절 | range slider + number | 20 | 좌측 패널 + 우측 패널 |
| 내 BGM 업로드 | file upload button | - | 좌측 패널 |

### 1-3. 동작 규칙

| 규칙 | 내용 |
|------|------|
| 적용 범위 | 전체 영상에 동일 BGM 적용 |
| 좌측/우측 패널 연동 | 어느 쪽에서 변경해도 동기화 |
| 볼륨 범위 | 0 ~ 100 (기본: 20) |
| 사용자 업로드 | 직접 파일 업로드 가능 |
