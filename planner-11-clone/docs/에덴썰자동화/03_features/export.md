# 영상 저장 및 렌더링 기능 명세

> **문서 경로**: `sunsaktool-docs/03_features/export.md`
> **작성일**: 2026-04-16

---

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| 기능명 | 영상 저장 (렌더링 + 다운로드) |
| 버튼명 | `🎬 영상 저장` |
| 위치 | 에디터 상단 GNB |
| 버튼 색상 | 그린 테두리 강조 |
| 출력 형식 | MP4 비디오 파일 |
| 해상도 | 1080p (기본) |

---

## 2. 렌더링 동작 플로우

```text
[🎬 영상 저장 클릭]

        ↓

[렌더링 시작 (서버 처리)]

        ↓

[고유화(Unique-fying) 기술 자동 적용]

  랜덤 노이즈 시드 생성 > 픽셀 값 미세 변조
  메타데이터 새로 구성

        ↓

[MP4 파일 생성 완료]

        ↓

[자동 다운로드]
```

---

## 3. 고유화 (Unique-fying) 기술

| 항목 | 내용 |
|------|------|
| 목적 | 중복 콘텐츠 탐지 회피를 위한 출력 변형 처리 |
| 원리 | 수백만 개의 랜덤 시드 생성 > 픽셀 값 미세 변조 |
| 시각적 차이 | 사람 눈에는 동일하게 보임 |
| AI 인식 | 플랫폼 분석 기준에서 새로운 원본으로 인식될 수 있도록 출력 특성 변경 |
| 메타데이터 | 파일 생성 정보 및 코덱 데이터 매번 새롭게 구성 |
| 효과 | 동일 대본 기반 결과물도 출력 단위로 구분 가능 |

---

## 4. 렌더링 속도

| 항목 | 내용 |
|------|------|
| 처리 방식 | 서버 렌더링 |
| 속도 특성 | 작업 큐 상태 및 처리 정책에 따라 변동 가능 |
| 우선순위 | 운영 설정에 따라 다르게 적용 가능 |

## 5. 영상 길이 제한

| 항목 | 내용 |
|------|------|
| 제한 기준 | 영상 길이 제한은 운영 정책 또는 환경 설정에 따라 달라질 수 있음 |
| 입력 단위 | 초(second) 기준 |
| 적용 시점 | 렌더링 요청 시 검증 |
