## 1. 버튼 (Button)

### 1-1. 버튼 유형별 스펙

| 유형 | 배경 | 텍스트 | border | 용도 |
|------|------|--------|--------|------|
| Primary (레드) | `#e82c43` | `#fff` | none | 주요 CTA, 수동 추가 |
| Primary (블루) | `#4a90e2` | `#fff` | none | AI 대본 생성, 모두 적용 |
| Gradient (핑크) | 핑크퍼플 | `#fff` | none | AI 이미지 생성 |
| Secondary (다크) | `#2a2a2a` | `#e0e0e0` | `1px solid #383838` | 저장, 불러오기 |
| Outline (그린) | transparent | `#e0e0e0` | `2px solid #27c93f` | 영상 저장 |
| Outline (옐로) | transparent | `#e0e0e0` | `2px solid #f0c040` | 보조 액션 버튼 |
| Outline (보라) | transparent | `#e0e0e0` | `2px solid #9b59b6` | 템플릿 선택 보조 버튼 |
| Ghost (투명) | transparent | `#aaa` | none | X 닫기, 보조 액션 |
| Disabled (비활성) | `#555` | `#aaa` | none | 비활성 상태 |
| FAB (원형) | `#e82c43` | `#fff` | none | 새 카드 추가 (+) |
| Tab Active | `#e82c43` | `#fff` | none | 활성 GNB 탭 |
| Tab Inactive | `#2a2a2a` | `#e0e0e0` | none | 비활성 GNB 탭 |

### 1-2. 버튼 공통 스타일

| 항목 | 값 |
|------|-----|
| border-radius | `6~8px` (일반), `4px 4px 0 0` (탭), `50%` (FAB) |
| padding | `8px 16px` (기본), `10px 15px` (탭) |
| font-weight | `700` |
| font-size | `13~16px` |
| cursor | `pointer` |
| transition | `all 0.2~0.3s ease` |

## 7. 모달 (Modal)

### 공통 스펙

| 항목 | 값 |
|------|-----|
| 오버레이 | `rgba(0,0,0,0.85)` 전체 화면 |
| 컨테이너 배경 | `#1e1e1e` ~ `#2a2a2a` |
| border-radius | `12~16px` |
| 최대 너비 | `600~700px` (일반), `450px` (영상 모달) |
| 닫기 방법 | X 버튼 클릭 또는 오버레이 클릭 또는 ESC |
| 등장 애니메이션 | scale(0) > scale(1), 0.3s |

### 모달 종류별 스펙

| 모달명 | 최대 너비 | 주요 구성 |
|--------|-----------|----------|
| 로그인 | 400px | 로고 + 안내 + Google 버튼 |
| AI 대본 생성 | 700px | textarea + 생성 결과 + 액션 버튼 |
| 글감 찾기 | 700px | 카테고리별 외부 링크 목록 + 닫기 |
| 영상 팝업 | 450px | video 태그 (9:16 비율) |
| PC 권장 안내 | 400px | 아이콘 + 안내 텍스트 + 계속하기 버튼 |
| 기능 안내 | 500px | 기능 설명 + 액션 버튼 |
