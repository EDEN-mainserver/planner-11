## 1. 신규 유저 정의

처음 서비스에 접속하는 비로그인 유저로, 서비스를 탐색하고 첫 영상을 만들기까지의 전체 경험을 다룹니다.

## 3. 이탈 포인트 및 대응

| 이탈 포인트 | 원인 | 대응 방법 |
|------------|------|----------|
| 랜딩 페이지에서 이탈 | 서비스 이해 부족 | 예시 영상 팝업, 성공 사례 강조 |
| 템플릿 선택 망설임 | 어떤 걸 써야 할지 모름 | "강력 추천" 배지, ▶ 예시 영상 제공 |
| 에디터 진입 후 이탈 | 복잡해 보임 | 가이드 영상 버튼, Tip 박스 상시 노출 |
| 대본 작성 막힘 | 아이디어 부족 | AI 대본 생성 버튼, 글감 찾기 버튼 |
| 이미지 없어서 막힘 | 소재 부족 | AI 이미지 생성, GIF 찾기, Ctrl+V 안내 |
| 로그인 귀찮음 | 마찰 | Google 원클릭 OAuth |
