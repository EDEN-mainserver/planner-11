### 2-4. 템플릿 소개 섹션 (`#templates`)

| 항목 | 내용 |
|------|------|
| 헤드 | "이런 퀄리티, 클릭 몇 번으로 가능합니다" |
| 구성 | 템플릿 카드 5종 |
| 카드 요소 | 프리뷰 이미지 + 버튼 + 배지 + 이름 + 설명 |
| 클릭 동작 | 예시 영상 팝업 모달 (9:16 비율, 다크 오버레이) |

**템플릿 카드 목록**

| 템플릿명 | 배지 | 설명 |
|----------|------|------|
| 썰 템플릿 | - | 텍스트+이미지 스토리텔링 |
| 지식 템플릿 | - | 정보성 뉴스 형식 |
| 유튜브 댓글 템플릿 | - | URL 기반 AI 댓글 자동 수집 |
| 커뮤니티 템플릿 | NEW | 실시간 베스트 게시판 UI |
| 랭킹 템플릿 | HOT | TOP 5/10 랭킹 영상 |

### 2-5. 주요 기능 섹션 (`#features`)

| 기능명 | 설명 |
|--------|------|
| AI 대본 생성 | 주제 입력 > 대본 자동 완성 |
| TTS (AI 성우) | 텍스트 > 음성 자동 변환 |
| BGM / 효과음 | 클릭 1번으로 사운드 선택 |
| GIF 검색 & 삽입 | 에디터 내 밈 검색 > 즉시 삽입 |
| AI 맞춤 이미지 생성 | 상황 묘사 > 썰쇼츠 스타일 이미지 |
