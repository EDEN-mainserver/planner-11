## F-G-01. 회원가입

### 개요
Google 계정을 통한 OAuth 방식으로 간편하게 AlphaCut 계정을 생성한다.

### 가입 방식
- **Google OAuth 2.0** (소셜 로그인 전용)
- 별도 이메일/비밀번호 등록 불필요
- 신용카드 등록 불필요

### 진입 경로
| 경로 | 설명 |
|------|------|
| 홈 Hero CTA | "무료로 시작하기" 버튼 클릭 |
| 상단 네비 | "무료로 시작하기" 버튼 클릭 |
| 쇼츠 변환 시도 | 비로그인 상태에서 변환 버튼 클릭 시 유도 |
| 상단 알림 배너 | "지금 회원가입하고 무료로 쇼츠를 제작해보세요!" |
| 요금제 페이지 | 각 플랜 "시작하기" 버튼 클릭 |

### 가입 완료 시 혜택
| 혜택 | 내용 |
|------|------|
| 무료 이용권 | **30분** 즉시 지급 |
| 기본 기능 | 템플릿, 기본 편집, 쇼츠 생성 즉시 이용 가능 |

### 처리 흐름
1. "무료로 시작하기" 클릭
2. Google 계정 선택 화면 팝업
3. Google 계정 선택 및 권한 허용
4. AlphaCut 회원 등록 완료
5. 홈 또는 내 프로젝트 화면으로 이동
6. 30분 무료 이용권 자동 지급 알림

## F-G-02. 로그인

### 개요
기존 AlphaCut 계정에 Google OAuth로 인증하여 서비스에 접근한다.

### 로그인 방식
- Google OAuth 2.0 (SSO)
- 별도 비밀번호 불필요

### 진입 경로
| 경로 | 설명 |
|------|------|
| 상단 네비 | "로그인" 버튼 클릭 |
| 비로그인 상태에서 기능 사용 시도 | 자동 로그인 화면으로 리다이렉트 |

### 처리 흐름
1. "로그인" 버튼 클릭
2. Google 계정 선택 화면 팝업
3. 계정 선택 → 인증 완료
4. 이전 페이지 또는 내 프로젝트 화면으로 이동

### 세션 유지
- 세션 유지 기간: 공개 명시 없음
- 자동 로그인: Google 계정 인증 상태 유지 시 자동 처리

## F-G-03. 내 프로젝트 관리

### 개요
로그인한 사용자가 자신의 쇼츠 프로젝트 목록을 확인하고 관리하는 공간이다.

### 접근 경로
- URL: `/my_project`
- 상단 네비게이션 (로그인 후 표시)
- 푸터 "서비스" 섹션 → "내 프로젝트"

### 제공 기능
| 기능 | 설명 |
|------|------|
| 프로젝트 목록 조회 | 생성한 모든 쇼츠 프로젝트 리스트 |
| 처리 상태 확인 | 처리 중 / 완료 / 오류 상태 표시 |
| 쇼츠 미리보기 | 생성된 쇼츠 인라인 재생 |
| 편집 화면 진입 | 선택한 쇼츠 편집 화면으로 이동 |
| 재생성 | 쇼츠 다시 만들기 |
| 다운로드 | 쇼츠 로컬 저장 |
| 유튜브 업로드 | 유튜브 채널에 직접 업로드 |
| 이용권 잔량 확인 | 남은 이용권(분) 표시 |

### 계정 설정
| 설정 항목 | 설명 |
|-----------|------|
| 멤버십 관리 | 현재 플랜 확인 및 업그레이드 |
| 추천인 코드 | 내 추천인 코드 확인 및 복사 |
| 구독 관리 | 구독 취소 또는 플랜 변경 |
