## F-H-01. 멤버십 구독 플랜

### 개요
월간 또는 연간 구독 방식의 멤버십 플랜을 선택하여 정기적으로 이용권을 제공받는다.

### 플랜 상세

#### 무료 플랜 (Free)
| 항목 | 내용 |
|------|------|
| 가격 | 0원 |
| 이용권 | 30분 (가입 시 1회 제공) |
| 가입 방법 | Google 계정, 신용카드 불필요 |
| 포함 기능 | 회원가입 즉시 30분 / 템플릿 / 기본 편집 |
| 제한 | 원본 화질 다운로드 불가, 무음 제거 불가 |

#### Basic 멤버십
| 항목 | 내용 |
|------|------|
| 월간 가격 | ₩5,817 / 월 |
| 연간 가격 | ₩69,800 / 년 (월 환산 시 절약) |
| 이용권 | **월 50분** 영상 변환 가능 |
| 포함 기능 | 기본 편집 / 템플릿 / 원본 화질 다운로드·업로드 |

#### Pro 멤버십 ⭐ 추천
| 항목 | 내용 |
|------|------|
| 월간 가격 | ₩16,650 / 월 |
| 연간 가격 | ₩199,800 / 년 |
| 이용권 | **월 150분** 영상 변환 가능 |
| 포함 기능 | 무음 구간 제거 / 커스텀 폰트 업로드 / 향후 추가되는 모든 기능 |

#### Max 멤버십
| 항목 | 내용 |
|------|------|
| 월간 가격 | ₩49,983 / 월 |
| 연간 가격 | ₩599,800 / 년 |
| 이용권 | **월 450분** 영상 변환 가능 |
| 포함 기능 | Pro 포함 전 기능 / 우선 처리 / 전담 고객 지원 |

#### Business / Enterprise
| 항목 | 내용 |
|------|------|
| 가격 | 문의 (이메일: teamweexe@gmail.com) |
| 이용권 | 맞춤형 커스텀 |
| 포함 기능 | 팀 계정 관리 / 전담 고객 지원 / 맞춤형 브랜딩 / 커스텀 기능 개발 |

## F-H-02. 결제 주기 토글 (월간 / 연간)

### 개요
요금제 페이지에서 월간·연간 결제 방식을 전환할 수 있다.

### UI 요소
- 토글 버튼: "월간" / "연간 16% 할인"
- 기본 선택값: 연간 (할인 강조)

### 연간 선택 시 혜택
| 혜택 | 내용 |
|------|------|
| 할인율 | **16%** 할인 |
| 추가 혜택 | ₩20,000 상당 유튜브 채널 성장 가이드북 무료 제공 |
| 가이드북 내용 | 알고리즘 공략법 / 썸네일·제목 전략 / 단계별 성장 로드맵 / 수익화 가이드 |

### 가격 변동 예시
| 플랜 | 월간 | 연간(월 환산) |
|------|------|--------------|
| Basic | ₩5,817 | 연 ₩69,800 |
| Pro | ₩16,650 | 연 ₩199,800 |
| Max | ₩49,983 | 연 ₩599,800 |

## F-H-03. 단건 이용권 구매

### 개요
멤버십 외에 필요할 때 일회성으로 이용권을 추가 구매한다.

### 정책
- 멤버십 이용권 소진 후 추가 구매 가능
- 단건 구매 가격: 요금제 페이지 내 상세 안내
- 구매한 이용권은 멤버십 이용권과 합산 사용

## F-H-04. 가이드북 단건 구매

### 개요
"유튜브 채널 성장 가이드북 by 알파컷"을 별도 구매할 수 있다.

### 상품 정보
| 항목 | 내용 |
|------|------|
| 가격 | ₩20,000 |
| 분량 | 66페이지 |
| 형식 | 전자책 (e-book) |
| 업데이트 | 지속적 업데이트 (1회 구매 후 계속 이용) |
| 포함 내용 | 알고리즘 공략법, 썸네일·제목 전략, 단계별 성장 로드맵, 수익화 가이드 |
| 특징 | 글로벌 톱 크리에이터 성장 전략 분석, 데이터 검증 핵심 |

### 국내 결제

### 해외 거주자 결제
- 계좌이체 방식으로 별도 처리
- 문의: teamweexe@gmail.com

### 환불 정책
- 환불 기간: 7일 이내
- 상세 내용: /refund 페이지 참조
