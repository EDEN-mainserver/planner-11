## 1. CS 챗봇 버튼 (우하단 플로팅)

### 1.1 속성
| 속성 | 값 |
|------|----|
| 위치 | 화면 우하단 fixed 고정 |
| 크기 | 원형 약 52~56px |
| 배경 | 파란색 (#3B82F6 계열) |
| 아이콘 | 채팅 말풍선 흰색 아이콘 |
| z-index | 최상위 레이어 |
| 그림자 | 드롭 섀도우 효과 |

### 1.2 초기 말풍선
| 속성 | 값 |
|------|----|
| 위치 | 챗봇 버튼 좌상단 방향 말풍선 |
| 내용 | 알파컷에 오신 것을 환영합니다! 궁금한 점이 있으신가요? |
| 닫기 버튼 | 말풍선 우측 X 버튼 |
| 자동 표시 | 페이지 접속 시 자동 노출 |
| 배경 | 흰색 or 연한 다크 카드 |
| 텍스트 색상 | 다크 텍스트 (흰 배경 기준) |

### 1.3 클릭 동작
| 동작 | 결과 |
|------|------|
| 챗봇 버튼 클릭 | 채팅 인터페이스 패널 오픈 (우하단 슬라이드업) |
| 채팅 패널 | 자동 응답 봇 대화 인터페이스 |
| 닫기 버튼 | 채팅 패널 닫힘 |

## 2. 상단 알림 배너 (우상단 플로팅)

### 2.1 속성
| 속성 | 값 |
|------|----|
| 위치 | 화면 우상단 fixed, GNB 하단 |
| 모양 | Pill/캡슐형 둥근 사각형 |
| 배경 | 오렌지→핑크 그라디언트 |
| 텍스트 색상 | 흰색 Bold |
| 아이콘 | 🎁 선물 이모지 (좌측) |
| z-index | GNB보다 위 |

### 2.2 내용
> 🎁 지금 회원가입하고 무료로 쇼츠를 제작해보세요!

### 2.3 동작
| 동작 | 결과 |
|------|------|
| 클릭 | 회원가입/시작 화면으로 이동 |
| 닫기 버튼 | 없음 (항상 표시) |
| 호버 | 약간 밝아지는 hover 효과 (추정) |

## 3. 모달 (Modal) — 이용권 확인

| 속성 | 값 |
|------|----|
| 트리거 | 재생성 버튼 클릭, 이용권 부족 시 |
| 배경 | 반투명 오버레이 (dimmed background) |
| 카드 크기 | 중형 다크 카드 |
| 구성 | 안내 텍스트 + 이용권 차감량 표시 + 확인/취소 버튼 |

## 4. 토스트 / 알림 메시지 (추정)

| 상황 | 메시지 | 위치 |
|------|--------|------|
| 쇼츠 생성 완료 | 쇼츠가 생성되었습니다! | 우상단 토스트 |
| 다운로드 완료 | 다운로드가 완료되었습니다. | 우상단 토스트 |
| 오류 발생 | 처리 중 오류가 발생했습니다. | 우상단 토스트 (빨간색) |
| 저장 완료 | 저장되었습니다. | 우상단 토스트 |
