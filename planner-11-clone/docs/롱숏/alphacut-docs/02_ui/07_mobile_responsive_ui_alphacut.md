> 작성일: 2026-04-16
> 분석 방법: 브라우저 390px (iPhone 14 Pro 기준) 뷰포트 테스트

## 2. GNB (헤더) — 모바일

### 변경 사항
- 네비게이션 링크 (가격안내/사용가이드/FAQ/무료도구) → **숨김**
- 로그인 버튼 → **숨김** (햄버거 메뉴로 이동)
- "무료로 시작하기" CTA 버튼 → **유지** (헤더 우측)
- **햄버거 메뉴 버튼 추가** (≡ "메뉴" 텍스트 버튼)

### 모바일 헤더 레이아웃
```
[로고 아이콘+텍스트]          [무료로 시작하기] [≡ 메뉴]
```

### 햄버거 메뉴 펼침 시
```
┌─────────────────────────────────┐
│  [X 닫기]                       │
├─────────────────────────────────┤
│  가격 안내                      │
│  사용 가이드                    │
│  FAQ                           │
│  무료 도구                      │
├─────────────────────────────────┤
│  로그인                        │
│  무료로 시작하기               │
└─────────────────────────────────┘
```
- 애니메이션: 우측에서 슬라이드 인 (추정) 또는 드롭다운
- 오버레이 배경 반투명 처리

---

## 3. Hero 섹션 — 모바일

### 변경 사항
- 헤드라인: 2줄 → **2줄 유지**, 폰트 크기 축소
- 입력 폼: 전체 너비(full-width)로 확장
- 탭 버튼: 좌우 50% 분할
- "쇼츠로 변환하기" 버튼: 전체 너비
- 쇼츠 갤러리: 가로 스크롤 유지

### 모바일 입력 영역
```
┌─────────────────────────────────┐
│  [🔗 유튜브 링크] [⬆ 파일 업로드]│  ← 50:50 탭
├─────────────────────────────────┤
│  🔗 유튜브 링크를 입력하세요...  │  ← 전체 너비
├─────────────────────────────────┤
│      쇼츠로 변환하기 →          │  ← 전체 너비
└─────────────────────────────────┘
```

---

## 4. 요금제 페이지 — 모바일

### 변경 사항
- 플랜 카드: 가로 배열 → **세로 스택 배열**
- 카드 너비: 전체 너비
- 월간/연간 토글: 전체 너비 유지
- 기능 체크리스트: 그대로 유지

### 모바일 카드 순서
```
[무료 플랜]
[Basic 멤버십]
[Pro 멤버십]
[Max 멤버십]
[Business/Enterprise]
```

## 6. 편집 화면 — 모바일 (추정)

> ⚠️ 편집 화면은 로그인 불가로 직접 확인 불가. 구조 기반 추정.

### 예상 모바일 레이아웃 변경
```
[미리보기 캔버스 (상단, 전체 너비)]
       ↕
[우측 패널 → 하단 슬라이드업 패널로 전환]
[하단 탭바: 텍스트|레이아웃|자막|요소|배경|후킹]
       ↕
[타임라인 → 하단 축약형 또는 스와이프 접근]
```

- 텍스트 입력 시 소프트 키보드 올라옴 → 캔버스 축소 가능성
- 타임라인 조작: 터치 드래그 지원 필요

## 7. 모바일 전용 주의 사항

| 항목 | 내용 |
|------|------|
| 파일 업로드 | 모바일에서 카메라 롤/파일 앱 접근 |
| YouTube URL | 유튜브 앱 공유 → 링크 복사 후 입력 |
| 편집 정밀도 | 타임라인 터치 조작 어려울 수 있음 |
| 다운로드 | 모바일 저장 경로 확인 필요 |
| CS 챗봇 | 하단 고정 플로팅 버튼 유지 |

## 8. 브레이크포인트 추정

| 브레이크포인트 | 레이아웃 |
|--------------|---------|
| < 768px (모바일) | 단일 컬럼, 햄버거 메뉴, 카드 세로 스택 |
| 768px ~ 1024px (태블릿) | 2컬럼 카드, 간소화된 헤더 |
| > 1024px (데스크탑) | 전체 GNB, 다단 카드 레이아웃 |
