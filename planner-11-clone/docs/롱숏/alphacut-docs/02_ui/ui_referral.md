## 1. 페이지 헤더 섹션

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 서브 레이블 | 수익 공유 프로그램 | 작은 배지 또는 레이블 |
| H1 | 알파컷으로 / 평생 수익을 만드세요 | 흰색 ExtraBold, 중앙 정렬 |
| 서브텍스트 | 추천인 링크를 통해 가입한 사용자가 결제할 때마다 10% 커미션을 받으세요. 매월 지속적으로 정산받을 수 있습니다. | 연한 회색, 중앙 |
| CTA 버튼 | 지금 시작하기 | Primary Gradient 버튼 |

## 2. 핵심 수치 배지 행 (3개 배지)

| 배지 | 수치 | 설명 |
|------|------|------|
| 1 | 10% | 커미션 비율 |
| 2 | 평생 | 반복 수익 |
| 3 | 무제한 | 초대 인원 |

### 배지 UI 속성
- 3개 나란히 가로 배치
- 각 배지: 다크 카드 배경 + 라운드 테두리
- 수치: 대형 Bold 흰색
- 설명: 소형 회색 텍스트

## 3. 예상 수익 계산기 섹션

| 요소 | 내용 |
|------|------|
| 섹션 제목 | 예상 수익 계산기 |
| 설명 | 초대 인원에 따른 예상 수익을 확인해보세요 |
| 슬라이더 레이블 | 초대 인원 |
| 슬라이더 범위 | 1명 ~ 1,000명 |
| 슬라이더 기본값 | 300명 |
| 현재값 표시 | 슬라이더 위 숫자 표시 |
| 월 예상 수익 | W594,000 (기본값 300명 기준) — 대형 강조 숫자 |
| 1년 예상 수익 | W7,128,000 — 대형 강조 숫자 |
| 계산 기준 주석 | * 월 W19,800 구독 기준, 10% 커미션 적용 |

### 계산기 UI 속성
- 다크 카드 배경 내 배치
- 슬라이더: 그라디언트 트랙, 원형 핸들
- 수익 숫자: 보라-블루 그라디언트 또는 흰색 Bold 대형

## 4. 작동 방식 섹션 (3단계)

| 요소 | 내용 |
|------|------|
| 섹션 제목 | 어떻게 작동하나요? |
| 서브 레이블 | 3단계로 시작하는 간단한 수익 창출 |
| 레이아웃 | 3열 스텝 카드 가로 배치 |

### 단계 카드 구성
| 단계 | 번호 | 제목 | 설명 |
|------|------|------|------|
| 1 | 1 | 초대 코드 생성 | 추천인 대시보드에서 나만의 고유한 초대 코드와 링크를 생성하세요. |
| 2 | 2 | 공유 | SNS, 블로그, 유튜브 등 어디에서든 초대 링크를 공유하세요. |
| 3 | 3 | 정산금 수령 | 다른 사용자가 결제할 때마다 커미션이 적립되고, 매월 정산됩니다. |

## 5. 추천 대상 섹션

| 요소 | 내용 |
|------|------|
| 섹션 제목 | 이런 분들에게 추천해요 |
| 서브텍스트 | 콘텐츠 크리에이터라면 누구나 참여할 수 있어요 |
| 레이아웃 | 3열 카드 배치 |

### 대상 카드 3종
| 카드 | 대상 | 설명 |
|------|------|------|
| 1 | 유튜버 | 영상 설명란이나 영상 내에서 알파컷을 소개하고 커미션을 받으세요. |
| 2 | SNS 인플루언서 | 인스타그램, 틱톡, 트위터 등에서 팔로워에게 공유하세요. |
| 3 | 블로거 & 뉴스레터 | 리뷰 글이나 뉴스레터에서 알파컷을 소개해보세요. |
