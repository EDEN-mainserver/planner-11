## 1. 블로그 홈 페이지 (/blog)

### 1.1 상단 네비게이션 (블로그 전용)
| 요소 | 내용 |
|------|------|
| 로고 | ALPHACUT 아이콘 + 텍스트 (좌측) |
| 링크 | 블로그 홈 (텍스트 링크, 로고 우측) |
| 메인 GNB 미노출 | 블로그 내에서는 간소화된 헤더 사용 |

### 1.2 헤더 섹션
| 요소 | 내용 |
|------|------|
| 브랜드명 | AI가 만드는 완벽한 숏폼, AlphaCut |
| H1 | 알파컷 블로그 |
| 서브텍스트 | 유튜브 쇼츠와 숏폼 콘텐츠 제작의 모든 것 |
| 설명 | 편집 노하우부터 AlphaCut 활용법까지, 필요한 정보를 한눈에 확인하세요. 초보자부터 전문가까지, 누구나 쉽게 따라할 수 있는 실전 가이드 |

### 1.3 카테고리 필터 탭
| 탭 | 필터 조건 |
|-----|----------|
| 사용 가이드 | category=guide |
| 아티클 | category=article |

### 1.4 게시물 카드 그리드
| 요소 | 내용 |
|------|------|
| 레이아웃 | 3열 카드 그리드 (모바일 1열) |
| 카드 구성 | 썸네일 이미지 + 카테고리 배지 + 제목 + 설명 + 작성자 + 날짜 |
| 썸네일 | 16:9 비율 이미지 |
| 카테고리 배지 | 사용 가이드 (작은 배지, 상단 좌측) |
| 제목 | Bold 흰색, 2행 말줄임 |
| 설명 | 연한 회색, 3행 말줄임 |
| 작성자 | 프로필 이미지(소형) + 이름 + 날짜 |

## 2. 개별 아티클 페이지 (/blog/[slug])

### 2.1 전체 레이아웃
- 좌측: 목차(TOC) 사이드바 (fixed sticky)
- 중앙: 아티클 본문 (메인 컨텐츠 영역)
- 우측: 다른 글 보기 사이드바

### 2.2 아티클 헤더
| 요소 | 내용 |
|------|------|
| H1 | 아티클 제목 |
| 메타 정보 | 작성자 프로필 이미지 + 이름 + 작성일 |
| 대표 이미지 | 전체 너비 썸네일 이미지 |

### 2.3 좌측 목차(TOC) 사이드바
| 요소 | 내용 |
|------|------|
| 레이블 | 목차 |
| 항목 | H2/H3 헤딩 자동 추출 및 앵커 링크 |
| 활성 항목 | 현재 뷰포트 위치 항목 강조 |
| 위치 | 좌측 fixed/sticky |

### 2.4 본문 콘텐츠
| 요소 | 스타일 |
|------|--------|
| 본문 텍스트 | 흰색/연한 회색, Regular ~16px, 적절한 행간 |
| H2 헤딩 | 흰색 Bold ~24px |
| H3 헤딩 | 흰색 SemiBold ~20px |
| 강조 텍스트 | Bold 또는 컬러 강조 |
| 링크 | 보라 또는 블루 컬러, underline |
| 이미지 | 전체 너비 또는 적절한 너비 |

### 2.5 우측 사이드바 (다른 글 보기)
| 요소 | 내용 |
|------|------|
| 레이블 | 다른 글 보기 |
| 카드 | 썸네일 + 제목 (1~2개 추천 글) |
