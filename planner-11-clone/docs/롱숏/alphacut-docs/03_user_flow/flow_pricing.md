# 요금제 구매 플로우 (FL-06)

> 파일 위치: alphacut-docs/03_user_flow/flow_pricing.md
> 작성일: 2026-04-16

---

## 플로우 개요

| 항목 | 내용 |
|------|------|
| 플로우 ID | FL-06 |
| 시작점 | 이용권 부족 팝업 or /pricing 직접 접근 |
| 종료점 | 결제 완료 + 이용권 지급 |

---

## 진입 경로 (3가지)

| 경로 | 트리거 |
|------|--------|
| A | 쇼츠 변환 시도 중 이용권 부족 → 팝업에서 '요금제 보기' 클릭 |
| B | 상단 네비 '가격 안내' 메뉴 클릭 |
| C | 푸터 '요금제' 링크 클릭 |

---

## 상세 플로우

```
┌─────────────────────────────────────────────┐
│  STEP 1: /pricing 페이지 접속               │
│  - 4개 플랜 카드 확인 (무료/Basic/Pro/Max)   │
│  - 결제 주기 토글 (월간 / 연간)              │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 2: 결제 주기 선택                     │
│  - 월간: 매월 결제                          │
│  - 연간: 16% 할인 + 전자책 무료 제공        │
│  - 토글 클릭 시 모든 플랜 가격 즉시 업데이트 │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 3: 플랜 선택                          │
│  - Basic / Pro / Max 중 원하는 플랜 확인     │
│  - 기능 체크리스트 비교 후 결정              │
│  - '시작하기' 버튼 클릭                      │
└────────────────────┬────────────────────────┘
                     │
             [로그인 상태 확인]
                     │
         ┌───────────┴───────────┐
         │                       │
      [로그인됨]             [비로그인]
         │                       │
         │              [로그인/회원가입]
         │              → 완료 후 결제 재개
         │
         ▼
┌─────────────────────────────────────────────┐
│  STEP 4: 결제 정보 입력                     │
│  - 신용카드 / 체크카드 정보 입력             │
│  - 해외 거주자: 계좌이체 방식 (별도 문의)    │

└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 5: 결제 처리                          │
│  - PG사 결제 처리                            │
│                                             │
│  ├── 성공 → STEP 6                          │
│  └── 실패 → 오류 메시지 + 재시도 버튼        │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 6: 결제 완료                          │
│  - 결제 완료 화면 or 토스트 표시             │
│  - 이용권 즉시 지급 (플랜별 분 단위)         │
│  - 구독 시작일 안내                          │
│  - 영수증/확인 이메일 발송                   │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 7: 서비스 이용 재개                   │
```

---

## 플랜별 결제 완료 후 지급 이용권

| 플랜 | 결제 완료 즉시 지급 | 이후 |
|------|-------------------|------|
| Basic | 50분 | 매월 갱신 |
| Pro | 150분 | 매월 갱신 |
| Max | 450분 | 매월 갱신 |

---

## 연간 구독 특이사항

| 항목 | 내용 |
|------|------|
| 할인 | 16% (월간 대비) |
| 추가 혜택 | 유튜브 채널 성장 가이드북 (W20,000 상당) 무료 |
| 결제 방식 | 연 1회 일시 결제 |
| 이용권 지급 방식 | 매월 갱신 (연간 일시 지급 아님) |
