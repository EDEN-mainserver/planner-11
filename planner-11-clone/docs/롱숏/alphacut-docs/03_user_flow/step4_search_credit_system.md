## 1. 검색 이용권(Search Credit) 개요

### 1-1. 정의
- 영상 변환 이용권(분 단위)과 **완전히 분리된 별도 결제 시스템**
- 단위: **회(횟수)**
- 사용 기능: 알파토픽 검색, 영상 검색, 채널 검색, 채널 전체 분석

### 1-2. 두 이용권 시스템 비교

| 구분 | 영상 변환 이용권 | 검색 이용권 |
|------|----------------|------------|
| 단위 | 분(minute) | 회(count) |
| 차감 시점 | 쇼츠 생성 시 | 검색/분석 실행 시 |
| 요금제 포함 여부 | ✅ 멤버십에 포함 | ⚠️ 별도 구매 |
| 무료 제공 | 가입 즉시 30분 | 채널 분석 핵심 요약 1회 무료 |
| 이월 여부 | ❌ 소멸 | ⚠️ 미확인 |

## 2. 검색 이용권 사용 기능별 명세

### 2-1. 알파토픽 (유튜브 바이럴 영상 검색기)
**접속 경로:** /tools/video-search

**기능 설명:**
- 키워드 입력 → 성과 높은 영상 수백 개 수집
- 알파지수(기여도+성과도+반응도 종합) 기반 바이럴 가능성 분석
- Shorts 여부, 조회수, 구독자, 게시일 등 정밀 필터링
- 검색 기록 저장 → 이전 결과 재열람 가능

**크레딧 차감 규칙:**
| 액션 | 차감 여부 |
|------|----------|
| 최초 키워드 검색 | ⚠️ 미확인 (로그인 불가) |
| 추가 검색 (더 깊게 탐색) | ✅ **검색 이용권 1회 차감** |
| 필터 변경/정렬 | ❌ 차감 없음 (추정) |
| 이전 결과 재열람 | ❌ 차감 없음 |

**근거:** blog/new-feature-alpha-topic 공식 가이드 명시
> "상단의 [추가 검색] 버튼을 누르면 AI가 더 깊은 곳에 있는 영상들까지 샅샅이 뒤져 리스트에 추가해 줍니다. (검색 이용권 1회 차감)"

**4가지 성과 지표:**
1. 기여도: 영상이 채널 성장에 기여한 정도
2. 성과도: 채널 체급 무관 독립적 성과
3. 반응도: 시청자 좋아요·댓글 반응 강도
4. 알파지수: 위 3가지 종합 바이럴 가능성 점수

**UI 특이사항:**
- 기본: Great/Good 등 배지(라벨) 표시
- [수치로 보기] 버튼 ON/OFF → 정확한 숫자로 전환 가능
- 영상 클릭 → 상세 팝업 (썸네일, 게시일, 구독자수)

### 2-2. 유튜브 채널 검색·벤치마킹
**접속 경로:** /tools/channel-search

**기능 설명:**
- 키워드 입력 → 관련 채널 수백 개 수집
- 채널별 성과 지표: 구독자, 총영상수, 누적조회수, 평균조회수
- 성장성 지표: 조회/구독 비율, 일평균 구독자 증가율
- 정밀 필터: 구독자 범위, 조회수 범위, 채널 개설일
- 채널 상세 페이지: 동영상 Top 10, Shorts Top 10, 최신 영상 확인 가능

**크레딧 차감 규칙:**
| 액션 | 차감 여부 |
|------|----------|
| 채널 검색 실행 | ✅ **검색 이용권 필요** |
| 채널 상세 열람 | ⚠️ 미확인 |
| 정렬/필터 변경 | ❌ 차감 없음 (추정) |

**근거:** channel-search 페이지 FAQ
> "채널 검색에는 검색 이용권이 필요합니다."

### 2-3. AI 유튜브 채널 분석기
**접속 경로:** /tools/channel-analysis

**기능 설명:**
- YouTube 채널 URL 입력 → AI 심층 분석
- 분석 범위: 댓글 심리, 시청자 정체성, 메타데이터 패턴, 성장 로드맵
- 소요 시간: 핵심 요약 1~2분, 전체 분석 5~10분

**분석 결과 항목:**
1. 핵심 요약 (채널 핵심 가치, 브랜드 원형)
2. 댓글 심리 분석 (긍정/부정/요청 패턴)
3. 시청자 정체성 프로파일 (시청자를 한 단어로 표현)
4. 메타데이터 성과 분석 (제목/태그/업로드 시간대 패턴)
5. 채널 성장 전략 로드맵 (단기/중기/장기)

**크레딧 차감 규칙 (가장 명확하게 확인된 항목):**
| 액션 | 차감 여부 |
|------|----------|
| 첫 채널 핵심 요약 분석 | ❌ **무료** |
| 전체 분석 리포트 | ✅ **검색 이용권 필요** |

**근거:** channel-analysis 페이지 FAQ
> "첫 채널 핵심 요약 분석은 무료로 체험할 수 있습니다. 전체 분석 리포트는 검색 이용권이 필요합니다."

## 3. 검색 이용권 구매 체계

### 3-1. 멤버십 포함 여부
⚠️ **중요:** pricing 페이지 확인 결과, **멤버십(Basic/Pro/Max) 요금제에 검색 이용권 포함 내역이 없음**

→ 검색 이용권은 **멤버십과 별도로 구매**하는 방식으로 추정

### 3-2. 구매 방식 (추정)
⚠️ 로그인 불가 상태 기반 추정:
- 로그인 후 내부 대시보드에서 검색 이용권 별도 충전 가능 예상
- pricing 페이지에 구매 UI가 없으므로, 로그인 상태에서만 접근 가능한 별도 페이지 존재 가능성

### 3-3. 미발견 정보 (로그인 필요)
- 검색 이용권 단가 (예: 1회당 XXX원)
- 패키지 구성 (예: 10회, 30회, 100회 등)
- 이월 정책 (만료일 여부)
- 멤버십과 번들 구매 시 할인 여부

## 5. 알파토픽 UX 흐름 (로그인 필요 기능)

```
[/tools/video-search] 접속
  ↓
[영상 검색 시작하기] 버튼 클릭
  ↓
[로그인 요구?] → 로그인 모달 표시 추정
  ↓ (로그인 후)
[키워드 입력창 활성화]
  ↓
키워드 입력 → [검색] 버튼 클릭
  ↓ (이용권 차감 여부 확인 불가)
[결과 리스트 표시]
  - 알파지수 / 기여도 / 성과도 / 반응도
  - Shorts 여부 필터
  - 게시일, 구독자 필터
  ↓
[추가 검색] 버튼 클릭 → 검색 이용권 1회 차감 ← 유일하게 공식 확인됨
  ↓
[더 깊은 영상 리스트 추가]
  ↓
영상 클릭 → 상세 팝업 표시
  ↓
[새로운 키워드로 검색] → 결과 초기화 후 재시작
```

## 6. 미결 항목 (로그인 후 확인 필요)

1. **최초 키워드 검색 시 이용권 차감 여부** (추가 검색만 명시됨)
2. **검색 이용권 단가 및 패키지 구성** (pricing 페이지에 미노출)
3. **검색 이용권 이월/소멸 정책** (변환 이용권은 이월 불가 확인됨)
4. **주제 추천(/tools/topic-recommendation) 크레딧 차감 규칙**
5. **채널 분석 후 채널 상세 페이지 반복 열람 시 차감 여부**
6. **멤버십 등급별 검색 이용권 기본 제공 여부** (현재 pricing에 미기재)

## 7. 개발/기획 시사점

- **2-tier 이용권 모델:** 변환(분) + 검색(회) 분리 → 서로 다른 과금 로직 필요
- **검색 이용권 UI 위치:** 로그인 후 헤더 또는 대시보드에 잔여 이용권 표시 추정 (변환 이용권과 별도 표시)
- **무료 체험 퍼널:** 채널 분석 핵심 요약 1회 무료 → 전체 분석 유도 → 검색 이용권 구매로 연결되는 upsell 구조
- **알파토픽 도구는 단독 이용 가능** (영상 변환 서비스와 무관하게 검색 이용권만으로 사용 가능)
