# 추천인 등록 및 정산 플로우 (FL-07)

---

## 플로우 개요

| 항목 | 내용 |
|------|------|
| 플로우 ID | FL-07 |
| 참여자 역할 | 추천인(파트너) / 피추천인(신규 사용자) |
| 시작점 | /referral-promo 페이지 접속 |
| 종료점 | 매월 커미션 정산 |

---

## A. 추천인(파트너) 등록 플로우

```
┌─────────────────────────────────────────────┐
│  STEP A-1: /referral-promo 페이지 접속      │
│  - 푸터 '추천인 보상 제도' 링크              │
│  - 블로그 어필리에이트 가이드 링크           │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP A-2: 프로그램 정보 확인               │
│  - 10% 커미션 / 평생 반복 수익 / 무제한 초대 │
│  - 예상 수익 계산기 조작                     │
│  - 작동 방식 3단계 확인                      │
│  - 파트너 이용약관 확인                      │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP A-3: '지금 시작하기' 클릭             │
│  - 로그인 상태 확인                          │
│  - 비로그인 → 로그인/회원가입 후 재진행      │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP A-4: 추천인 대시보드 진입             │
│  - 고유 초대 코드 자동 생성                  │
│  - 초대 링크 생성 (코드 포함 URL)            │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP A-5: 코드/링크 복사 및 공유           │
│  - 복사 버튼으로 클립보드에 복사             │
│  - 공유 채널별 활용:                         │
│    · 유튜브: 영상 설명란 삽입                │
│    · SNS: 인스타/틱톡/트위터 게시            │
│    · 블로그: 리뷰 글 내 링크 삽입            │
│    · 뉴스레터: 구독자에게 발송               │
└─────────────────────────────────────────────┘
```

---

## B. 피추천인(신규 가입) 플로우

```
┌─────────────────────────────────────────────┐
│  STEP B-1: 초대 링크 클릭                   │
│  - 추천인의 SNS/블로그/유튜브에서 링크 클릭  │
│  - 알파컷 홈(/) 랜딩 (초대 코드 파라미터 포함)│
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP B-2: 회원가입                         │
│  - Google OAuth로 가입                       │
│  - 초대 코드 자동 연결 (쿠키/파라미터 기반)  │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP B-3: 유료 구독 결제                   │
│  - Basic/Pro/Max 플랜 구독                  │
│  - 결제 완료 시 추천인에게 커미션 적립       │
└─────────────────────────────────────────────┘
```

---

## C. 커미션 정산 플로우 (매월)

```
┌─────────────────────────────────────────────┐
│  매월 1일: 전월 커미션 집계                  │
│  - 피추천인의 유료 구독 결제 합산            │
│  - 10% 커미션 계산                          │
│  - 3.3% 원천징수 공제                        │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  세금 정보 제출 확인                         │
│  - 주민등록번호 등 세금 처리 정보 제출 여부  │
│  - 미제출 시 정산 보류                       │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  정산금 지급                                 │
│  - 등록된 계좌로 커미션 입금                 │
│  - 정산 내역 대시보드에서 확인 가능          │
└─────────────────────────────────────────────┘
```

---

## 커미션 발생/소멸 조건

| 조건 | 커미션 상태 |
|------|-------------|
| 피추천인 유료 구독 유지 + 파트너 활동 | 커미션 발생 (매월) |
| 피추천인 구독 취소 | 해당 월 이후 커미션 중단 |
| 피추천인 환불 | 해당 커미션 취소 |
| 파트너 1년 미로그인 | 파트너 자격 상실, 미정산 커미션 소멸 |
| 수수료율 변경 | 30일 전 공지 후 변경 적용 |
