# Step 2. 이용권(크레딧) UI 상태 명세 + 크레딧 흐름
> 작성일: 2026-04-16
> 근거: 공식 가이드 (블로그 /blog/guide), 요금제 페이지, 가이드 시리즈
> ⚠️ 로그인 불가 상태 기반 추정 명세 — 실제 구현과 다를 수 있음

## A. 이용권(크레딧) 종류 2가지

알파컷의 이용권은 **2가지 별도 체계**로 운영됩니다.

| 종류 | 단위 | 용도 | 충전 방법 |
|------|------|------|----------|
| **영상 변환 이용권** | 분(minute) | 쇼츠 생성 / 재생성 / AI 나레이션 | 멤버십 가입 (매월 자동 지급) |
| **검색 이용권** | 회(count) | 알파토픽, 채널 분석, 영상 검색, 채널 검색, 썸네일 생성 | 별도 구매 |

---

## B. 영상 변환 이용권 — 차감 규칙 전체

### B-1. 차감 시점

| 액션 | 차감 여부 | 차감량 |
|------|----------|--------|
| 쇼츠 생성 (최초) | ✅ 차감 | 원본 영상 길이 (초 단위 내림 → 분) |
| 재생성 | ✅ 차감 | 최초 차감량의 **50%** |
| AI 나레이션 생성 | ✅ 차감 | **1분 이용권 1매** |
| 편집 (자막/레이아웃/텍스트 수정 등) | ❌ 미차감 | 0 |
| 다운로드 | ❌ 미차감 | 0 |
| 템플릿 저장/불러오기 | ❌ 미차감 | 0 |
| 업로드 정보(메타데이터) 추천 | ❌ 미차감 추정 | 0 |

### B-2. 차감량 계산 규칙

```
[차감량 계산]
  입력 영상 길이 → 초 단위 내림 → 분 단위 변환

  예시:
  ┌──────────────────┬──────────────────┬──────────┐
  │ 입력 영상 길이   │ 차감 (최초 생성) │ 재생성   │
  ├──────────────────┼──────────────────┼──────────┤
  │ 3분 59초         │ 3분              │ 1.5분    │
  │ 10분 50초        │ 10분             │ 5분      │
  │ 30분 00초        │ 30분             │ 15분     │
  └──────────────────┴──────────────────┴──────────┘

[구간 설정 시 절약]
  영상 전체가 아닌 특정 구간만 분석 시 → 해당 구간 길이만큼만 차감
  예: 60분 영상에서 10분 구간만 선택 → 10분만 차감
```

### B-3. 쇼츠 생성 수량 규칙

```
  입력 영상 2분당 쇼츠 1개 생성 (2분 단위 올림 계산)

  ┌──────────────────┬──────────────┐
  │ 입력 영상 길이   │ 생성 쇼츠 수 │
  ├──────────────────┼──────────────┤
  │ 1초 ~ 2분        │ 1개          │
  │ 2분 1초 ~ 4분    │ 2개          │
  │ 4분 1초 ~ 6분    │ 3개          │
  │ ...              │ ...          │
  └──────────────────┴──────────────┘
  ※ AI 맥락 분석에 따라 소폭 편차 가능
```

### B-4. 플랜별 월 지급량 및 단가

| 플랜 | 월 지급 | 월간 금액 | 연간 금액 | 10분당 단가(월간) | 10분당 단가(연간) |
|------|--------|----------|----------|----------------|----------------|
| 무료 | 30분 (1회) | 0원 | — | — | — |
| Basic | 50분 | 6,900원 | 69,800원 | 1,380원 | 1,160원 |
| Pro | 150분 | 19,800원 | 199,800원 | 1,320원 | 1,110원 |
| Max | 450분 | 57,900원 | 599,800원 | 1,290원 | 1,110원 |

⚠️ 월 이용권은 **이월 불가** — 결제 주기 내 미사용분 자동 소멸

## C. 검색 이용권 — 차감 규칙

| 도구 | 차감량 | 비고 |
|------|--------|------|
| 알파토픽 키워드 검색 | 1회 | 결과 리스트 최초 수집 |
| 알파토픽 추가 검색 | 1회 | 기존 리스트에 추가 수집 |
| 채널 분석 (핵심 요약) | 0회 (무료) | 첫 채널만 |
| 채널 분석 (전체 리포트) | 1회 | 댓글/심리/메타/로드맵 포함 |
| 주제 추천 | 1회 추정 | 채널 분석 완료 후 사용 |
| 영상 검색 | 1회 | 키워드 검색 1회 |
| 채널 검색 | 1회 | 키워드 검색 1회 |
| 썸네일 생성 | 0회 (첫 3장 무료) | 이후 1장당 차감 추정 |

## D. 이용권 잔여량 표시 UI (추정)

### D-1. 표시 위치 추정

```
[로그인 후 헤더]
  ┌─────────────────────────────────────────────────────────┐
  │ ALPHACUT  가격안내  사용가이드  FAQ  무료도구   [잔여: 45분] [프로필] │
  └─────────────────────────────────────────────────────────┘
  → 이용권 잔여량을 헤더 우측에 텍스트 또는 배지로 표시 (추정)

[대시보드 상단]
  ┌─────────────────────────────────────┐
  │  이번 달 잔여 이용권                 │
  │  ████████░░░░  45분 / 150분 (Pro)   │
  │  [추가 이용권 구매]                  │
  └─────────────────────────────────────┘
  → 프로그레스 바 + 잔여분/총량 텍스트

[세부 설정 모달 (생성 전)]
  → 선택한 구간 길이와 잔여 이용권 비교 표시
  예: "선택 구간: 10분 | 잔여 이용권: 45분 → 생성 후 35분 남음"
```

### D-2. 잔여량 상태별 표시

| 잔여량 상태 | 표시 방식 | 색상 |
|------------|----------|------|
| 충분 (50% 이상) | 정상 표시 | 초록/기본 |
| 주의 (10~50%) | 강조 표시 | 노랑/주황 |
| 부족 (10% 미만) | 경고 표시 | 빨강 |
| 소진 (0분) | 잠금 표시 + 업그레이드 CTA | 빨강 + 버튼 |

## E. 이용권 부족/소진 시 UX 흐름

### E-1. 잔여량 부족 경고 (생성 시도 시)

```
[쇼츠로 변환하기 버튼 클릭]
  └─> 잔여 이용권 확인
        ├─ 잔여량 ≥ 입력 영상 길이: 정상 생성 진행
        └─ 잔여량 < 입력 영상 길이:
              └─> 경고 메시지:
                    "이용권이 부족합니다.
                     필요: 10분 | 잔여: 3분"
                    └─> CTA 분기:
                          ├─ [구간 설정으로 줄이기] → 세부 설정 모달 재진입
                          │   (잔여량 이하로 구간 조정 유도)
                          └─ [멤버십 업그레이드] → /pricing
```

### E-2. 잔여량 완전 소진 (0분)

```
[대시보드 진입 시 - 잔여 0분 상태]
  └─> 인라인 알림 배너:
        "이번 달 이용권을 모두 사용했습니다.
         다음 결제일: 2026-05-01 | 또는 지금 바로 업그레이드"
        [요금제 보러가기] → /pricing

[변환하기 버튼 클릭 시]
  └─> 버튼 비활성화 또는 클릭 시 즉시 모달:
        "잔여 이용권이 없습니다."
        [멤버십 시작하기] / [플랜 업그레이드]
```

### E-3. 이용권 소진 후 결제 완료 시

```
[/pricing → 플랜 선택 → 결제 완료]
  └─> 즉시 이용권 지급 (결제 즉시 반영)
        └─> 대시보드 복귀 시:
              잔여 이용권 새 플랜 기준으로 갱신
              예: 0분 → 150분 (Pro 전환 시)
```

### E-4. 월 갱신 시점 흐름

```
[매월 결제일 도달]
  └─> 자동 결제 처리
        └─> 이용권 자동 초기화 + 새 이용권 지급
              └─> (추정) 이메일 또는 인앱 알림:
                    "이번 달 이용권 150분이 지급되었습니다."

[미사용 이용권 처리]
  └─> 잔여 이용권 소멸 (이월 없음)
        └─> 새 이용권으로 교체

[갱신 전 D-3 알림 추정]
  └─> "이용권이 3일 후 소멸됩니다. 남은 OO분을 활용하세요"
```

## F. 전체 크레딧 흐름 한눈에 보기

```
[이용권 지급]
  가입 → 무료 30분 즉시 지급
  멤버십 결제 → 플랜별 월 이용권 즉시 지급
  매월 결제일 → 자동 갱신 지급

       ↓

[이용권 차감]
  쇼츠 생성     → 영상 길이(분) 차감
  재생성        → 영상 길이 × 50% 차감
  AI 나레이션   → 1분 고정 차감
  구간 설정     → 구간 길이만큼만 차감 (절약 가능)
  편집/다운로드 → 차감 없음

       ↓

[이용권 소멸]
  매월 결제 주기 만료 → 잔여 이용권 소멸 (이월 없음)
  환불 조건 해당 시   → 미사용분 환불 가능 (7일 이내)

       ↓

[검색 이용권 — 별도 체계]
  별도 구매 → 알파토픽/채널분석/영상검색/채널검색/썸네일 사용
  영상 변환 이용권과 완전 분리된 체계
```

| # | 항목 | 중요도 |

| 4 | 이용권 소진 시 변환 버튼 비활성 vs 클릭 후 모달 방식 | 🟡 중간 |
| 5 | 멤버십 해지 후 이용권 즉시 소멸 vs 주기 만료까지 유지 | 🟡 중간 |
