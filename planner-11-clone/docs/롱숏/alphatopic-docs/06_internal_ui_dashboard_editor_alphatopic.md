## A. 대시보드 (내 프로젝트) — 로그인 후 화면

### A-1. 영상 입력 영역 (상단)

```
┌─────────────────────────────────────────────────────────┐
│  [🔗 유튜브 링크]         [⬆ 파일 업로드  업로드]        │  ← 탭
├─────────────────────────────────────────────────────────┤
│  🔗 유튜브 영상 링크를 입력하세요 (예: https://...)       │  ← 텍스트 입력
├─────────────────────────────────────────────────────────┤
│              쇼츠로 변환하기 →                           │  ← CTA 버튼
└─────────────────────────────────────────────────────────┘
```

**생성 전 세부 설정 (모달)**
- 진입: 영상 확인 화면에서 썸네일 하단 [세부 설정] 클릭
- 구간 설정: 슬라이더로 시작~끝 지점 지정 (미설정 시 전체)
- 쇼츠 길이: 4가지 옵션 라디오 선택 (미설정 시 자동 40초 내외)
- 제목 언어: 한국어(기본) / 영어
- 자막 언어: 한국어 / 영어 / 자동(AI 감지, 기본값)
- 템플릿 선택: 내 템플릿 목록에서 선택 (미선택 시 최근 사용 템플릿 자동 적용)

### A-2. 프로젝트 목록 (하단)

| 컴포넌트 | 내용 |
|---------|------|
| 쇼츠 카드 | 9:16 썸네일 + 제목 + 상태 배지 |
| 상태 | 처리 중(스피너) / 완료(초록) / 오류(빨강) |
| 처리 시간 | 영상 5~10분, 길수록 비례 증가 |
| 액션 버튼 | 편집 / 다운로드 / 재생성하기 |
| 재생성 | 기존 쇼츠 유지 + 새 쇼츠 추가, 이용권 50% 차감 |

## B. 편집 화면 전체 구조

```
┌────────────────────────────────────────────────────────────┐
│  [헤더: 프로젝트명 / 저장 / 업로드 정보 추천 / 재생성하기]   │
├───────────────┬────────────────────┬───────────────────────┤
│               │                    │  우측 패널 탭          │
│  (좌측 패널)  │  미리보기 캔버스    │  [텍스트][레이아웃]    │
│  쇼츠 목록    │  (9:16 고정)       │  [자막][요소][배경]    │
│  카드 리스트  │                    │  [썸네일][AI후킹]      │
│               │                    │                       │
│               │                    │  (선택된 탭 내용)      │
│               │                    │                       │
├───────────────┴────────────────────┴───────────────────────┤
│  하단 타임라인 패널                                          │
│  [[ ] | 🗑️  +돋보기 -돋보기  <<  >>  무음제거]  속도설정   │
│  ════════════════════════════════════════ 타임라인 트랙     │
│  ──────────────── 자막 트랙 ───────────────────────────    │
│                             [AI 후킹 탭] ●                 │
└────────────────────────────────────────────────────────────┘
```

## C. 템플릿 페이지

### C-1. 구조
```
┌────────────────────────────────────────────────────────┐
│  [추천 템플릿]  [내 템플릿]  [새 템플릿 생성]           │  ← 탭 + 버튼
├────────────────────────────────────────────────────────┤
│  좌측: 템플릿 목록 썸네일 그리드                        │
│  우측: 선택된 템플릿 설정 패널                          │
│    ├─ A. 텍스트 및 정보 설정                           │
│    ├─ B. 레이아웃 설정                                 │
│    ├─ C. 자막 설정 (애니메이션 포함)                   │
│    ├─ D. 요소 및 배경 설정                             │
│    └─ E. 영상 효과 (하이라이트 훅, 배속)               │
├────────────────────────────────────────────────────────┤
│  우측 상단: [저장하기] / [덮어쓰기] 버튼               │
└────────────────────────────────────────────────────────┘
```

### C-2. 쇼츠 구조 7요소
| # | 요소 | 설명 |
|---|------|------|
| ① | 쇼츠 제목 | AI 생성 후킹 제목 (기본 2줄, 줄별 스타일 가능) |
| ② | 채널 정보 | 프로필 이미지 + 채널명 |
| ③ | 원본 영상 제목 | 유튜브 제목 또는 파일명 |
| ④ | 레이아웃 | 영상 배치 영역 (비율, 위치, 듀얼) |
| ⑤ | 배경 | 색상 / 이미지 / 유리효과 |
| ⑥ | 요소 | 이미지/비디오/오디오/텍스트 오버레이 |
| ⑦ | 자막 | AI STT 자동 생성, 위치/스타일/애니메이션 |

## D. 알파토픽 기능 명세 (Beta)

### D-1. 진입
- 헤더 상단 메뉴 [알파토픽] 클릭 (로그인 필요)
- 검색 이용권 필요 (영상 변환 이용권과 별도)

### D-2. 검색 기능
```
[검색창]
  └─ 키워드 입력 (예: "재테크", "다이어트 식단")
        └─ [검색] 버튼 클릭
              └─ AI가 해당 키워드 성과 영상 수집·분석
                    └─ 검색 이용권 1회 차감
```

### D-3. 결과 화면

#### 4가지 핵심 지표
| 지표 | 의미 |
|------|------|
| 📈 기여도 | 해당 채널 성장에 기여한 정도 |
| 🚀 성과도 | 채널 체급 대비 타겟 도달 성과 |
| 💬 반응도 | 좋아요/댓글 등 시청자 참여도 |
| ⚡ 알파지수 | 전체 데이터 종합 바이럴 가능성 점수 |

#### 표시 방식
- 배지 라벨: Great / Good / Average / Low
- 수치 보기 토글: 정확한 숫자 데이터로 전환

#### 정렬 기능
- 컬럼 헤더 클릭: 조회수 / 구독자 / 알파지수 정렬 (오름/내림)

#### 영상 상세 팝업
- 클릭 시: 썸네일, 정확한 게시일, 채널 구독자 수, 지표 상세

### D-4. 확장/필터 기능

| 기능 | 설명 |
|------|------|
| 추가 검색 | [추가 검색] 버튼 → 더 깊은 곳 영상 수집 (이용권 1회 추가) |
| 구독자 필터 | 슬라이더 → 특정 체급 채널만 필터 |
| 지표 필터 | 성과도 'Great'인 영상만 등 |
| 기간/형식 필터 | 최근 1개월, 쇼츠만 보기 등 |
| 결과 내 검색 | 하단 입력창 → 수집된 리스트 내 텍스트 필터 |
| 새 키워드 검색 | [새로운 키워드로 검색하기] → 기존 결과 초기화 |

## E. 업로드 정보 추천 기능

### E-1. 진입
- 생성된 쇼츠 확인 페이지 → [업로드 정보 추천] 버튼

### E-2. 생성 항목
| 항목 | 설명 |
|------|------|
| 제목 | 유튜브 알고리즘 최적화 제목 |
| 설명글 | 영상 내용 기반 Description |
| 태그/해시태그 | 노출 최적화 키워드 목록 |
| 카테고리 | 유튜브 카테고리 추천 |

### E-3. 활용
- 각 항목 옆 [복사] 버튼 → 클립보드 복사
- 유튜브 스튜디오 업로드 창에 붙여넣기
