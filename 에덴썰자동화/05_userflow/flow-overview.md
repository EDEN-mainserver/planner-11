# 전체 유저플로우 개요

> **문서 경로**: `sunsaktool-docs/05_userflow/flow-overview.md`
> **작성일**: 2026-04-16

---

## 1. 전체 플로우 다이어그램

```text
[sunsaktool.com 랜딩 접속]

        ↓

  첫 방문 유저?

        ├─ YES → [랜딩 페이지] → [스크롤 탐색]
        └─ NO  → [/select-template 또는 /app 직접 진입]

        ↓

[CTA 클릭: 🚀 숏폼 만들러 가기]

        ↓

[/select-template > 템플릿 선택]

        ↓

비회원 상태 > 에디터 진입 가능 (일부 기능 제한 가능)

        ↓

[템플릿 카드 클릭]

        ↓

[/app > 에디터 진입]

        ├─ [채널 설정]
        │    [헤더 / 로고 / 색상 / 폰트]
        │
        ├─ [AI 대본 생성]
        │    [주제 입력 > 대본 자동 생성]
        │
        ├─ [카드 편집]
        │    [텍스트 작성 / 수정]
        │
        ├─ [미디어 추가]
        │    [이미지 / GIF / AI 이미지]
        │
        └─ [사운드 설정]
             [BGM / 효과음 / TTS]

        ↓

[미리보기 재생 확인]
[재생 버튼으로 전체 확인]

        ↓

수정 / 저장

        ↓

[🎬 영상 저장 클릭]
[렌더링 + 고유화 처리]

        ↓

[MP4 다운로드 완료]

        ↓

[SNS 업로드: YouTube / Reels / TikTok]
```

---

## 2. 인증 분기 플로우

```text
[기능 사용 시도]

        ↓

로그인 상태인가?

        ├─ YES → [바로 기능 사용]
        └─ NO  → [잠금 아이콘 표시]
                    ↓
               [로그인 모달 팝업]
                    ↓
               [Google 계정으로 시작하기]
                    ↓
               [Google OAuth 인증]
                    ↓
               [로그인 완료 > 기능 활성화]
```

---

## 3. 영상 제작 핵심 루프

```text
[카드 작성]

        ↓

[TTS 미리듣기로 확인]

        ↓

[미디어 추가 (이미지 / GIF / AI 이미지)]

        ↓

[효과음 설정]

        ↓

[전체 미리보기]

        ↓

OK?

        ├─ NO  → [카드 수정으로 돌아가기]
        └─ YES → [영상 저장]
```

---

## 4. 페이지별 진입 경로 요약

| 진입 경로 | 도착 페이지 | 조건 |
|-----------|------------|------|
| 직접 URL 입력 | 각 페이지 | 누구나 |
| 랜딩 CTA 버튼 | `/select-template` | 누구나 |
| 템플릿 카드 클릭 | `/app` | 누구나 (기능 일부 제한 가능) |
| GNB 탭 버튼 | `/app` (템플릿 전환) | 에디터 내에서 |
| 로고 클릭 | `/select-template` | 에디터 내에서 |
| 보조 이동 버튼 | 별도 연결 페이지 | 노출 위치에서 |
