## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 목적 | 숏폼 영상 제작 핵심 작업 공간 |
| 레이아웃 | 3-Column (좌측 설정 / 중앙 캔버스 / 우측 카드 패널) |

## 2. 전체 레이아웃

```text
[상단 GNB]

                                     + 수동 추가
  좌측                                새 프로젝트 시작
  설정       중앙 미리보기 캔버스     💾 저장   📂 불러오기
  패널         (9:16 비율)             AI 대본 생성 (고도화)
                                     🎨 AI 맞춤 이미지 생성
  [스크롤                            💡 글감 찾기
   가능]                             배경음악: [dropdown] 🔴

                                     🎙 전체 카드 AI 음성 설정
                                     목소리 [dropdown]
                                     속도 [slider] [모두적용]

                                     💡 Tip 박스
                                     📋 이미지 붙여넣기 안내

                                     [카드 1]
                                     [카드 2]
                                     ...
            [+ 새 카드 추가 버튼]
            [ 타임라인 슬라이더]
```

## 6. 우측 카드 패널

### 6-1. 상단 액션 버튼 영역

| 버튼명 | 색상 | 기능 |
|--------|------|------|
| + 수동 추가 | 레드 | 빈 카드 수동 추가 |
| 새 프로젝트 시작 | 다크 | 현재 작업 초기화 후 새 프로젝트 시작 |
| 💾 저장 | 다크 | 현재 작업 저장 |
| 📂 불러오기 | 다크/옐로 | 저장된 작업 불러오기 |

### 6-2. AI 기능 버튼 영역

| 버튼명 | 색상 | 기능 |
|--------|------|------|
| AI 대본 생성 (고도화) | 블루 그라데이션 | 주제 입력 > 전체 대본 자동 생성 |
| 🎨 AI 맞춤 이미지 생성 | 핑크/퍼플 | 묘사 입력 > 썰쇼츠 스타일 이미지 생성 |
| 💡 글감 찾기 | 블루 | 콘텐츠 아이디어 추천 |

### 6-3. 배경음악 인라인 설정

| 요소 | 타입 | 설명 |
|------|------|------|
| 배경음악 라벨 | text | "배경음악:" |
| BGM 선택 | dropdown | 내장 BGM 목록 |
| BGM 재생 버튼 | icon (🔴) | 현재 선택 BGM 미리듣기 |
| 볼륨 슬라이더 | range | 0~100 (기본: 20) |

### 6-4. 전체 카드 AI 음성 일괄 설정

| 요소 | 타입 | 기본값 |
|------|------|--------|
| 목소리 | dropdown | 표준 남성 1 (또는 이전 선택값) |
| 속도 | range slider | 1.30x ~ 1.35x |
| 모두 적용 버튼 | button (블루) | 전체 카드 동일 설정 일괄 적용 |

### 6-5. 안내 박스 (상시 노출)

| 박스명 | 내용 | 색상 |
|--------|------|------|
| Tip 박스 | "스크립트의 각 줄을 더블클릭하여 영상에 나오는 순서와 효과음을 편집할 수 있어요!" | 노란 테두리 |
| 이미지 붙여넣기 안내 | "이미지 붙여넣기(Ctrl+V) 지원! 구글에서 이미지를 검색 후 '이미지 복사'하여 바로 붙여넣어 보세요." | 파란 배경 |

### 6-6. 스크립트 카드

**카드 구조 상세**

```text
[번호: 1]                          [🗑 삭제]

  [텍스트 입력 영역]
  (contenteditable div, 멀티라인)

  길이(수동) [숫자] s    [미디어 추가]
  GIF 찾기                [출처 추가]

  목소리: [dropdown]
  속도: [range slider]  X.XXx
  카드 효과음: [dropdown]
```

**카드 내 요소 상세**

| 요소 | 타입 | 설명 |
|------|------|------|
| 카드 번호 | text | 자동 부여 (1, 2, 3...) |
| 🗑 삭제 버튼 | button (레드) | 해당 카드 즉시 삭제 |
| 텍스트 입력 | contenteditable div | 스크립트 본문 입력 |
| 길이(수동/자동) | toggle + number | 카드 재생시간 설정 |
| 미디어 추가 | button | 이미지 파일 업로드 |
| GIF 찾기 | button | 밈/GIF 검색 모달 |
| 출처 추가 | button | 출처 텍스트 입력 |
| 목소리 | dropdown | 카드별 개별 목소리 선택 |
| 속도 | range slider | 카드별 개별 속도 설정 |
| 카드 효과음 | dropdown | SFX 50종+ 선택 |

**카드 인터랙션**

| 동작 | 결과 |
|------|------|
| 텍스트 영역 클릭 | 커서 활성화, 텍스트 입력 |
| 텍스트 영역 더블클릭 | 줄 단위 편집 모드 진입 |
| 카드 드래그 | 카드 순서 변경 (드래그 앤 드롭) |
| Ctrl+V (이미지 복사 후) | 해당 카드에 이미지 즉시 삽입 |
| 🗑 클릭 | 확인 없이 즉시 삭제 |

### 6-7. 새 카드 추가 버튼

| 요소 | 설명 |
|------|------|
| 위치 | 우측 하단 고정 (FAB 스타일) |
| 모양 | 원형 + 버튼, 레드 배경 |
| 동작 | 마지막 카드 하단에 빈 카드 추가 |
