## 4. 메인 콘텐츠: 템플릿 모음

### 4-1. 헤더

- 대제목: "어떤 숏폼을 만들까요?"
- 소제목: "시작할 템플릿을 선택해주세요."

### 4-2. 템플릿 카드 목록

| # | 템플릿명 | 상태 | 배지 | 에디터 진입 URL |
|---|----------|------|------|----------------|
| 1 | 썰 템플릿 | 운영 중 | 메인, 강력 추천 | `/app?template=ssul` |
| 2 | 지식 템플릿 | 운영 중 | 메인, 강력 추천 | `/app?template=knowledge` |
| 3 | 커뮤니티 템플릿 | 운영 중 | - | `/app?template=community` |
| 4 | 유튜브 댓글 템플릿 | 운영 중 | - | `/app?template=youtube` |
| 5 | 랭킹/TOP 시리즈 | 운영 중 | - | `/app?template=ranking` |
| 6 | 메시지 템플릿 | 준비 중 | COMING SOON | 진입 불가 |

### 4-3. 예정 기능 카드

| 기능명 | 상태 | 내용 |
|--------|------|------|
| 🤖 AI 유튜브 자동 클립 | 개발 예정 | 긴 영상 > 하이라이트 쇼츠 자동 추출 |

### 4-4. 템플릿 카드 구성 요소

```text
[배지: 메인] [배지: 강력 추천]

  [프리뷰 이미지]
     예시 영상 버튼

  [템플릿명]
  [설명 텍스트]
  가이드 보기 (있는 경우)
```

- 카드 클릭 > 에디터(`/app`) 해당 템플릿으로 진입
- 예시 영상 > 팝업 모달로 예시 영상 재생

## 5. 메인 콘텐츠: 공지사항

### 5-1. 공지 카드 구성

```text
[태그] [날짜]
[제목 (이모지 포함)]
[내용 요약]
```

### 5-2. 공지 태그 종류

| 태그명 | 색상 | 의미 |
|--------|------|------|
| EVENT | 파랑 | 이벤트/할인 |
| NEW FEATURE | 초록 | 신규 기능 추가 |
| HOT FIX | 빨강 | 긴급 버그 수정 |
| ACCESS OPEN | 초록 | 접근 권한 개방 |
| UPDATE | 주황 | 기능 업데이트 |
| IMPROVED | 보라 | 기능 개선 |
| GRAND OPEN | 레드 | 정식 오픈 |
| HOT UPDATE | 레드 | 주요 업데이트 |
| SPEED UP | 파랑 | 성능 개선 |
| NEW TEMPLATE | 초록 | 템플릿 신규 추가 |
| 해결됨 | 회색 | 이슈 해결 |

## 6. 메인 콘텐츠: 수익화 가이드

### 6-1. 가이드 단계 구성

| STEP | 제목 | 핵심 내용 |
|------|------|-----------|
| 01 | 계정 분리 전략 | 채널별 컨셉 분리 (썰/지식/댓글 채널 분리 운영) |
| 02 | 템플릿별 채널 방향성 | 각 채널에 맞는 템플릿 전용 사용 |
| 03 | 원소스 멀티유즈 | YouTube Shorts + Instagram Reels + TikTok 동시 업로드 |

### 6-2. 채널 분리 전략 예시

| 채널 | 컨셉 | 권장 템플릿 |
|------|------|-------------|
| 🔴 A 채널 | 썰/유머 (네이트판, 커뮤니티 베스트) | 썰 템플릿 |
| 🟢 B 채널 | 지식/정보 (뉴스, 생활 꿀팁, 상식) | 지식 템플릿 |
| 🟡 C 채널 | 반응/여론 (유튜브 베댓, 해외 반응) | 댓글 템플릿 |
