# TTS (텍스트 > 음성) 기능 명세

> **문서 경로**: `sunsaktool-docs/03_features/tts.md`
> **작성일**: 2026-04-16

---

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| 기능명 | TTS (Text-to-Speech) AI 음성 생성 |
| 엔진 | Google Cloud TTS API (ko-KR) |
| 적용 단위 | 카드별 개별 설정 OR 전체 일괄 적용 |
| 위치 1 | 우측 패널 "전체 카드 AI 음성 일괄 설정" 영역 |
| 위치 2 | 각 카드 하단 목소리/속도 개별 설정 |

---

## 2. 목소리 옵션

| 라벨 | Google TTS 코드 | 성별 |
|------|-----------------|------|
| 표준 남성 1 | ko-KR-Standard-C | 남성 |
| 표준 남성 2 | ko-KR-Standard-D | 남성 |
| 표준 여성 1 | ko-KR-Standard-A | 여성 |
| 표준 여성 2 | ko-KR-Standard-B | 여성 |

---

## 3. 속도 설정

| 항목 | 내용 |
|------|------|
| 조절 방식 | range slider + 수치 직접 표시 |
| 범위 | 0.5x ~ 2.0x |
| 기본값 | 1.35x |
| 표시 형식 | 소수점 2자리 (예: 1.35x) |

---

## 4. 전체 일괄 적용

| 항목 | 내용 |
|------|------|
| 위치 | 우측 패널 상단 "전체 카드 AI 음성 일괄 설정" 섹션 |
| 버튼 | `모두 적용` (블루) |
| 동작 | 설정한 목소리 + 속도를 전체 카드에 동시 적용 |
| 주의 | 개별 카드에서 이미 설정된 값도 덮어씀 |

---

## 5. 카드별 개별 설정

| 항목 | 내용 |
|------|------|
| 위치 | 각 카드 하단 목소리 dropdown + 속도 slider |
| 독립성 | 카드별로 다른 목소리/속도 설정 가능 |
| 미리듣기 | 카드 내 `길이(수동/자동)` 버튼으로 해당 카드 음성 미리 재생 |

---

## 6. 길이 설정 모드

| 모드 | 설명 |
|------|------|
| 자동 | TTS 음성 길이에 맞춰 카드 재생시간 자동 계산 |
| 수동 | 숫자 직접 입력 (초 단위) |

---

## 7. 음성 품질 처리

| 항목 | 내용 |
|------|------|
| 겹침 제거 | 지능형 음성 엔진으로 문장 간 겹침 자동 탐지 및 제거 |
| 공백 처리 | 불필요한 초기 무음(0.1초 단위) 자동 제거 |
| 볼륨 최적화 | TTS 볼륨 2.5배 증폭 (BGM에 묻히지 않도록) |
| 저장 일치 | 미리보기에서 들은 사운드 그대로 영상 저장 |
