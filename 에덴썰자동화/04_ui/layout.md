## 1. 에디터 전체 레이아웃 구조

```text
[GNB 상단바] 고정 / 높이: ~60px / 배경: #1e1e1e

  좌측 패널         중앙 캔버스 영역              우측 패널
  width: ~160px     flex: 1 (가변)                width: ~340~400px
  overflow-y:       배경: #111 계열               overflow-y: scroll
  scroll            padding: 20px                 배경: #1e1e1e

  [영상설정]          [스크립트 & 타임라인]
  상단 헤더          9:16 캔버스                  + 수동추가 / 새프로젝트
  콘텐츠 정보        (세로형)                     저장 / 불러오기
  스크립트           실시간 렌더링                AI 대본 생성 버튼
  스타일             워터마크 오버레이            AI 이미지 / 글감찾기
  BGM 설정           [재생] Xs                    배경음악 인라인
                                                   전체 TTS 설정
                                                   Tip 박스
  하단 스크롤바 (좌우)                              이미지 붙여넣기 안내
                                                   카드 1, 2, 3...
                                                   [+ FAB 버튼]
```

---

## 2. 영역별 상세 스펙

### 2-1. GNB (상단 네비게이션 바)

| 항목 | 값 |
|------|-----|
| 높이 | ~60px |
| 배경 | `#1e1e1e` |
| position | sticky / fixed top |
| 좌측 | SunsakTool 로고 |
| 중앙 | 템플릿 탭 버튼 그룹 |
| 우측 | 가이드영상/저장/프로필 버튼 그룹 |
| 탭 버튼 높이 | 100% (GNB 높이 채움) |
| 활성 탭 | 배경 `#e82c43`, 텍스트 `#ffffff` |
| 비활성 탭 | 배경 `#2a2a2a`, 텍스트 `#e0e0e0` |
| 탭 border-radius | `4px 4px 0 0` |

### 2-2. 좌측 설정 패널

| 항목 | 값 |
|------|-----|
| 너비 | ~160px (고정) |
| 배경 | `#1e1e1e` |
| overflow | scroll (세로) |
| 구분 | 섹션 헤더로 영역 구분 |
| 반응형 | 좁은 화면에서 숨김 처리 |
| 우측 테두리 | `1px solid #383838` |

**섹션 순서**

1. [영상 설정] 헤더
2. 상단 헤더 설정 (채널명, 색상, 폰트, 아이콘)
3. 커스텀 헤더 이미지
4. 오른쪽 로고
5. 콘텐츠 정보 (제목, 조회수, 색상)
6. 기본 스크립트 스타일 (폰트, 크기, 색상, 정렬, 줄간격, 자간)
7. 배경음악 (BGM 선택, 볼륨)

### 2-3. 중앙 캔버스 영역

| 항목 | 값 |
|------|-----|
| 가로 | flex: 1 (가변) |
| 배경 | `#111` 계열 어두운 배경 |
| 캔버스 비율 | 9:16 (세로형 숏폼) |
| 캔버스 배경 | 흰색 (`#ffffff`) |
| 캔버스 최대 너비 | ~480px |
| 중앙 정렬 | flexbox center |
| 하단 컨트롤 | 재생버튼 + range slider + 재생시간 |

### 2-4. 우측 카드 패널

| 항목 | 값 |
|------|-----|
| 너비 | ~340~400px (고정) |
| 배경 | `#1e1e1e` |
| overflow | scroll (세로) |
| 상단 | "스크립트 & 타임라인" 섹션 헤더 |
| 버튼 행 1 | + 수동추가 / 새 프로젝트 시작 |
| 버튼 행 2 | 저장 / 불러오기 |
| 버튼 행 3 | AI 대본 생성 (전체 너비) |
| 버튼 행 4 | AI 이미지 생성 / 글감 찾기 (2분할) |
| 인라인 | 배경음악 dropdown + 볼륨 슬라이더 |
| TTS 섹션 | 전체 카드 AI 음성 일괄 설정 |
| 상시 안내 | Tip 박스 + 이미지 붙여넣기 안내 |
| 카드 영역 | 스크롤 가능한 카드 목록 |
| FAB 버튼 | 우하단 고정 원형 + 버튼 |

## 3. 반응형 동작

| 화면 너비 | 동작 |
|-----------|------|
| 1280px 이상 | 3컬럼 정상 표시 |
| 992px ~ 1280px | 좌측 패널 축소 표시 |
| 992px 미만 | PC 접속 권장 안내 팝업 표시 |
| 모바일 | 경고 모달 > "불편함을 감수하고 계속하기" 버튼으로 진입 가능 |

## 4. 모달 레이아웃

```text
전체 화면 오버레이 (rgba(0,0,0,0.85))

    모달 컨테이너
    배경: #1e1e1e ~ #2a2a2a
    border-radius: 12~16px
    max-width: 600~700px
    padding: 30~40px

    [아이콘] 제목
    설명 텍스트
    입력 영역 (textarea / input)
    액션 버튼 행
```

---

## 5. 스크립트 카드 레이아웃

```text
border: 1px solid #383838
카드 헤더 (drag handle)                          활성 시: #e82c43
[번호]                         [🗑 삭제]         border-radius: 8px

[텍스트 입력 div]                                min-height: ~80px
contenteditable                                  배경: #1a1a1a
padding: 10~12px

길이(자동/수동) [숫자]s  [미디어 추가]           하단 컨트롤 행 1
[GIF 찾기]               [출처 추가]             하단 컨트롤 행 2

목소리:      [dropdown]                          TTS 설정
속도:        [slider] 1.35x
카드 효과음: [dropdown]
```

```text
  좌측 사이드바    메인 콘텐츠
  width: ~250px    flex: 1
  배경: #111       배경: #0a0a0a

  메뉴 목록        [선택된 탭 콘텐츠]
  (세로 리스트)    - 템플릿 카드 그리드
                   - 공지 카드 목록
                   - 가이드 단계 카드

  활성 메뉴:
  좌측 레드 바
```
