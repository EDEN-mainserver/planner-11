## 개요

AI가 생성한 쇼츠 결과물을 확인하고, 재생성·다운로드·유튜브 업로드 자동화까지
결과물의 전체 생명주기를 관리하는 기능 묶음이다.

---

## F-E-01. 결과물 확인 (내 프로젝트)

### 개요
사용자가 생성한 모든 쇼츠 프로젝트를 한 페이지에서 확인한다.

### 접근 경로
- URL: `/my_project`
- 상단 네비게이션 → 로그인 후 접근 가능
- 푸터 → "내 프로젝트" 링크

### 표시 정보
| 항목 | 설명 |
|------|------|
| 썸네일 | 쇼츠 대표 이미지 미리보기 |
| 제목 | AI 추천 제목 또는 수정된 제목 |
| 생성일 | 쇼츠 생성 날짜/시간 |
| 원본 영상 정보 | 입력 영상 제목 또는 파일명 |
| 상태 | 처리 중 / 완료 / 오류 |

### 기능
| 기능 | 설명 |
|------|------|
| 쇼츠 미리보기 재생 | 인라인 또는 모달로 영상 재생 |
| 편집 화면 이동 | 해당 쇼츠 편집 화면으로 진입 |
| 재생성 | 쇼츠 다시 만들기 (이용권 50% 소모) |
| 다운로드 | 로컬 저장 |
| 유튜브 업로드 | 직접 유튜브 채널에 업로드 |

### 지원 플랜
- 전 플랜 지원 (로그인 필수)

## F-E-02. 쇼츠 재생성

### 개요
결과물이 불만족스러울 경우 AI가 동일 영상에서 다른 구간을 추출하여 쇼츠를 새로 생성한다.

### 접근 경로
- 결과물 확인 화면 → "재생성하기" 버튼
- 내 프로젝트 페이지 → 각 프로젝트 카드 내 버튼

### 이용권 정책
| 구분 | 소모량 |
|------|--------|
| 최초 생성 | 원본 영상 전체 시간(분) |
| 재생성 | 최초 소모량의 **50%** |

### 동작 방식
1. "재생성하기" 버튼 클릭
2. 이용권 잔량 및 차감량 확인 팝업 표시
3. 사용자 확인 → AI가 다른 하이라이트 구간으로 재추출
4. 새 결과물 생성 완료

### 목적
- 첫 결과물이 마음에 들지 않을 경우 저렴한 비용으로 재시도 가능
- 다양한 하이라이트 구간 중 최적 결과물 선택

### 지원 플랜
- 전 플랜 지원

## F-E-03. 쇼츠 다운로드

### 개요
생성 완료된 쇼츠 영상 파일을 사용자 기기에 저장한다.

### 다운로드 옵션
| 옵션 | 설명 |
|------|------|
| 원본 화질 | 원본 영상의 해상도 그대로 저장 |
| 압축 화질 | 파일 크기 축소 (무료 플랜 제한 추정) |

### 파일 형식
- MP4 (추정)

### 제약 조건
- **원본 화질 다운로드**: Basic 이상 전용
- 무료 플랜: 다운로드 가능 여부 및 화질 공개 명시 불분명

### 지원 플랜
| 무료 | Basic | Pro | Max |
|------|-------|-----|-----|
| △ | ✅ | ✅ | ✅ |

## F-E-04. 유튜브 자동 업로드

### 개요
편집 완료된 쇼츠를 알파컷 내에서 직접 유튜브 채널에 업로드한다.
AI가 제목, 태그, 카테고리, 설명글을 자동 생성하여 함께 제공한다.

### 자동 생성 항목
| 항목 | 설명 |
|------|------|
| 제목 | 클릭 유도형 제목 AI 추천 |
| 해시태그 | 알고리즘 노출 최적화 태그 |
| 카테고리 | 유튜브 카테고리 자동 분류 |
| 설명글 | 영상 내용 기반 설명 텍스트 |

### 동작 방식
1. 편집 완료 후 "유튜브 업로드" 버튼 클릭
2. 유튜브 계정 연동 확인 (OAuth 인증)
3. AI 추천 제목·태그·카테고리·설명글 표시
4. 사용자가 내용 수정 후 "업로드" 최종 확인
5. 유튜브 채널에 쇼츠 업로드 완료

### 유의 사항
- 유튜브 계정 연동 필요 (최초 1회)
- 업로드 전 반드시 내용 검토 권장

### 지원 플랜
- 전 플랜 지원 (공식 가이드 내 안내)
