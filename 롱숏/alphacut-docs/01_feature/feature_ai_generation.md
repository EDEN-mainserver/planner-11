## F-B-01. AI 하이라이트 추출

### 개요
AI가 원본 영상 전체를 자동 분석하여 쇼츠로 만들 핵심 구간을 선별한다.

### 처리 방식
| 분석 기준 | 설명 |
|-----------|------|
| 시청자 관심 유도 장면 | 반응을 이끌어낼 수 있는 장면 우선 선별 |
| 정보 밀도 높은 부분 | 중요한 정보가 집약된 구간 식별 |
| 감정적 몰입도 | 감정 곡선이 높은 순간 (웃음, 놀람, 공감 등) 탐지 |
| 발화 분석 | 음성 내용 기반 주제 중요도 판별 |

### 생성 비율
| 원본 영상 길이 | 생성 쇼츠 수 |
|---------------|-------------|
| 2분 | 약 1개 |
| 10분 | 약 5개 |
| 20분 | 약 10개 |
| N분 | N ÷ 2개 (편차 있음) |

### 지원 플랜
- 무료 / Basic / Pro / Max 전 플랜 지원

### 처리 순서
1. 영상 입력 완료
2. 오디오 트랜스크립션 (음성 → 텍스트 변환)
3. 장면 중요도 스코어링
4. 상위 N개 구간 추출
5. 각 구간에 자막·템플릿 자동 적용
6. 쇼츠 결과물 생성 완료

## F-B-02. AI 자동 자막 생성

### 개요
음성 인식(STT) 기술로 쇼츠 영상의 대사를 자동으로 텍스트 자막으로 변환하여 삽입한다.

### 동작 방식
1. 생성된 쇼츠 클립의 오디오 트랙 분석
2. 발화 구간 탐지 → 텍스트 변환
3. 타임라인에 자막 자동 배치
4. 선택된 템플릿 스타일 적용

### 자막 편집 기능 (후처리)
| 편집 항목 | 설명 |
|-----------|------|
| 오타 수정 | 자막 텍스트 직접 수정 |
| 문장 병합 | 짧게 쪼개진 자막 합치기 |
| 타임라인 싱크 조절 | 자막 시작/끝 시간 미세 조정 |
| 스타일 변경 | 폰트, 크기, 색상, 위치 변경 |

### 지원 언어
- 한국어 (주요 지원)
- 추가 언어: 공개 명시 없음

### 지원 플랜
- 전 플랜 지원 (무료 포함)

## F-B-03. 무음 구간 자동 제거

### 개요
영상 내 말이 없거나 소리가 없는 구간(공백/침묵)을 AI가 자동 감지하여 제거함으로써 속도감 있는 쇼츠를 만든다.

### 동작 방식
1. 오디오 파형 분석
2. 발화 없는 구간(무음) 자동 마킹
3. 해당 구간 컷 아웃 처리
4. 자연스러운 이어붙이기

### 효과
- 지루한 구간 제거로 시청 이탈률 감소
- 동일 내용을 더 짧은 시간에 전달
- 쇼츠 특성(빠른 호흡)에 최적화

### 제약 조건
- **Pro 멤버십 이상** 전용 기능
- 무음 감지 민감도 조절 가능 여부: 공개 명시 없음

### 지원 플랜
| 무료 | Basic | Pro | Max |
|------|-------|-----|-----|
| ❌ | ❌ | ✅ | ✅ |

## F-B-04. AI 후킹 보이스 (나레이션)

### 개요
영상 인트로에 전문 성우 스타일의 AI 음성 나레이션을 자동으로 생성하여 삽입한다.

### 동작 방식
1. 영상의 핵심 내용 요약 텍스트 자동 생성
2. TTS(Text-to-Speech) AI 음성으로 변환
3. 영상 시작 부분에 나레이션 오디오 삽입

### 목적
- 시청자의 첫 3~5초 이탈 방지
- 영상 몰입감 2배 향상 (공식 표현)
- 전문 성우 없이 고퀄리티 인트로 연출

### 설정 옵션
| 옵션 | 내용 |
|------|------|
| ON/OFF 토글 | 후킹 보이스 사용 여부 선택 |
| 음성 스타일 | 선택 가능 여부 공개 명시 없음 |

### 지원 플랜
- 전 플랜 지원 (편집 화면 내 설정)

## F-B-05. 첫 3초 후킹 (하이라이트 배치)

### 개요
영상 전체 중 가장 임팩트 있는 클라이맥스 장면을 AI가 자동으로 찾아내어 영상 첫 3초에 배치한다.

### 동작 방식
1. 추출된 쇼츠 클립 전체 분석
2. 가장 몰입도 높은 장면(클라이맥스) 자동 식별
3. 해당 장면을 영상 맨 앞 3초로 이동 배치
4. 이후 본편 영상 이어지도록 재구성

### 목적
- 유튜브 Shorts / TikTok 알고리즘 대응
- 시청 유지율(Retention Rate) 향상
- 첫 3초 이탈률 최소화

### 자동화 수준
- 완전 자동 (별도 사용자 조작 불필요)
- 결과물 미리보기 후 수동 변경 가능

### 지원 플랜
- 전 플랜 지원

## F-B-06. AI 제목 자동 추천

### 개요
생성된 쇼츠에 클릭을 유도하는 제목을 AI가 자동으로 추천한다.
유튜브 업로드 시 태그, 카테고리, 설명글까지 함께 자동 생성된다.

### 생성 항목
| 항목 | 설명 |
|------|------|
| 쇼츠 제목 | 클릭을 유도하는 후킹형 제목 |
| 해시태그 | 알고리즘 노출 최적화 태그 |
| 카테고리 | 유튜브 업로드용 카테고리 자동 분류 |
| 설명글 | 영상 내용 요약 설명 텍스트 |

### 위치
- 결과물 확인 화면 내 편집 메뉴
- 유튜브 업로드 연동 화면

### 편집 가능 여부
- 추천된 제목/태그 직접 수정 가능

### 지원 플랜
- 전 플랜 지원 (고급 기능 가이드 참조)
