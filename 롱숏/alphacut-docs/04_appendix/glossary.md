## 1. AlphaCut 전용 용어

| 용어 | 설명 |
|------|------|
| 알파컷 (AlphaCut) | AI 기반 쇼츠 자동 생성 SaaS 서비스. alphacut.video |
| 알파토픽 (AlphaTopic) | AlphaCut의 키워드 기반 유튜브 트렌드 분석 기능. Beta 서비스 |
| 알파지수 | AlphaCut AI가 계산한 영상의 종합 잠재력 점수 (알파토픽 전용 지표) |
| 이용권 | 영상 처리에 소모되는 분(min) 단위 크레딧. 플랜별 월 지급량 상이 |
| 구간 설정 | 영상 전체가 아닌 특정 시작~종료 시간만 선택하여 처리하는 기능. 이용권 절약 목적 |
| 후킹 보이스 | 영상 인트로에 삽입하는 AI 음성 나레이션. 시청자 이탈 방지 목적 |
| 첫 3초 후킹 | 영상 내 클라이맥스 장면을 자동으로 찾아 첫 3초에 배치하는 기능 |
| 재생성 | 동일 원본 영상으로 다른 하이라이트 구간의 쇼츠를 새로 생성. 이용권 50% 소모 |
| SandPaper | AlphaCut 제공 템플릿 중 하나. 거친 질감의 트렌디한 배경 스타일 |
| Minimal | AlphaCut 제공 템플릿 중 하나. 깔끔한 미니멀 스타일 |
| Vibrant | AlphaCut 제공 템플릿 중 하나. 생동감 있는 컬러풀 디자인 |
| GreenLine | AlphaCut 제공 템플릿 중 하나. 네온 그린 라인 포인트 스타일 |
| Newsletter | AlphaCut 제공 템플릿 중 하나. 정보 전달형 뉴스레터 스타일 |
| Primary Gradient | AlphaCut 브랜드 주 그라디언트. 파란-보라-핑크-오렌지 좌→우 방향 |
| 어필리에이트 | AlphaCut 추천인(파트너) 프로그램. 10% 커미션, 무제한 초대, 매월 정산 |
| 추천인 | 고유 초대 코드/링크를 통해 신규 사용자를 유입시키는 파트너 사용자 |
| 피추천인 | 추천인의 초대 링크를 통해 가입한 신규 사용자 |
| 초대 코드 | 추천인에게 발급되는 고유 알파벳/숫자 조합 코드 |
| 유튜브 채널 성장 가이드북 | AlphaCut이 제작한 66페이지 전자책. 연간 구독 시 무료 제공, 단건 ₩20,000 |
| CS 챗봇 | 모든 페이지 우하단에 고정된 실시간 고객 지원 채팅 버튼 |
| 내 프로젝트 | 로그인 사용자의 생성 쇼츠 목록 및 관리 페이지. URL: /my_project |
| 알고리즘 기여도 | 알파토픽에서 제공하는 지표. 유튜브 알고리즘이 영상을 추천한 정도 추정 수치 |
| 우선 처리 (Priority) | Max 플랜 이상에서 제공하는 AI 처리 대기열 우선권 |

## 2. 일반 용어

| 용어 | 설명 |
|------|------|
| 쇼츠 (Shorts) | 세로형(9:16) 단편 영상. 유튜브 Shorts / TikTok / Instagram Reels 포맷 |
| 숏폼 (Short-form) | 짧은 영상 콘텐츠 형식 통칭. 일반적으로 60초 이내 |
| 롱폼 (Long-form) | 긴 영상 콘텐츠. AlphaCut에서는 쇼츠 생성의 원본 영상으로 사용 |
| 하이라이트 | AI가 원본 영상에서 핵심 내용으로 선별한 구간 |
| STT | Speech-to-Text. 음성을 텍스트로 변환하는 기술. 자막 생성에 사용 |
| TTS | Text-to-Speech. 텍스트를 음성으로 변환하는 기술. 후킹 보이스에 사용 |
| 타임라인 | 영상 편집 화면 하단에 표시되는 시간 축 기반 클립 편집 영역 |
| 타임코드 | 영상 내 특정 시점을 나타내는 시간 값 (예: 00:01:23) |
| 플레이헤드 | 타임라인에서 현재 재생 위치를 나타내는 세로선 |
| 썸네일 | 영상의 대표 이미지. 플랫폼 목록에서 클릭률에 영향 |
| 시청 유지율 | Retention Rate. 시청자가 영상을 얼마나 오래 보는지를 나타내는 지표 |
