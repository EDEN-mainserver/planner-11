## 1. 버튼 컴포넌트

### 1-1. Primary CTA 버튼
- 텍스트: "무료로 시작하기" / "쇼츠로 변환하기" / "시작하기"
- 스타일: 그라디언트 배경 (보라 → 핑크), 흰색 텍스트, 라운드 모서리
- 상태: default / hover / loading (처리 중 스피너)
- 사용 위치: Hero 섹션, 요금제 카드, CTA 섹션

### 1-2. Secondary 버튼
- 텍스트: "로그인", "문의하기", "구매하기"
- 스타일: 외곽선(Outline) 또는 어두운 배경, 흰색 텍스트
- 상태: default / hover

### 1-3. 텍스트 링크 버튼
- 텍스트: "멤버십 사용 안내", "추가 검색", "새로운 키워드로 검색하기"
- 스타일: 밑줄 또는 색상 강조 텍스트

### 1-4. 아이콘+텍스트 버튼
- 예시: "유튜브 링크 🔗", "파일 업로드 ⬆", "재생성하기"
- 사용 위치: 입력 탭, 편집 화면 액션 버튼

### 1-5. 토글 버튼 (월간/연간)
- 그룹형 토글: 월간 | 연간 (선택된 항목 강조 표시)
- 사용 위치: 요금제 페이지 상단

### 1-6. 플로팅 버튼
- "CS 챗봇과 대화하기" — 화면 하단 우측 고정
- 원형 아이콘 버튼, 클릭 시 채팅 UI 슬라이드업

### 1-7. 편집 타임라인 버튼 (인터페이스 전용)
| 버튼 | 기능 |
|------|------|
| [ (왼쪽 삭제) | 플레이바 기준 좌측 구간 삭제 |
| ] (오른쪽 삭제) | 플레이바 기준 우측 구간 삭제 |
| | (분할) | 현재 구간 분할 |
| 🗑️ (삭제) | 선택 구간 제거 |
| + 돋보기 | 타임라인 확대 |
| - 돋보기 | 타임라인 축소 |
| << / >> | 먼 구간 탐색 이동 |
| 무음 구간 제거 | AI 무음 자동 삭제 (Pro+) |

## 2. 입력 폼 컴포넌트

### 2-1. YouTube URL 입력 필드
- 타입: text input
- 플레이스홀더: "유튜브 영상 링크를 입력하세요 (예: https://www.youtube.com/watch?v=...)"
- 아이콘: 링크 아이콘 (좌측 내부)
- 제출: "쇼츠로 변환하기 →" 버튼 (하단 또는 우측)
- 유효성: URL 형식 검사

### 2-2. 파일 업로드 영역
- 타입: 드래그 앤 드롭 + 클릭 업로드
- 지원 형식: mp4, mov 등
- 상태: idle / 드래그 오버 / 업로드 중(프로그레스바) / 완료
- 주의: 업로드 중 탭 닫기 금지 경고

### 2-3. 탭 전환 컴포넌트 (입력 방식)
- 탭1: "🔗 유튜브 링크" (기본 활성)
- 탭2: "⬆ 파일 업로드" + "업로드" 뱃지
- 스타일: 선택된 탭 그라디언트 배경, 미선택 탭 어두운 배경

### 2-4. 세부 설정 모달 (생성 전)
| 설정 항목 | UI 타입 | 옵션 |
|----------|---------|------|
| 구간 설정 | 슬라이더 (2개 핸들) | 타임라인 시작~끝 드래그 |
| 쇼츠 길이 | 라디오 버튼 4개 | 15~30초 / 30~60초 / 60~90초 / 90~120초 |
| 제목 언어 | 드롭다운 Select | 한국어 / 영어 |
| 자막 언어 | 드롭다운 Select | 한국어 / 영어 / 자동 |
| 템플릿 선택 | 썸네일 그리드 선택 | 내 템플릿 목록 |

### 2-5. 제목 재생성 입력창 (편집 화면)
- 버튼: "AI에게 제목 추천 받기"
- 확장 옵션: 드롭다운 → "지시사항과 함께 생성하기"
- 프롬프트 텍스트 입력 (자유 입력)
- 결과: 5개 제목 목록 → 클릭 선택

### 2-6. 알파토픽 검색창
- 타입: text input
- 플레이스홀더: 키워드 입력 (예: "재테크", "다이어트 식단")
- 버튼: "검색" / "추가 검색" / "새로운 키워드로 검색하기"
- 결과 내 검색: 하단 별도 텍스트 필터 입력창

## 3. 카드 컴포넌트

### 3-1. 요금제 카드
```
┌──────────────────┐
│  플랜명 (heading) │
│  가격 (큰 폰트)   │
│  /월              │
│  연간 금액        │
│  변환 가능 시간   │
│  [시작하기] 버튼  │
│  ───────────────  │
│  ✓ 기능 리스트    │
│  ✓ 기능 리스트    │
└──────────────────┘
```
- 상태: default / 추천(highlighted)
- 호버 시 테두리 강조

### 3-2. 쇼츠 결과 카드 (프로젝트 목록)
- 썸네일 이미지 (9:16 비율)
- 쇼츠 제목 텍스트
- 상태 배지: 처리 중 / 완료 / 오류
- 버튼: 편집 / 다운로드 / 재생성

### 3-3. 크리에이터 후기 카드 (슬라이더)
- 프로필 이미지 (원형)
- 이름 + 채널명
- 후기 텍스트
- 별점 또는 강조 문구

### 3-4. 블로그 아티클 카드
- 썸네일 이미지
- 카테고리 배지 (사용 가이드 / 아티클)
- 제목
- 요약 텍스트
- 작성자 + 날짜

### 3-5. 알파토픽 영상 결과 카드
- 썸네일
- 영상 제목
- 채널명 + 구독자 수
- 4가지 지표 배지: 기여도 / 성과도 / 반응도 / 알파지수
- 클릭 시 팝업 모달

## 4. 모달 컴포넌트

### 4-1. 로그인/회원가입 모달
- Google OAuth 버튼 (Google 로고 + "Google 계정으로 계속하기")
- 닫기(X) 버튼
- 바텀: 이용약관 / 개인정보처리방침 동의 안내

### 4-2. 세부 설정 모달 (쇼츠 생성 전)
- 트리거: "[확인] 버튼 전 [세부 설정] 클릭"
- 내용: 구간 슬라이더, 길이 선택, 언어, 템플릿
- 영상 썸네일 미리보기 포함
- 버튼: [확인] / [취소]

### 4-3. AI 나레이션 생성 모달
- 텍스트 입력창 (멘트 입력)
- 성우 선택 (라디오 버튼 또는 드롭다운)
- [변환하기] 버튼
- 결과: 속도/음량 조절 슬라이더 + [쇼츠에 추가하기] 버튼

### 4-4. 알파토픽 영상 상세 팝업
- 썸네일 대형
- 제목, 정확한 게시일, 채널 구독자 수
- 4개 지표 상세 수치
- 닫기 버튼

### 4-5. 404 에러 페이지
- 배경: 검은색
- "404 Page Not Found"
- "The page you requested does not exist or has been moved."
- [Go to Home] 버튼 (파란색 라운드)

## 5. 네비게이션 컴포넌트

### 5-1. GNB (Global Navigation Bar)
```
[로고(아이콘+텍스트)] [가격안내] [사용가이드] [FAQ] [무료도구▼]  [로그인] [무료로시작하기]
```
- 상단 고정 (sticky)
- 배경: 어두운 반투명 (blur)
- 로그인 후: [로그인] → [계정 아이콘 / 닉네임]으로 변경 예상

### 5-2. 무료 도구 드롭다운 메뉴
- 버튼: "무료 도구 ▼"
- 드롭다운 항목: 알파토픽 등 무료 기능 목록

### 5-3. 모바일 햄버거 메뉴
- 트리거: 우측 상단 "메뉴" 버튼 (≡ 아이콘)
- 내용: GNB 링크 세로 배열 + CTA 버튼
- 슬라이드 인 애니메이션 (우측에서)

### 5-4. 로그인 후 사이드바/상단 메뉴 (추정)
- 항목: 내 프로젝트, 템플릿, 알파토픽, 요금제, 설정
- 현재 페이지 강조 표시

### 6-1. 입력 방식 탭 (Hero)

- 사용 가이드 | 아티클

### 6-4. 편집 패널 탭 (우측 패널)
- 텍스트 | 레이아웃 | 자막 | 요소 | 배경 | 썸네일 | AI후킹

### 6-5. 알파토픽 필터 탭
- 추천 템플릿 | 내 템플릿 (템플릿 페이지)

## 7. 슬라이더/캐러셀 컴포넌트

### 7-1. 크리에이터 쇼츠 슬라이딩 갤러리
- 무한 루프 가로 스크롤 (자동 재생)
- 쇼츠 썸네일 + 크리에이터 프로필 세트
- 두 줄 배치

### 7-2. 크리에이터 후기 캐러셀
- Previous / Next 화살표 버튼
- 슬라이드 인디케이터 (점 또는 선)

### 7-3. 구간 설정 슬라이더 (세부 설정 모달)
- 범위 슬라이더 (Range Slider)
- 시작 핸들 + 끝 핸들 2개
- 타임라인 프리뷰 위에 오버레이

### 7-4. 예상 수익 계산기 슬라이더 (추천인 페이지)
- 단일 핸들 슬라이더
- 초대 인원 입력 (1명 ~ 1,000명)
- 월/연 예상 수익 실시간 계산 표시

## 8. 아코디언 컴포넌트

### 8-1. FAQ 아코디언 (홈/요금제)
- 질문 헤딩 + 펼침 버튼 (+ / ▼)
- 클릭 시 답변 영역 슬라이드 다운
- 한 번에 1개만 열리는 방식 (Exclusive) 또는 다중 가능

## 9. 뱃지/태그 컴포넌트

### 9-1. 소셜 증명 뱃지
- "# 유튜버 10,000명의 검증"
- "# 서울대생 제작/운영"
- "# 탑급 MCN 파트너사 도입"
- 스타일: 작은 둥근 뱃지, 배경 어두운 반투명

### 9-2. 할인 뱃지
- "16% 할인" — 연간 탭 옆
- 초록색 또는 강조색 배경

### 9-3. 알파토픽 지표 배지
- Great / Good / Average / Low
- 색상: Great=초록, Good=파랑, Average=노랑, Low=빨강

### 9-4. 멤버십 전용 기능 뱃지
- "Pro+" 또는 잠금 아이콘 — 무음 구간 제거, 커스텀 폰트 등
- 비활성 기능에 오버레이

### 9-5. 상태 배지 (편집 화면)
- 🟢 초록불 — AI 후킹 적용됨
- 처리 중 / 완료 / 오류 상태

## 10. 알림/배너 컴포넌트

### 10-1. 상단 프로모션 배너
- "🎁 지금 회원가입하고 무료로 쇼츠를 제작해보세요!"
- 위치: GNB 바로 아래 (전체 폭)
- 색상: 강조색 배경 (보라/핑크)
- 닫기 버튼 없음 (항상 노출)

### 10-2. 인라인 경고/안내 메시지
- "⚠️ 업로드 중 창을 닫지 마세요!"
- "⛔ 하이라이트 훅과 AI 나레이션은 동시 사용 불가"
- 스타일: 아이콘 + 배경색 강조 박스

### 10-3. 이용권 소진 알림 (추정)
- 잔여 시간 0분 도달 시 팝업 또는 인라인 알림
- CTA: "요금제 업그레이드하기" 버튼

## 11. 레이아웃 컴포넌트 (편집 화면)

### 11-1. 미리보기 캔버스
- 9:16 세로 비율 고정
- 드래그로 텍스트/요소 위치 이동
- 더블클릭: 텍스트 인라인 편집

### 11-2. 우측 패널 (설정)
- 탭형 패널: 텍스트 / 레이아웃 / 자막 / 요소 / 배경 / AI후킹
- 스크롤 가능

### 11-3. 하단 타임라인
- 타임라인 트랙: 영상 구간 (활성화=밝음, 비활성=남색)
- 플레이바 (재생 헤드)
- 자막 트랙 (별도 레인)
- AI후킹 탭 전환 시 후킹 전용 타임라인

### 11-4. 데드존 오버레이
- "데드존 표시" 토글 → 유튜브 UI 버튼 가이드 표시
- "유튜브 레이아웃 표시" 토글

### 11-5. 듀얼 레이아웃
- 상/하 분할: 50:50 / 30:70 / 70:30
- 각 프레임 독립 조작 가능
- 구간별 다른 레이아웃 적용 가능

## 12. 자막 컴포넌트 (편집 화면)

| 기능 | UI 요소 |
|------|---------|
| 자막 활성화 | 체크박스 토글 |
| 오타 수정 | 클릭 → 인라인 텍스트 편집 |
| 자막 추가 | 줄 사이 [+추가] 버튼 (hover 시 노출) |
| 자막 병합 | 줄 사이 [병합] 버튼 |
| 자막 삭제 | 줄 우측 상단 🗑️ 아이콘 |
| 폰트/크기 | 드롭다운 + 숫자 입력 |
| 색상/테두리 | 컬러 피커 |
| 자간/줄간격 | 숫자 슬라이더 |
| 한줄 글자수 | 5단계 슬라이더 |
| 애니메이션 | 6종 라디오/버튼 선택 |
| 배경색 | 컬러 피커 (기존 자막 덮기 용도) |
| 위치 이동 | 미리보기에서 드래그 |
| 싱크 조절 | 타임라인 자막 바 드래그 |
| 빈 공간 추가 | 타임라인 빈 공간 드래그 |
