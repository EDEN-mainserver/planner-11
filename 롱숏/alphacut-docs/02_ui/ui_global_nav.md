## 1. 글로벌 네비게이션 바 (GNB)

### 1.1 레이아웃 구조
- 좌측: 로고 (아이콘 + ALPHACUT 텍스트) → 홈(/) 링크
- 중앙: 메뉴 링크 4개
- 우측: 로그인 버튼 + 무료로 시작하기 버튼

### 1.2 레이아웃 속성
| 속성 | 값 |
|------|----|
| 위치 | 화면 상단 sticky 고정 |
| 높이 | 약 60px |
| 배경 | 반투명 다크 (blur backdrop) |
| 하단 테두리 | 반투명 흰색 1px 구분선 |

### 1.3 메뉴 링크 목록
| 메뉴명 | 링크 | 비고 |
|--------|------|------|
| 가격 안내 | /pricing | 텍스트 링크 |
| 사용 가이드 | /blog?category=guide | 텍스트 링크 |
| FAQ | #faq-title | 앵커 링크 |
| 무료 도구 | 드롭다운 | 알파토픽 등 하위 메뉴 |

### 1.4 우측 버튼 영역
| 버튼 | 스타일 | 동작 |
|------|--------|------|
| 로그인 | Outline (흰 테두리+흰 텍스트) | 로그인 화면 이동 |
| 무료로 시작하기 | Primary Gradient 필 | 회원가입 화면 이동 |

### 1.5 모바일 대응
| 요소 | 내용 |
|------|------|
| 메뉴 | 햄버거 버튼(≡) 으로 대체 |
| 버튼 텍스트 | 무료 시작 (축약) |
| 메뉴 오픈 방식 | Drawer 또는 드롭다운 |

## 2. 상단 알림 배너

| 속성 | 값 |
|------|----|
| 위치 | 화면 우상단 플로팅 |
| 모양 | Pill (둥근 캡슐형) |
| 배경 | 오렌지→핑크 그라디언트 |
