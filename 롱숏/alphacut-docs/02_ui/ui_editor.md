# 편집 화면 UI 명세 (UI: Editor)

> 파일 위치: alphacut-docs/02_ui/ui_editor.md
> 접근 경로: 내 프로젝트 → 쇼츠 카드 → 편집 버튼
> 작성일: 2026-04-16

> 주의: 로그인 후 내부 화면으로 직접 접근 불가 — 블로그 스크린샷 및 공개 자료 기반 분석

---

## 1. 전체 레이아웃 구조

```
┌─────────────────────────────────────────────────────────┐
│                    상단 툴바 (Toolbar)                   │
├──────────────┬──────────────────────┬───────────────────┤
│              │                      │                   │
│  좌측 패널   │   중앙 미리보기      │   우측 속성 패널  │
│  (쇼츠 목록) │   (영상 캔버스)      │   (편집 옵션)     │
│              │                      │                   │
├──────────────┴──────────────────────┴───────────────────┤
│                  하단 타임라인 (Timeline)                │
└─────────────────────────────────────────────────────────┘
```

---

## 2. 상단 툴바

| 요소 | 위치 | 내용 |
|------|------|------|
| 로고 | 좌측 | ALPHACUT 로고 (홈 이동) |
| 프로젝트명 | 중앙 좌측 | 현재 편집 중인 쇼츠 제목 |
| 탭 메뉴 | 중앙 | 100분 / 내 프로젝트 / 영상 설정 / 쇼츠 생성하기 / 가져오기 등 |
| 템플릿 버튼 | 우측 | 템플릿 선택 패널 열기 |
| 저장 버튼 | 우측 | 현재 편집 저장 |
| 다운로드 버튼 | 우측 | 쇼츠 다운로드 |
| 업로드 버튼 | 우측 | 유튜브 업로드 |

---

## 3. 좌측 패널 — 쇼츠 목록

| 요소 | 내용 |
|------|------|
| 썸네일 목록 | 생성된 쇼츠 클립 썸네일 세로 나열 |
| 선택 상태 | 현재 편집 중인 클립 하이라이트 |
| 클립 번호 | 1, 2, 3... 순서 표시 |
| 재생 시간 | 각 클립 길이 표시 |

---

## 4. 중앙 미리보기 (영상 캔버스)

| 요소 | 내용 |
|------|------|
| 비율 | 9:16 세로형 (쇼츠 표준) |
| 배경 | 선택된 템플릿 배경 실시간 반영 |
| 영상 레이어 | 원본 영상 클립 표시 |
| 자막 레이어 | AI 생성 자막 오버레이 |
| 텍스트 레이어 | 제목 텍스트 오버레이 |
| 재생 컨트롤 | 재생/일시정지, 음소거, 타임코드 표시 |
| 요소 선택 | 캔버스 내 텍스트/이미지 클릭 시 선택 핸들 표시 |

---

## 5. 우측 속성 패널

### 5.1 쇼츠 제목 편집
| 요소 | 내용 |
|------|------|
| 레이블 | 쇼츠 제목 |
| 입력 필드 | AI 추천 제목 표시, 직접 수정 가능 |
| AI 추천 버튼 | 새 제목 AI 재생성 |

### 5.2 텍스트 스타일 편집
| 요소 | 내용 |
|------|------|
| 폰트 선택 드롭다운 | 폰트 목록 펼침 선택 |
| 폰트 크기 | 슬라이더 또는 직접 입력 |
| 글자 색상 | 컬러 피커 |
| 텍스트 정렬 | 좌/중/우 정렬 아이콘 버튼 |

### 5.3 레이아웃 설정
| 요소 | 내용 |
|------|------|
| 영상 비율 | 9:16 / 1:1 / 16:9 선택 |
| 영상 위치 | 화면 내 영상 영역 위치 조정 |
| 자막 위치 | 상단/하단/오버레이 |

### 5.4 AI 후킹 설정
| 요소 | 내용 |
|------|------|
| 후킹 보이스 토글 | ON/OFF |
| 첫 3초 후킹 토글 | ON/OFF |
| 무음 제거 토글 | ON/OFF (Pro 이상) |

---

## 6. 하단 타임라인

| 요소 | 내용 |
|------|------|
| 타임라인 트랙 | 시간 축 기준 클립 표시 |
| 영상 트랙 | 영상 클립 블록 |
| 자막 트랙 | 자막 블록 (타임코드 기반) |
| 오디오 트랙 | 오디오 파형 표시 |
| 플레이헤드 | 현재 재생 위치 표시 세로선 |
| 줌 컨트롤 | 타임라인 확대/축소 |
| 컷 도구 | 특정 지점 분할 |
| 삭제 도구 | 선택 구간 제거 |

---

## 7. 자막 편집 모드

| 요소 | 내용 |
|------|------|
| 자막 목록 | 타임코드 + 자막 텍스트 리스트 |
| 텍스트 수정 | 인라인 편집 |
| 병합 버튼 | 인접 자막 합치기 |
| 타임코드 조정 | 시작/끝 시간 직접 수정 |
| 스타일 일괄 적용 | 전체 자막에 스타일 한번에 적용 |

## 8. 편집 화면 컬러 & 스타일

| 영역 | 배경 |
|------|------|
| 툴바 | 짙은 다크 (#0D0D14 계열) |
| 좌측 패널 | 다크 (#111118) |
| 중앙 캔버스 배경 | 중간 다크 (#1A1A25) |
| 우측 패널 | 다크 (#111118) |
| 타임라인 | 짙은 다크 (#0D0D14) |
| 액센트 | 그라디언트 (그라디언트 버튼, 선택 핸들 등) |
