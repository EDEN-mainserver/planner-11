## 2. 결제 주기 토글

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 토글 컨테이너 | 월간 / 연간 2개 옵션 | 라운드 Pill 컨테이너 |
| 월간 (비활성) | 월간 텍스트 | 투명 배경, 회색 텍스트 |
| 연간 (활성/기본) | 연간 텍스트 + 16% 할인 배지 | 다크 필 배경, 흰색 텍스트 |
| 할인 배지 | 16% 할인 | 초록 배경, 흰색 텍스트, 라운드 |

### 연간 선택 시 추가 안내 배너
- 내용: 연간 멤버십 가입 시 2만원 상당의 전자책 무료 제공
- 스타일: 가로로 긴 다크 배지형 텍스트 배너 (중앙 정렬)

## 3. 플랜 카드 영역 (4열 그리드)

### 3.1 전체 레이아웃
- 데스크톱: 4열 그리드 (무료 | Basic | Pro | Max)
- 태블릿: 2열, 모바일: 1열 (추정)
- Pro 플랜 카드에만 상단 '추천' 배지 표시

### 3.2 무료 플랜 카드
| 요소 | 내용 |
|------|------|
| 테두리 색상 | 오렌지 |
| 아이콘 | 선물 상자 아이콘 (오렌지 배경 흰색 아이콘) |
| 제목 | 지금 가입하면 |
| 강조 문구 | 30분 무료 (오렌지 대형 텍스트) |
| 서브텍스트 | Google 계정으로 간편 가입 |
| CTA 버튼 | 무료로 시작하기 (오렌지 배경) |
| 기능 목록 | 회원가입 즉시 30분 제공 / 템플릿 기능 / 기본 편집 기능 / 신용카드 등록 불필요 |

## 6. 기능 목록 스타일 (공통)

- 각 기능 앞에 체크마크(✓) 아이콘 (초록)
- 미지원 기능: 표시 안 함 (목록에서 제외)
- 폰트: Regular ~14px, 연한 회색
