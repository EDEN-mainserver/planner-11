## 1. 서비스 개요

AlphaCut은 롱폼 영상(유튜브 링크 또는 파일)을 입력하면  
AI가 자동으로 핵심 하이라이트를 추출하여 쇼츠(숏폼 영상)를 생성하는 SaaS 서비스다.

| 항목 | 내용 |
|------|------|
| 서비스명 | AlphaCut (알파컷) |
| 핵심 가치 | 링크 1개 입력 → 쇼츠 N편 자동 생성 |
| 가격 포인트 | 쇼츠 1편당 약 300원 수준 |
| 주요 타겟 | 유튜버, 1인 크리에이터, SNS 인플루언서, 콘텐츠 마케터 |
| 가입 방법 | Google OAuth (신용카드 불필요) |
| 무료 혜택 | 가입 즉시 30분 무료 이용권 지급 |

## F-G-01. 회원가입

### 개요
Google 계정을 통한 OAuth 방식으로 간편하게 AlphaCut 계정을 생성한다.

### 가입 방식
- **Google OAuth 2.0** (소셜 로그인 전용)
- 별도 이메일/비밀번호 등록 불필요
- 신용카드 등록 불필요

### 진입 경로
| 경로 | 설명 |
|------|------|
| 홈 Hero CTA | "무료로 시작하기" 버튼 클릭 |
| 상단 네비 | "무료로 시작하기" 버튼 클릭 |
| 쇼츠 변환 시도 | 비로그인 상태에서 변환 버튼 클릭 시 유도 |
| 상단 알림 배너 | "지금 회원가입하고 무료로 쇼츠를 제작해보세요!" |
| 요금제 페이지 | 각 플랜 "시작하기" 버튼 클릭 |

### 가입 완료 시 혜택
| 혜택 | 내용 |
|------|------|
| 무료 이용권 | **30분** 즉시 지급 |
| 기본 기능 | 템플릿, 기본 편집, 쇼츠 생성 즉시 이용 가능 |

### 처리 흐름
1. "무료로 시작하기" 클릭
2. Google 계정 선택 화면 팝업
3. Google 계정 선택 및 권한 허용
4. AlphaCut 회원 등록 완료
5. 홈 또는 내 프로젝트 화면으로 이동
6. 30분 무료 이용권 자동 지급 알림

## F-G-02. 로그인

### 개요
기존 AlphaCut 계정에 Google OAuth로 인증하여 서비스에 접근한다.

### 로그인 방식
- Google OAuth 2.0 (SSO)
- 별도 비밀번호 불필요

### 진입 경로
| 경로 | 설명 |
|------|------|
| 상단 네비 | "로그인" 버튼 클릭 |
| 비로그인 상태에서 기능 사용 시도 | 자동 로그인 화면으로 리다이렉트 |

### 처리 흐름
1. "로그인" 버튼 클릭
2. Google 계정 선택 화면 팝업
3. 계정 선택 → 인증 완료
4. 이전 페이지 또는 내 프로젝트 화면으로 이동

### 세션 유지
- 세션 유지 기간: 공개 명시 없음
- 자동 로그인: Google 계정 인증 상태 유지 시 자동 처리

## F-A-01. 유튜브 링크 입력

### 개요
사용자가 YouTube 영상 URL을 입력하면 AI가 해당 영상을 원격으로 분석하여 쇼츠를 생성한다.

### UI 위치
- 홈 페이지(/) Hero 섹션
- 탭 UI → "유튜브 링크" 탭 선택 상태

### 입력 요소
| 요소 | 설명 |
|------|------|
| 텍스트 입력 필드 | YouTube URL 직접 타이핑 또는 붙여넣기 |
| 플레이스홀더 | `유튜브 영상 링크를 입력하세요 (예: https://www.youtube.com/watch?v=...)` |
| 제출 버튼 | "쇼츠로 변환하기 →" |

### 허용 URL 형식
- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`

### 처리 흐름
1. 사용자가 URL 입력
2. "쇼츠로 변환하기" 버튼 클릭
3. 로그인 상태 확인
   - 비로그인 → 로그인/회원가입 화면으로 이동
   - 로그인 → 이용권 잔량 확인
4. 이용권 충분 → 구간 설정 옵션 제공 후 AI 처리 시작
5. 이용권 부족 → 요금제 구매 유도 팝업 표시

### 이용권 차감 규칙
- 원본 영상 재생 시간(분) 단위로 차감
- 예: 10분짜리 영상 → 이용권 10분 소모
- 구간 설정 시 → 선택된 구간(분)만 소모

### 제약 조건
- 로그인 필수
- 이용권 잔량 이상의 영상은 처리 불가
- 비공개/연령 제한 영상은 처리 불가 (추정)

## F-A-02. 파일 업로드

### 개요
사용자가 로컬에 저장된 동영상 파일을 직접 업로드하여 쇼츠를 생성한다.

### UI 위치
- 홈 페이지(/) Hero 섹션
- 탭 UI → "파일 업로드" 탭 선택 상태

### 입력 요소
| 요소 | 설명 |
|------|------|
| 파일 선택 영역 | 클릭 또는 드래그 앤 드롭으로 파일 선택 |
| 업로드 아이콘 | ↑ 아이콘 표시 |

### 처리 흐름
1. 사용자가 "파일 업로드" 탭 클릭
2. 파일 선택 (탐색기 또는 드래그 앤 드롭)
3. 파일 업로드 진행 (진행률 표시 추정)
4. 업로드 완료 → AI 처리 시작 (유튜브 링크 방식과 동일)

### 이용권 차감 규칙
- 업로드한 파일의 재생 시간(분) 단위로 차감
- 유튜브 링크 방식과 동일한 정책 적용

### 제약 조건
- 로그인 필수
- 지원 파일 형식: 동영상 파일 (상세 포맷 공개 명시 없음)
- 파일 크기 제한: 공개 명시 없음

## F-A-03. 구간 설정 (선택적 기능)

### 개요
영상 전체가 아닌 특정 구간만 선택하여 처리함으로써 이용권 소모를 절약한다.

### UI 위치
- 영상 입력 후 AI 처리 시작 전 단계 (추정)
- 블로그 가이드: "이용권 아끼는 구간 설정 꿀팁" 참고

### 입력 요소
| 요소 | 설명 |
|------|------|
| 시작 시간 | 처리 시작 지점 설정 (분:초) |
| 종료 시간 | 처리 종료 지점 설정 (분:초) |

### 처리 흐름
1. 영상 입력 완료 후 구간 설정 화면 진입
2. 시작/종료 시간 입력 또는 슬라이더 조작
3. 선택 구간 확인 후 AI 처리 시작

### 이용권 절약 효과
| 원본 영상 | 선택 구간 | 소모 이용권 |
|-----------|-----------|-------------|
| 30분 영상 | 10분 구간 | 10분 소모 |
| 60분 영상 | 20분 구간 | 20분 소모 |

### 제약 조건
- 선택 구간 내에서만 쇼츠 생성
- 구간이 짧을수록 생성 쇼츠 수 감소

## F-B-01. AI 하이라이트 추출

### 개요
AI가 원본 영상 전체를 자동 분석하여 쇼츠로 만들 핵심 구간을 선별한다.

### 처리 방식
| 분석 기준 | 설명 |
|-----------|------|
| 시청자 관심 유도 장면 | 반응을 이끌어낼 수 있는 장면 우선 선별 |
| 정보 밀도 높은 부분 | 중요한 정보가 집약된 구간 식별 |
| 감정적 몰입도 | 감정 곡선이 높은 순간 (웃음, 놀람, 공감 등) 탐지 |
| 발화 분석 | 음성 내용 기반 주제 중요도 판별 |

### 생성 비율
| 원본 영상 길이 | 생성 쇼츠 수 |
|---------------|-------------|
| 2분 | 약 1개 |
| 10분 | 약 5개 |
| 20분 | 약 10개 |
| N분 | N ÷ 2개 (편차 있음) |

### 지원 플랜
- 무료 / Basic / Pro / Max 전 플랜 지원

### 처리 순서
1. 영상 입력 완료
2. 오디오 트랜스크립션 (음성 → 텍스트 변환)
3. 장면 중요도 스코어링
4. 상위 N개 구간 추출
5. 각 구간에 자막·템플릿 자동 적용
6. 쇼츠 결과물 생성 완료

## F-B-02. AI 자동 자막 생성

### 개요
음성 인식(STT) 기술로 쇼츠 영상의 대사를 자동으로 텍스트 자막으로 변환하여 삽입한다.

### 동작 방식
1. 생성된 쇼츠 클립의 오디오 트랙 분석
2. 발화 구간 탐지 → 텍스트 변환
3. 타임라인에 자막 자동 배치
4. 선택된 템플릿 스타일 적용

### 자막 편집 기능 (후처리)
| 편집 항목 | 설명 |
|-----------|------|
| 오타 수정 | 자막 텍스트 직접 수정 |
| 문장 병합 | 짧게 쪼개진 자막 합치기 |
| 타임라인 싱크 조절 | 자막 시작/끝 시간 미세 조정 |
| 스타일 변경 | 폰트, 크기, 색상, 위치 변경 |

### 지원 언어
- 한국어 (주요 지원)
- 추가 언어: 공개 명시 없음

### 지원 플랜
- 전 플랜 지원 (무료 포함)

## F-B-03. 무음 구간 자동 제거

### 개요
영상 내 말이 없거나 소리가 없는 구간(공백/침묵)을 AI가 자동 감지하여 제거함으로써 속도감 있는 쇼츠를 만든다.

### 동작 방식
1. 오디오 파형 분석
2. 발화 없는 구간(무음) 자동 마킹
3. 해당 구간 컷 아웃 처리
4. 자연스러운 이어붙이기

### 효과
- 지루한 구간 제거로 시청 이탈률 감소
- 동일 내용을 더 짧은 시간에 전달
- 쇼츠 특성(빠른 호흡)에 최적화

### 제약 조건
- **Pro 멤버십 이상** 전용 기능
- 무음 감지 민감도 조절 가능 여부: 공개 명시 없음

## F-B-04. AI 후킹 보이스 (나레이션)

### 개요
영상 인트로에 전문 성우 스타일의 AI 음성 나레이션을 자동으로 생성하여 삽입한다.

### 동작 방식
1. 영상의 핵심 내용 요약 텍스트 자동 생성
2. TTS(Text-to-Speech) AI 음성으로 변환
3. 영상 시작 부분에 나레이션 오디오 삽입

### 목적
- 시청자의 첫 3~5초 이탈 방지
- 영상 몰입감 2배 향상 (공식 표현)
- 전문 성우 없이 고퀄리티 인트로 연출

### 설정 옵션
| 옵션 | 내용 |
|------|------|
| ON/OFF 토글 | 후킹 보이스 사용 여부 선택 |
| 음성 스타일 | 선택 가능 여부 공개 명시 없음 |

### 지원 플랜
- 전 플랜 지원 (편집 화면 내 설정)

## F-B-05. 첫 3초 후킹 (하이라이트 배치)

### 개요
영상 전체 중 가장 임팩트 있는 클라이맥스 장면을 AI가 자동으로 찾아내어 영상 첫 3초에 배치한다.

### 동작 방식
1. 추출된 쇼츠 클립 전체 분석
2. 가장 몰입도 높은 장면(클라이맥스) 자동 식별
3. 해당 장면을 영상 맨 앞 3초로 이동 배치
4. 이후 본편 영상 이어지도록 재구성

### 목적
- 유튜브 Shorts / TikTok 알고리즘 대응
- 시청 유지율(Retention Rate) 향상
- 첫 3초 이탈률 최소화

### 자동화 수준
- 완전 자동 (별도 사용자 조작 불필요)
- 결과물 미리보기 후 수동 변경 가능

### 지원 플랜
- 전 플랜 지원

## F-B-06. AI 제목 자동 추천

### 개요
생성된 쇼츠에 클릭을 유도하는 제목을 AI가 자동으로 추천한다.
유튜브 업로드 시 태그, 카테고리, 설명글까지 함께 자동 생성된다.

### 생성 항목
| 항목 | 설명 |
|------|------|

| 해시태그 | 알고리즘 노출 최적화 태그 |
| 카테고리 | 유튜브 업로드용 카테고리 자동 분류 |
| 설명글 | 영상 내용 요약 설명 텍스트 |

### 위치
- 결과물 확인 화면 내 편집 메뉴
- 유튜브 업로드 연동 화면

### 편집 가능 여부

## F-E-01. 결과물 확인 (내 프로젝트)

사용자가 생성한 모든 쇼츠 프로젝트를 한 페이지에서 확인한다.

### 표시 정보
| 항목 | 설명 |
|------|------|
| 썸네일 | 쇼츠 대표 이미지 미리보기 |
| 제목 | AI 추천 제목 또는 수정된 제목 |
| 생성일 | 쇼츠 생성 날짜/시간 |
| 원본 영상 정보 | 입력 영상 제목 또는 파일명 |

| 기능 | 설명 |
|------|------|
| 쇼츠 미리보기 재생 | 인라인 또는 모달로 영상 재생 |
| 편집 화면 이동 | 해당 쇼츠 편집 화면으로 진입 |
| 재생성 | 쇼츠 다시 만들기 (이용권 50% 소모) |
| 다운로드 | 로컬 저장 |
| 유튜브 업로드 | 직접 유튜브 채널에 업로드 |

## F-E-02. 쇼츠 재생성

### 개요
결과물이 불만족스러울 경우 AI가 동일 영상에서 다른 구간을 추출하여 쇼츠를 새로 생성한다.

### 접근 경로
- 결과물 확인 화면 → "재생성하기" 버튼
- 내 프로젝트 페이지 → 각 프로젝트 카드 내 버튼

### 이용권 정책
| 구분 | 소모량 |
|------|--------|
| 최초 생성 | 원본 영상 전체 시간(분) |
| 재생성 | 최초 소모량의 **50%** |

### 동작 방식
1. "재생성하기" 버튼 클릭
2. 이용권 잔량 및 차감량 확인 팝업 표시
3. 사용자 확인 → AI가 다른 하이라이트 구간으로 재추출
4. 새 결과물 생성 완료

### 목적
- 첫 결과물이 마음에 들지 않을 경우 저렴한 비용으로 재시도 가능
- 다양한 하이라이트 구간 중 최적 결과물 선택

### 지원 플랜
- 전 플랜 지원

| 설명글 | 영상 내용 기반 설명 텍스트 |

### 동작 방식

4. 사용자가 내용 수정 후 "업로드" 최종 확인
5. 유튜브 채널에 쇼츠 업로드 완료

### 유의 사항
- 유튜브 계정 연동 필요 (최초 1회)
- 업로드 전 반드시 내용 검토 권장

# 템플릿 기능 명세 (Feature: Template)

## 개요

매번 반복 설정 없이 일관된 비주얼의 쇼츠를 빠르게 생성하는 기능이다.

---

## F-D-01. 제공 템플릿 선택

### 개요
AlphaCut이 사전 제작한 디자인 템플릿을 선택하여 쇼츠에 일괄 적용한다.

### 제공 템플릿 목록

| 템플릿명 | 스타일 설명 | 특징 |
|----------|-------------|------|
| SandPaper | 트렌디한 배경 디자인 | 거친 질감의 배경, 현대적 느낌 |
| Minimal | 깔끔한 미니멀 스타일 | 여백 강조, 텍스트 중심 |
| Vibrant | 생동감 있는 컬러풀 디자인 | 강한 색감, 에너지 넘치는 느낌 |
| GreenLine | 네온 그린 라인 스타일 | 형광 그린 라인 포인트 |
| Newsletter | 뉴스레터 스타일 디자인 | 정보 전달형, 텍스트 레이아웃 강조 |

### 템플릿 적용 요소
| 요소 | 설명 |
|------|------|
| 배경 디자인 | 영상 외 영역의 배경 그래픽 |
| 자막 스타일 | 폰트, 색상, 위치, 크기 |
| 레이아웃 구성 | 영상 영역과 텍스트 영역 배치 |
| 로고 이미지 | 채널 로고 삽입 위치 및 스타일 |

### 동작 방식
1. AI 쇼츠 생성 시 또는 편집 화면에서 템플릿 선택
2. 선택 즉시 쇼츠 전체에 해당 템플릿 스타일 적용
3. 미리보기로 결과 확인 후 다른 템플릿으로 전환 가능

### 지원 플랜
- 전 플랜 지원 (무료 포함)

---

## F-D-02. 사용자 정의 템플릿 저장

### 개요
사용자가 직접 설정한 편집 옵션(자막 스타일, 레이아웃, 배경 등)을
커스텀 템플릿으로 저장하여 이후 프로젝트에서 재사용한다.

### 저장 가능 설정 항목
| 항목 | 설명 |
|------|------|
| 자막 스타일 | 폰트, 크기, 색상, 위치 |
| 영상 레이아웃 | 화면 비율, 영상 영역 크기 및 위치 |
| 배경 디자인 | 커스텀 배경 색상 또는 이미지 |
| 커스텀 폰트 | 업로드된 폰트 포함 (Pro 이상) |
| 로고 위치 | 채널 로고 삽입 위치 |

### 동작 방식
1. 편집 화면에서 원하는 스타일 설정 완료
2. "템플릿으로 저장" 버튼 클릭
3. 템플릿 이름 입력 후 저장
4. 다음 프로젝트 시작 시 저장된 템플릿 목록에서 선택 적용

### 목적
- 쇼츠 제작 시간 단축 (효율 200% 향상 — 공식 표현)
- 채널 브랜딩 일관성 유지
- 매번 같은 설정 반복 제거

### 지원 플랜
- 전 플랜 지원

# 알파토픽 기능 명세 (Feature: AlphaTopic)

> 파일 위치: alphacut-docs/01_feature/feature_alphatopic.md  
> 기능 ID: F-F  
> 작성일: 2026-04-16  
> 상태: **Beta 서비스** (2026년 1월 21일 출시)

---

## 개요

키워드 기반으로 유튜브 트렌드를 데이터 분석하여
"지금 뜨는 영상"의 이유를 찾아내는 리서치 도구다.
콘텐츠 주제 선정에 감이 아닌 데이터 근거를 제공한다.

---

## F-F-01. 키워드 검색 (트렌드 분석 시작)

### 개요
분석하고 싶은 키워드를 입력하면 해당 키워드 관련 유튜브 영상 데이터를 수집·분석한다.

### 입력
- 키워드 텍스트 입력
- 검색 실행

### 출력
- 해당 키워드 관련 유튜브 영상 목록
- 각 영상의 4가지 핵심 지표 표시

### 접근 경로
- 상단 네비게이션 → "무료 도구" 드롭다운 내 알파토픽
- 또는 직접 URL 접근 (추정)

---

## F-F-02. 4가지 핵심 지표 분석

### 개요
검색 결과 각 영상에 대해 4가지 핵심 지표를 계산하여 표시한다.

### 지표 상세

| 지표명 | 설명 |
|--------|------|
| 조회수 성과 | 해당 영상의 조회수 절대값 및 채널 평균 대비 성과 |
| 알고리즘 기여도 | 유튜브 알고리즘이 해당 영상을 얼마나 추천했는지 추정 수치 |
| 알파지수 | AlphaCut AI가 자체 계산한 영상의 종합 잠재력 점수 |
| 정렬 및 수치 보기 | 각 지표 기준으로 목록 정렬 / 수치 상세 보기 |

### 정렬 기준
- 조회수 높은 순
- 알고리즘 기여도 높은 순
- 알파지수 높은 순
- 최신순

---

## F-F-03. 영상 상세 정보 조회

### 개요
목록의 개별 영상 클릭 시 해당 영상의 상세 분석 데이터를 확인한다.

### 표시 정보
| 항목 | 설명 |
|------|------|
| 영상 제목 | 원본 유튜브 제목 |
| 채널 정보 | 채널명, 구독자 수 |
| 조회수 | 현재 조회수 |
| 좋아요/댓글 수 | 인게이지먼트 지표 |
| 업로드일 | 게시 날짜 |
| 알파지수 상세 | 항목별 점수 분해 |

---

## F-F-04. 필터 및 정교한 검색

### 개요
초기 검색 결과를 다양한 조건으로 좁혀 원하는 분석 범위를 설정한다.

### 필터 항목
| 필터 | 설명 |
|------|------|
| 업로드 기간 | 최근 7일 / 30일 / 3개월 / 1년 등 |
| 조회수 범위 | 최소~최대 조회수 설정 |
| 채널 구독자 규모 | 소형/중형/대형 채널 필터 |
| 영상 길이 | 숏폼 / 미드폼 / 롱폼 구분 |
| 추가 키워드 | 검색 결과 내 추가 키워드로 범위 축소 |

---

## F-F-05. 결과 내 검색 및 새 키워드 검색

### 개요
현재 검색 결과 안에서 다시 검색하거나, 완전히 새로운 키워드로 재검색한다.

### 기능
| 기능 | 설명 |
|------|------|

| 새 키워드 검색 | 기존 결과 초기화 후 새 키워드로 재시작 |

# 추천인 프로그램 기능 명세 (Feature: Referral)

> 파일 위치: alphacut-docs/01_feature/feature_referral.md  
> 기능 ID: F-I  
> 작성일: 2026-04-16  
> 페이지 URL: /referral-promo

---

## 개요

AlphaCut 사용자가 고유 초대 링크/코드를 생성하여 타인에게 공유하고,
공유를 통해 가입한 사용자가 결제할 때마다 커미션을 받는 어필리에이트 프로그램이다.

### 핵심 수치
| 항목 | 내용 |
|------|------|
| 커미션 비율 | **10%** (피추천인 결제 금액 기준) |
| 정산 주기 | 매월 |
| 수익 유형 | 평생 반복 수익 (Recurring) |
| 초대 인원 제한 | **무제한** |

---

## F-I-01. 추천인 코드/링크 생성

### 개요
추천인 대시보드에서 자신만의 고유 초대 코드와 공유 링크를 생성한다.

### 접근 경로
- URL: `/referral-promo` → "지금 시작하기" 버튼
- 로그인 후 추천인 대시보드 진입

### 생성 항목
| 항목 | 설명 |
|------|------|
| 초대 코드 | 고유 알파벳/숫자 조합 코드 |
| 초대 링크 | 코드가 포함된 AlphaCut 가입 URL |

### 동작 방식
1. 로그인 후 추천인 대시보드 접근
2. "초대 코드 생성" 버튼 클릭
3. 고유 코드 및 링크 자동 생성
4. 복사 버튼으로 클립보드 복사

### 공유 채널 (권장)
- 유튜브 영상 설명란 및 영상 내 언급
- 인스타그램, 틱톡, 트위터 등 SNS
- 블로그 리뷰 글
- 뉴스레터

---

## F-I-02. 커미션 수익 구조

### 개요
추천인 링크를 통해 가입한 사용자가 AlphaCut 유료 구독을 결제할 때마다 커미션이 적립된다.

### 커미션 정책
| 항목 | 내용 |
|------|------|
| 커미션 비율 | 결제 금액의 10% |
| 적립 조건 | 피추천인의 유료 구독 유지 상태 |
| 반복 수익 | 피추천인이 구독을 유지하는 한 매월 커미션 발생 |
| 정산 주기 | 매월 정기 정산 |
| 최소 정산 금액 | 공개 명시 없음 |

### 예상 수익 계산 예시 (월 ₩19,800 구독 기준, 10% 커미션)
| 초대 인원 | 월 예상 수익 | 연 예상 수익 |
|-----------|-------------|-------------|
| 1명 | ₩1,980 | ₩23,760 |
| 10명 | ₩19,800 | ₩237,600 |
| 100명 | ₩198,000 | ₩2,376,000 |
| 300명 | ₩594,000 | ₩7,128,000 |
| 500명 | ₩990,000 | ₩11,880,000 |
| 1,000명 | ₩1,980,000 | ₩23,760,000 |

### 세금 처리
| 항목 | 내용 |
|------|------|
| 소득 분류 | 사업소득 (소득세법 기준) |
| 원천징수율 | **3.3%** (소득세 3% + 지방소득세 0.3%) |
| 정산 시 필요 정보 | 주민등록번호 등 세금 처리 정보 |
| 미제출 시 | 정산 보류 |

## F-I-03. 예상 수익 계산기

### 개요
초대 인원 수에 따른 월간/연간 예상 수익을 실시간으로 계산하여 보여주는 인터랙티브 계산기다.

### UI 구성
| 요소 | 설명 |
|------|------|
| 인원 슬라이더 | 1명 ~ 1,000명 범위 슬라이드 |
| 월 예상 수익 표시 | 슬라이더 조작 시 실시간 업데이트 |
| 연 예상 수익 표시 | 월 수익 × 12 자동 계산 |
| 계산 기준 안내 | "월 ₩19,800 구독 기준, 10% 커미션 적용" |

### 계산 공식
- 월 예상 수익 = 초대 인원 × 19,800 × 0.1
- 연 예상 수익 = 월 예상 수익 × 12
