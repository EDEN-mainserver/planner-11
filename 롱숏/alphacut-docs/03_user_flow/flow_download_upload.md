# 다운로드 & 유튜브 업로드 플로우 (FL-05)

> 파일 위치: alphacut-docs/03_user_flow/flow_download_upload.md
> 작성일: 2026-04-16

---

## 플로우 개요

| 항목 | 내용 |
|------|------|
| 플로우 ID | FL-05 |
| 시작점 | 편집 완료 화면 또는 결과물 확인 화면 |
| 종료점 | 로컬 저장 완료 or 유튜브 업로드 완료 |

---

## A. 다운로드 플로우

```
┌─────────────────────────────────────────────┐
│  STEP A-1: 다운로드 버튼 클릭               │
│  - 편집 화면 상단 툴바 '다운로드' 버튼       │
│  - 결과물 확인 화면 카드의 다운로드(↓) 버튼  │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP A-2: 플랜 확인                        │
│                                             │
│  원본 화질 가능 여부?                        │
│  ├── Basic 이상 → 원본 화질 다운로드         │
│  └── 무료 플랜  → 화질 제한 or 업그레이드 유도│
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP A-3: 다운로드 진행                    │
│  - 파일 렌더링 (필요 시)                     │
│  - 브라우저 다운로드 트리거                  │
│  - 파일명: [쇼츠 제목].mp4                   │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP A-4: 다운로드 완료                    │
│  - '다운로드가 완료되었습니다' 토스트 표시   │
│  - 로컬 저장 폴더에 MP4 파일 저장            │
└─────────────────────────────────────────────┘
```

---

## B. 유튜브 업로드 플로우

```
┌─────────────────────────────────────────────┐
│  STEP B-1: 유튜브 업로드 버튼 클릭          │
│  - 편집 화면 상단 툴바 '업로드' 버튼         │
│  - 결과물 카드의 유튜브 아이콘 버튼           │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP B-2: 유튜브 계정 연동 확인            │
│                                             │
│  연동된 유튜브 계정 있음?                    │
│  ├── Yes → STEP B-3으로 이동                │
│  └── No  → Google OAuth 인증 팝업           │
│           → 유튜브 채널 권한 허용             │
│           → 연동 완료 후 STEP B-3로 이동     │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP B-3: AI 추천 메타데이터 표시          │
│                                             │
│  AI가 자동 생성한 항목:                      │
│  - 제목 (클릭 유도형, 수정 가능)             │
│  - 해시태그 (알고리즘 최적화, 수정 가능)      │
│  - 카테고리 (자동 분류, 변경 가능)           │
│  - 설명글 (영상 내용 요약, 수정 가능)        │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP B-4: 메타데이터 검토 및 수정          │
│  - 사용자가 AI 추천 내용 확인                │
│  - 필요 시 제목/태그/설명글 직접 수정        │
│  - 공개 설정 선택 (공개/비공개/예약)         │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP B-5: 업로드 확인 및 실행              │
│  - '업로드' 최종 버튼 클릭                   │
│  - 유튜브 채널에 쇼츠 업로드 진행            │
│  - 업로드 진행률 표시                        │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP B-6: 업로드 완료                      │
│  - '유튜브에 업로드되었습니다!' 토스트        │
│  - 유튜브 영상 링크 제공 (클릭 시 유튜브 이동)│
│  - 내 프로젝트로 돌아가기                    │
└─────────────────────────────────────────────┘
```

---

## 플랜별 다운로드 제한

| 플랜 | 다운로드 가능 | 화질 |
|------|-------------|------|
| 무료 | 제한적 (추정) | 표준 화질 |
| Basic | 가능 | 원본 화질 |
| Pro | 가능 | 원본 화질 |
| Max | 가능 | 원본 화질 |

---

## 오류 케이스

| 오류 | 메시지 | 대응 |
|------|--------|------|
| 유튜브 계정 연동 실패 | 계정 연동에 실패했습니다 | 재시도 버튼 |
| 유튜브 업로드 실패 | 업로드 중 오류가 발생했습니다 | 재시도 or 로컬 저장 대안 안내 |
| 다운로드 파일 생성 오류 | 파일 생성에 실패했습니다 | 재시도 버튼 |
