## 플로우 개요

| 항목 | 내용 |
|------|------|
| 플로우 ID | FL-02 |
| 대상 사용자 | 로그인된 사용자 |
| 시작점 | 홈(/) Hero 섹션 또는 /my_project |
| 종료점 | 쇼츠 결과물 확인 화면 |
| 전제 조건 | 로그인 상태, 이용권 잔량 보유 |

## 상세 플로우

```
┌─────────────────────────────────────────────┐
│  STEP 1: 홈(/) 또는 /my_project 접속        │
│  - 로그인 상태 확인                          │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 2: '유튜브 링크' 탭 선택              │
│  - 기본 활성 탭 (그라디언트 배경)            │
│  - 링크 아이콘(🔗) + '유튜브 링크' 텍스트   │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 3: YouTube URL 입력                   │
│  - 텍스트 필드에 URL 직접 입력 또는 붙여넣기 │
│  - 형식: youtube.com/watch?v=XXX            │
│         또는 youtu.be/XXX                   │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 4: '쇼츠로 변환하기 →' 버튼 클릭     │
└────────────────────┬────────────────────────┘
                     │
             [로그인 상태 확인]
                     │
         ┌───────────┴───────────┐
         │                       │
      [로그인됨]             [비로그인]
         │                       │
         │              [로그인 화면으로 이동]
         │              → 로그인 후 STEP 4 재개
         │
         ▼
┌─────────────────────────────────────────────┐
│  STEP 5: URL 유효성 검증                    │
│                                             │
│  유효한 YouTube URL?                        │
│  ├── Yes → 다음 단계                        │
│  └── No  → 오류 메시지 표시                 │
│           '올바른 유튜브 URL을 입력하세요'   │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 6: 이용권 잔량 확인                   │
│                                             │
│  이용권 충분? (영상 길이 ≤ 잔여 이용권)      │
│  ├── Yes → 다음 단계                        │
│  └── No  → 이용권 부족 팝업 표시            │
│           → 요금제 구매 유도 (FL-06)         │
└────────────────────┬────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 7: 구간 설정 (선택)                   │

│  - 구간 지정 시 이용권 절약 가능             │
│  - 시작/종료 시간 입력                       │
│  - '전체 처리' 선택 시 이 단계 스킵          │

                     │
                     ▼
┌─────────────────────────────────────────────┐
│  STEP 8: AI 처리 시작                       │
│  - 이용권 차감 (영상 길이 또는 구간 길이)    │
│  - 로딩 화면 표시                            │
│    · 진행률 표시 (% 또는 단계 표시)          │
│    · '하이라이트를 분석하고 있어요...' 등    │
└────────────────────┬────────────────────────┘
                     │
         ┌───────────┴───────────┐
         │                       │
      [처리 완료]            [처리 실패]
         │                       │
         │               [오류 메시지 표시]

         │
         ▼
┌─────────────────────────────────────────────┐
│  STEP 9: 결과물 확인 화면                   │
│  - 생성된 쇼츠 목록 표시                     │
│  - 썸네일 + AI 추천 제목 + 길이              │
│  - 미리보기 / 편집 / 재생성 / 다운로드 버튼  │
│  → FL-04 (편집) 또는 FL-05 (다운로드/업로드) │
└─────────────────────────────────────────────┘
```

## AI 처리 내부 단계

| 내부 처리 단계 | 설명 |
|----------------|------|
| 1. 영상 다운로드 | YouTube URL에서 영상 원본 수집 |
| 2. 오디오 트랜스크립션 | 음성 → 텍스트 변환 (STT) |
| 3. 장면 중요도 분석 | 구간별 하이라이트 점수 산출 |
| 4. 클립 추출 | 상위 N개 구간 선정 (2분당 1클립) |
| 5. 자막 생성 | 각 클립 자막 자동 생성 |
| 6. 첫 3초 후킹 배치 | 클라이맥스 구간 인트로 배치 |
| 7. 제목 추천 | AI 클릭 유도형 제목 생성 |
| 8. 템플릿 적용 | 기본 템플릿 또는 저장 템플릿 적용 |
| 9. 쇼츠 렌더링 | 최종 MP4 파일 생성 |

---

## 이용권 차감 정책

| 케이스 | 차감량 |
|--------|--------|
| 전체 영상 처리 | 원본 영상 재생 시간(분) |
| 구간 설정 처리 | 선택된 구간 시간(분) |
| 처리 실패 | 환불 또는 재시도 가능 (정책 확인 필요) |

---

## 오류 케이스 처리

| 오류 상황 | 표시 메시지 | 대응 |
|-----------|------------|------|
| 잘못된 URL | 올바른 유튜브 URL을 입력해 주세요 | 입력 필드 포커스 |
| 비공개 영상 | 접근할 수 없는 영상입니다 | 공개 영상 안내 |
| 이용권 부족 | 이용권이 부족합니다 | 요금제 페이지 이동 버튼 |
| 서버 오류 | 처리 중 오류가 발생했습니다. 재시도해 주세요 | 재시도 버튼 |
