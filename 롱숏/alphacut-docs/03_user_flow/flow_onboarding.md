# 신규 사용자 온보딩 플로우 (FL-01)

> 파일 위치: alphacut-docs/03_user_flow/flow_onboarding.md
> 작성일: 2026-04-16

---

## 플로우 개요

| 항목 | 내용 |
|------|------|
| 플로우 ID | FL-01 |
| 대상 사용자 | 최초 방문 비로그인 신규 사용자 |
| 시작점 | 홈 페이지(/) 접속 |
| 종료점 | 첫 쇼츠 생성 시작 또는 서비스 탐색 |
| 핵심 목표 | 회원가입 완료 + 30분 무료 이용권 획득 |

---

## 단계별 UI 요소 매핑

| 단계 | 화면/페이지 | 핵심 UI 요소 | 사용자 액션 |
|------|-----------|-------------|-------------|
| 1 | 홈 (/) | 전체 페이지 | 페이지 로드 확인 |
| 2 | Hero 섹션 | H1, 소셜 배지, 갤러리 | 시각적 탐색 |
| 3 | 서비스/기능/가격 섹션 | 기능 카드, 후기 카드, FAQ | 스크롤 탐색 |
| 4 | CTA 버튼 | 그라디언트 버튼 4종 | 클릭 |
| 5 | Google OAuth 팝업 | 계정 목록 | 계정 선택 |
| 6 | Google 권한 허용 | 허용 버튼 | 클릭 |
| 7 | 가입 완료 | 토스트 메시지 | 확인 |
| 8 | 홈 or 내 프로젝트 | 입력창 or 빈 프로젝트 | 쇼츠 생성 시작 |

## 이탈 포인트 및 대응

| 이탈 포인트 | 원인 | 대응 UI |
|-------------|------|----------|
| STEP 2 이탈 | 서비스 가치 이해 부족 | 소셜 배지, 2100% 조회수 수치 강조 |
| STEP 3 이탈 | 가격 불신 | 300원/쇼츠 강조, 후기 캐러셀 |
| STEP 5 이탈 | Google 계정 없음 or 개인정보 우려 | 신용카드 불필요 문구, 간편 가입 강조 |
| STEP 8 이탈 | 첫 사용 방법 모름 | 빈 상태 CTA, 사용 가이드 링크 |
