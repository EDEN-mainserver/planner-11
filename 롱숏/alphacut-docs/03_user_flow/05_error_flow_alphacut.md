## 상세 에러 플로우

---

### E01. 미로그인 접근 처리

```
[비로그인 상태에서 보호된 페이지 접근]
  └─> /my_project 접근 시
        └─ 즉시 홈(/)으로 리다이렉트
              └─ Hero 섹션 노출 (로그인 유도 UI)
                    └─ 로그인 버튼 클릭 → 로그인 모달

  └─> 요금제 "시작하기" 버튼 클릭 시
        └─ 로그인 모달 오픈
              └─ Google 로그인 완료 → 결제 페이지 이동

  └─> 추천인 프로그램 "지금 시작하기" 클릭 시
        └─ 로그인 모달 오픈
              └─ 로그인 완료 → 추천인 대시보드 이동
```

**처리 원칙:**
- 보호된 페이지는 홈 리다이렉트 또는 모달로 처리 (페이지 자체 차단)
- 로그인 후 원래 의도한 페이지/액션으로 복귀

### E02. 이용권 소진

```
[쇼츠 생성 시도]
  └─> 잔여 이용권 확인
        ├─ 충분: 정상 생성 진행
        └─ 부족/소진:
              └─ 인라인 알림: "이용권이 부족합니다. 현재 잔여: X분"
                    └─ CTA 버튼: "멤버십 업그레이드하기" → /pricing 이동
                          └─ 또는 추가 이용권 구매 옵션 제공 (추정)
```

**UX 가이드:**
- 잔여 이용권이 10분 이하일 때 사전 경고 표시 (추정)
- 이용권 만료 전 이메일 알림 발송 가능성 있음
- 이용권은 결제 주기 내 미사용분 자동 소멸 (이월 불가)

### E03. 잘못된 YouTube URL 입력

```
[Hero 또는 대시보드 입력창]
  └─> 사용자가 유튜브 URL이 아닌 텍스트 입력 후 제출
        └─ 유효성 검사 실패
              └─ 인라인 오류 메시지: "올바른 유튜브 URL을 입력해주세요"
                    └─ 입력창 테두리 빨간색 강조
                          └─ 포커스 유지 + 재입력 유도
```

### E04. 비공개/삭제된 YouTube 영상

```
[유효한 URL 제출 후 서버 처리 중]
  └─> 유튜브 영상 접근 불가 감지
        └─ 오류 메시지: "해당 영상에 접근할 수 없습니다"
              └─ 원인 안내: 비공개, 삭제, 지역 제한 등
                    └─ 이용권 차감 없음 (생성 전 단계 오류)
                          └─ [다른 링크 시도하기] 버튼
```

### E05. 파일 업로드 실패

```

  │     └─ "지원하지 않는 파일 형식입니다. mp4, mov 형식을 사용해주세요."
```

### E06. 업로드 중 탭 닫기 시도

## 에러 처리 UX 원칙

1. **이용권 차감 전 오류**: 이용권을 소모하지 않음 (명확히 고지)
2. **편집/다운로드는 이용권 소모 없음**: 생성 시에만 차감
3. **재생성은 50% 차감**: 오류로 인한 재시도와 일반 재생성 구분 필요
4. **오류 메시지는 인라인 우선**: 모달 팝업 최소화
5. **항상 다음 액션 제시**: 오류 메시지 + 해결 버튼 함께 표시
6. **고객 지원 연결**: 해결 불가 시 챗봇 또는 이메일(teamweexe@gmail.com) 안내
